# revfactory/harness

A meta-skill that designs domain-specific agent teams, defines specialized agents, and generates the skills they use.

项目定位  
一个用于设计领域特定代理团队的元技能框架，帮助构建和管理专业化代理及其技能。

核心功能  
- 设计领域特定代理团队  
- 定义代理角色与职责  
- 生成代理使用的技能  
- 支持多代理协作  
- 提供可扩展的架构

技术特点  
基于 HTML 构建，强调模块化与可扩展性，适合集成到不同开发环境中，专注于代理系统的设计与管理。

适用场景  
- AI 代理系统开发  
- 复杂任务自动化  
- 多代理协作系统  
- 领域专用智能体构建  
- 智能助手平台搭建

推荐理由  
提供结构化方法设计和管理代理团队，适合需要构建复杂 AI 协作系统的开发者，具有良好的扩展性和灵活性。
