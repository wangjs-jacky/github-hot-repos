# yzfly/Awesome-MCP-ZH

MCP 资源精选， MCP指南，Claude MCP，MCP Servers, MCP Clients

项目定位  
一个专注于 MCP（Model Collaboration Protocol）资源的精选指南，提供 MCP 指南、服务器、客户端及相关工具。

核心功能  
- 收录 MCP 相关资源和工具  
- 提供 MCP 服务器和客户端信息  
- 包含 Claude、DeepSeek、Qwen 等模型的 MCP 支持  
- 整理 MCP 协议文档和最佳实践  
- 汇总 MCP 服务目录和开发资源

技术特点  
项目聚焦于 MCP 协议生态，整合多个大模型厂商的 MCP 实现方案，便于开发者快速查找和使用相关资源。

适用场景  
- 开发者寻找 MCP 服务和工具  
- 研究者了解 MCP 协议实现  
- 企业部署多模型协作系统  
- 学习如何集成不同大模型  
- 构建跨平台 AI 服务架构

推荐理由  
该项目为 MCP 生态提供了全面的资源汇总，适合希望深入了解或应用 MCP 协议的开发者和研究者。
