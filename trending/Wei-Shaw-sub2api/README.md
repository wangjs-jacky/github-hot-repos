# Wei-Shaw/sub2api

Sub2API-CRS2 一站式开源中转服务，让 Claude、Openai 、Gemini、Antigravity订阅统一接入，支持拼车共享，更高效分摊成本，原生工具无缝使用。

**项目定位**  
Sub2API-CRS2 是一个开源中转服务，将多个 AI 订阅接口统一接入，支持共享使用以降低成本，提升效率。

**核心功能**  
- 支持 Claude、OpenAI、Gemini 等多平台 API 接入  
- 提供拼车共享功能，分摊订阅成本  
- 原生工具无缝集成，简化使用流程  
- 支持 CRIS2 协议，增强兼容性  

**技术特点**  
采用 Go 语言开发，性能高效，模块化设计便于扩展，支持多种 AI 接口协议，具备良好的可维护性和稳定性。

**适用场景**  
- 个人开发者需要多平台 API 支持  
- 团队共享 AI 订阅资源降低成本  
- 需要统一管理不同 AI 服务的用户  
- 构建 AI 中间件或代理服务  

**推荐理由**  
功能全面、易于部署，适合需要整合多个 AI 服务的用户，能有效节省成本并提升使用效率。
