# ykdojo/claude-code-tips

45 tips for getting the most out of Claude Code, from basics to advanced - includes a custom status line script, cutting the system prompt in half, using Gemini CLI as Claude Code's minion, and Claude Code running itself in a container. Also includes the dx plugin.

项目定位  
一个为 Claude Code 用户提供高效使用技巧的指南，涵盖基础到高级功能，提升开发效率和体验。

核心功能  
- 提供 45 条 Claude Code 使用技巧  
- 包含自定义状态行脚本  
- 支持将 Gemini CLI 作为 Claude Code 的辅助工具  
- 实现 Claude Code 在容器中运行  
- 集成 dx 插件增强功能

技术特点  
项目以 JavaScript 为主，聚焦于 AI 工具的高效集成与自动化流程优化，强调实用性和可扩展性。

适用场景  
- 开发者提升 Claude Code 使用效率  
- 自动化代码生成与调试流程  
- 构建 AI 驱动的开发工作流  
- 简化系统提示词管理  
- 实现多 AI 工具协同工作

推荐理由  
适合希望最大化利用 Claude Code 功能的开发者，提供全面技巧和实用工具，提升生产力和开发体验。
