# grab/cursor-talk-to-figma-mcp

TalkToFigma: MCP integration between AI Agent (Cursor, Claude Code) and Figma, allowing Agentic AI to communicate with Figma for reading designs and modifying them programmatically.

项目定位  
该项目实现了 AI 代理（如 Cursor、Claude Code）与 Figma 的 MCP 集成，使 AI 能够读取和程序化修改设计。

核心功能  
- AI 代理与 Figma 通信  
- 读取 Figma 设计内容  
- 程序化修改 Figma 设计  
- 支持生成式 AI 和 LLM 协作  
- 提供模型上下文协议支持

技术特点  
基于 JavaScript 构建，利用 MCP 协议实现 AI 与设计工具的交互，强调自动化和智能设计协作。

适用场景  
- UI/UX 设计自动化  
- 快速迭代设计原型  
- AI 辅助设计修改  
- 开发者与设计师协作  
- 生成式 AI 驱动的设计流程

推荐理由  
适合希望将 AI 引入设计流程的开发者和设计师，提升设计效率和自动化水平。
