# shadcn/improve

Use your most capable model to audit your codebase and write plans for cheaper models to execute.

项目定位  
一个用于代码审计和优化的工具，利用高级模型分析代码库并为低配模型生成执行计划，提升代码质量与效率。

核心功能  
- 利用强大模型对代码进行深度审计  
- 生成适用于低成本模型的执行方案  
- 提供代码优化建议  
- 支持多语言代码分析  
- 自动化生成改进策略

技术特点  
基于先进模型进行代码分析，结合自动化策略生成，提升代码维护效率，适合不同性能需求的环境。

适用场景  
- 企业代码质量审查  
- 开发团队协作中的代码优化  
- 低算力环境下的代码执行规划  
- 自动化代码改进流程  
- 代码审计与安全检查

推荐理由  
通过高效模型与低成本模型协作，提升代码分析与优化效率，适合需要兼顾性能与成本的开发场景。
