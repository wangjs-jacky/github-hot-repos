# phuryn/pm-skills

PM Skills Marketplace: 100+ agentic skills, commands, and plugins — from discovery to strategy, execution, launch, and growth.

项目定位  
PM Skills Marketplace 是一个提供 100+ 智能体技能、命令和插件的平台，帮助产品经理从发现到增长的全流程中提升效率。

核心功能  
- 提供多种智能体技能和插件  
- 支持产品管理全流程（策略、执行、发布、增长）  
- 集成 Claude AI 工具链  
- 促进协作与自动化工作流  
- 便于快速查找和应用相关技能

技术特点  
项目聚焦于智能体技能的组织与分发，结合 AI 工具如 Claude 进行扩展，具备良好的可集成性和模块化设计，适合开发者和产品经理使用。

适用场景  
- 产品经理日常任务自动化  
- 团队协作与知识共享  
- AI 工具集成与扩展  
- 产品策略制定与执行  
- 快速原型开发与测试

推荐理由  
该仓库为产品经理提供了丰富的智能体技能资源，提升工作效率并支持 AI 工具的深度使用，是产品管理领域的实用工具集合。
