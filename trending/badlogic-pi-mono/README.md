# badlogic/pi-mono

AI agent toolkit: coding agent CLI, unified LLM API, TUI & web UI libraries, Slack bot, vLLM pods

项目定位  
一个面向 AI 代理的工具包，提供 CLI、UI、API 等多种交互方式，便于开发和部署 AI 代理应用。

核心功能  
- 编程代理 CLI 工具  
- 统一 LLM API 接口  
- TUI 和 Web UI 库  
- Slack 机器人集成  
- vLLM pods 支持

技术特点  
基于 TypeScript 构建，支持多端交互，提供灵活的 API 和 UI 模块，适合快速构建 AI 代理应用。

适用场景  
- 开发 AI 编程助手  
- 构建 AI 驱动的聊天机器人  
- 部署 AI 代理到 Slack 等平台  
- 快速搭建 Web 或终端界面  
- 集成大模型服务进行推理

推荐理由  
功能全面，适合开发者快速构建 AI 代理应用，支持多种交互方式，具备良好的扩展性和易用性。
