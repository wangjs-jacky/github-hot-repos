# supertone-inc/supertonic

Lightning-Fast, On-Device, Multilingual TTS — running natively via ONNX.

**项目定位**  
一款快速、本地化、多语言的文本转语音工具，通过 ONNX 实现原生运行，适用于移动设备和嵌入式系统。

**核心功能**  
- 支持多种语言的文本转语音  
- 本地运行，无需网络连接  
- 高性能，低延迟  
- 轻量级设计，适合移动设备  

**技术特点**  
使用 Swift 编写，基于 ONNX 运行模型，实现跨平台和高效推理，支持多种编程语言生态。

**适用场景**  
- 移动应用中的语音反馈  
- 嵌入式系统的语音交互  
- 离线环境下的语音生成  
- 多语言应用程序的本地化支持  

**推荐理由**  
轻量、高效、多语言支持，适合需要本地化 TTS 的应用场景，尤其适合对性能和隐私有要求的项目。
