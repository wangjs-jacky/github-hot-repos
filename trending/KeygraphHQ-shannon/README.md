# KeygraphHQ/shannon

Shannon Lite is an autonomous, white-box AI pentester for web applications and APIs. It analyzes your source code, identifies attack vectors, and executes real exploits to prove vulnerabilities before they reach production.

项目定位  
Shannon Lite 是一款用于 Web 应用和 API 的自主白盒渗透测试工具，旨在通过代码分析和真实漏洞利用，在生产前发现并验证安全问题。

核心功能  
- 自动化源码分析以识别攻击面  
- 执行真实漏洞利用以验证风险  
- 提供详细的漏洞报告和修复建议  
- 支持多种 Web 技术栈的检测  
- 与 CI/CD 流程集成进行持续安全审计

技术特点  
基于 TypeScript 构建，具备良好的可扩展性和跨平台能力，结合静态分析与动态测试技术，实现高效、精准的安全检测。

适用场景  
- 开发阶段的自动化安全审计  
- CI/CD 流程中的安全检查  
- 安全团队的渗透测试支持  
- 企业级 Web 应用的安全评估  
- 第三方组件的安全风险排查

推荐理由  
Shannon Lite 为开发者和安全团队提供了一种高效、自动化的安全测试方式，能够提前发现潜在漏洞，提升应用安全性，适合现代 DevSecOps 实践。
