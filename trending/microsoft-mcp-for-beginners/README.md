# microsoft/mcp-for-beginners

This open-source curriculum introduces the fundamentals of Model Context Protocol (MCP) through real-world, cross-language examples in .NET, Java, TypeScript, JavaScript, Rust and Python. Designed for developers, it focuses on practical techniques for building modular, scalable, and secure AI workflows from session setup to service orchestration.

项目定位  
该项目是微软推出的开源课程，旨在通过多种编程语言介绍 Model Context Protocol (MCP) 的基础概念和实际应用，帮助开发者构建模块化、可扩展且安全的 AI 工作流。

核心功能  
- 提供跨语言的 MCP 实现示例（.NET、Java、TypeScript 等）  
- 从会话设置到服务编排的完整 AI 流程演示  
- 强调安全性和可扩展性设计  
- 支持开发者快速上手 MCP 协议  

技术特点  
项目采用 Jupyter Notebook 作为教学载体，结合多种主流编程语言，提供直观的代码示例和实验环境，便于开发者理解和实践 MCP 协议的核心技术。

适用场景  
- AI 开发者学习 MCP 协议的入门教程  
- 需要构建跨语言 AI 服务的团队参考  
- 企业希望提升 AI 工作流安全性和可维护性的场景  
- 教育机构用于 AI 相关课程的教学材料  

推荐理由  
该项目为开发者提供了全面的 MCP 学习资源，涵盖多种语言和实际应用场景，适合初学者和希望深入理解 AI 工作流架构的开发者。
