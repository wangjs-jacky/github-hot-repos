# Gitlawb/openclaude

Claude Code opened to any LLM — OpenAI, Gemini, DeepSeek, Ollama, and 200+ models via OpenAI-compatible API shim

项目定位  
Gitlawb/openclaude 是一个开源项目，允许用户通过 OpenAI 兼容 API 模板，将 Claude 代码库适配到多种大语言模型，如 OpenAI、Gemini、DeepSeek 等，实现多模型统一调用。

核心功能  
- 支持多种 LLM 模型接入  
- 提供 OpenAI 兼容 API 接口  
- 实现模型间代码共享与调用  
- 适配 200+ 种不同模型  
- 开源可扩展架构

技术特点  
基于 TypeScript 构建，具备良好的可扩展性和跨平台兼容性，通过 API 封装实现多模型统一接口，提升开发效率和灵活性。

适用场景  
- 多模型对比测试  
- 企业级 LLM 集成  
- 开发者快速接入多种模型  
- AI 应用原型开发  
- 个人项目模型多样化需求

推荐理由  
该项目为开发者提供了便捷的多模型接入方案，简化了不同 LLM 的使用流程，适合需要灵活调用多种模型的场景，具有较高的实用价值和扩展性。
