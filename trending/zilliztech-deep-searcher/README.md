# zilliztech/deep-searcher

Open Source Deep Research Alternative to Reason and Search on Private Data. Written in Python.

项目定位  
一个开源的深度研究替代方案，用于在私有数据上进行推理和搜索，基于 Python 实现。

核心功能  
- 支持多种大模型（如 Claude、Grok、Llama 等）进行推理  
- 提供向量数据库集成（如 Milvus）  
- 实现基于 RAG 的智能搜索与分析  
- 支持多模型协同工作流  

技术特点  
采用 Python 编写，结合向量数据库与大语言模型，实现高效、灵活的私有数据深度研究能力。

适用场景  
- 企业内部知识库检索与分析  
- 个性化 AI 助手构建  
- 多模型联合推理任务  
- 私有数据的智能问答系统  

推荐理由  
适合需要在私有数据上进行高效推理与搜索的开发者和研究人员，支持多种主流模型与数据库，具备良好的扩展性和灵活性。
