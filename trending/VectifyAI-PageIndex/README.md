# VectifyAI/PageIndex

📑 PageIndex: Document Index for Vectorless, Reasoning-based RAG

**项目定位**  
PageIndex 是一个基于推理的 RAG（检索增强生成）系统，旨在通过无向量的方式提升文档索引和信息检索效率，优化大模型的上下文理解能力。

**核心功能**  
- 无需向量数据库的文档索引机制  
- 基于推理的上下文生成与检索  
- 支持复杂查询和多步骤推理任务  
- 提高 RAG 系统的准确性和效率  

**技术特点**  
采用无向量索引技术，结合推理机制实现高效的信息检索，减少对传统向量数据库的依赖，提升系统灵活性和性能。

**适用场景**  
- 需要高效文档检索的 AI 应用  
- 复杂查询和多步骤推理任务  
- 降低向量数据库依赖的 RAG 系统构建  
- 需要高精度信息提取的智能助手  

**推荐理由**  
PageIndex 提供了一种创新的 RAG 实现方式，适合需要高效、精准信息检索的 AI 场景，尤其适合对向量数据库依赖较低的项目。
