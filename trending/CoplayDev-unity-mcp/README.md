# CoplayDev/unity-mcp

Unity MCP acts as a bridge, allowing AI assistants (like Claude, Cursor) to interact directly with your Unity Editor via a local MCP (Model Context Protocol) Client. Give your LLM tools to manage assets, control scenes, edit scripts, and automate tasks within Unity.

项目定位  
Unity MCP 是一个连接 AI 助手与 Unity 编辑器的桥梁，使 LLM 能直接操作 Unity 项目，提升游戏开发效率。

核心功能  
- 允许 AI 直接控制 Unity 编辑器  
- 支持脚本编辑、场景管理、资源处理  
- 通过 MCP 协议实现本地通信  
- 集成多种 AI 模型如 Claude、Cursor、Copilot  
- 自动化游戏开发任务

技术特点  
基于 C# 实现，采用 Model Context Protocol 协议，支持与主流 AI 工具无缝集成，提升开发效率和自动化能力。

适用场景  
- 游戏开发中的自动化任务  
- AI 辅助脚本编写与调试  
- 场景和资源管理  
- 快速原型开发  
- 提升开发团队协作效率

推荐理由  
为 Unity 开发者提供强大的 AI 集成功能，简化开发流程，提升生产力，适合希望利用 LLM 优化游戏开发的团队。
