# getpaseo/paseo

Orchestrate coding agents remotely from your phone, desktop and CLI

项目定位  
Paseo 是一个用于远程协调代码代理的工具，允许开发者通过手机、桌面和 CLI 管理和调度 AI 编码代理，提升开发效率。

核心功能  
- 远程控制和管理 AI 代码代理  
- 支持多种 AI 编码模型如 Claude-Code、Copilot 等  
- 提供跨平台访问（手机、桌面、CLI）  
- 实现代码生成与执行的自动化流程  
- 集成开发工具以增强编码体验

技术特点  
使用 TypeScript 构建，具备良好的跨平台兼容性，支持与多种 AI 编码服务集成，提供灵活的 API 和命令行接口，便于开发者定制和扩展。

适用场景  
- 开发者在移动设备上进行代码编写和调试  
- 多人协作开发时的代码代理协调  
- 自动化代码生成与测试流程  
- 使用多种 AI 工具进行代码优化和分析  
- 快速部署和管理 AI 编码代理任务

推荐理由  
Paseo 提供了统一的界面来管理多个 AI 编码代理，简化了开发流程，适合希望提升编码效率并利用多种 AI 工具的开发者。
