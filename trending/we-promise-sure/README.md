# we-promise/sure

The personal finance app for everyone (by everyone)

项目定位  
一个由社区驱动的个人财务管理应用，旨在为用户提供简单、透明的财务工具。

核心功能  
- 记录和管理个人收支  
- 生成财务报告与图表  
- 支持多用户协作与数据共享  
- 提供预算规划与目标追踪功能  

技术特点  
采用 Ruby 编写，注重代码可读性与可维护性，具备良好的扩展性和社区协作机制。

适用场景  
- 个人日常财务管理  
- 家庭支出记录与分析  
- 小型团队财务协作  
- 个人理财目标设定与跟踪  

推荐理由  
开源且由社区驱动，适合希望参与或学习财务类应用开发的开发者，功能实用且易于定制。
