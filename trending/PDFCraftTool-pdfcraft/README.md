# PDFCraftTool/pdfcraft

PDFCraft is a free, privacy-focused PDF toolkit that runs entirely in your browser. With 90+ professional tools, you can edit, convert, merge, split, and secure your PDF files without ever uploading them to a server.

项目定位  
PDFCraft 是一个在浏览器中运行的免费 PDF 工具包，专注于隐私保护，提供多种 PDF 编辑和处理功能，无需上传文件到服务器。

核心功能  
- PDF 编辑与转换  
- 文件合并与拆分  
- 安全加密与密码保护  
- 无服务器依赖的本地处理  
- 支持 90 多种专业工具

技术特点  
使用 TypeScript 构建，完全在浏览器端运行，确保用户数据隐私，采用现代前端技术实现高效、流畅的用户体验。

适用场景  
- 个人用户处理私密文档  
- 企业内部 PDF 文件管理  
- 需要避免云端存储的敏感文件操作  
- 快速编辑和格式化 PDF 内容  
- 多个 PDF 文件的合并与分割需求

推荐理由  
PDFCraft 提供强大且隐私友好的 PDF 处理功能，无需依赖服务器，适合注重数据安全的用户，功能全面且易于使用。
