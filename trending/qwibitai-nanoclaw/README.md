# qwibitai/nanoclaw

A lightweight alternative to OpenClaw that runs in containers for security. Connects to WhatsApp, Telegram, Slack, Discord, Gmail and other messaging apps,, has memory, scheduled jobs, and runs directly on Anthropic's Agents SDK

项目定位  
一个轻量级的 OpenClaw 替代方案，支持在容器中运行以提高安全性，连接多种消息平台并具备记忆与任务调度功能。

核心功能  
- 连接 WhatsApp、Telegram、Slack 等多个消息平台  
- 支持记忆功能和定时任务  
- 直接运行在 Anthropic 的 Agents SDK 上  
- 容器化部署提升安全性  

技术特点  
基于 TypeScript 开发，采用容器化架构，支持多平台集成，具备良好的扩展性和安全性设计。

适用场景  
- 企业自动化消息处理  
- 个人智能助手开发  
- 安全敏感环境下的 AI 应用部署  
- 多平台消息整合与管理  

推荐理由  
轻量、安全且功能丰富，适合需要多平台集成和容器化部署的 AI 助手项目，尤其适合对安全性要求高的场景。
