# unicity-astrid/book

The canonical reference for Astrid OS: kernel, capsules, host ABI, the bus, and the security model.

项目定位  
Astrid OS 的官方参考实现，涵盖内核、胶囊、主机 ABI、总线及安全模型，为操作系统开发提供核心指导。

核心功能  
- 提供 Astrid OS 内核设计与实现  
- 定义胶囊（capsule）模块化架构  
- 规范主机与设备间的 ABI 接口  
- 实现系统总线通信机制  
- 建立安全模型与权限管理

技术特点  
基于 Perl 语言实现，注重模块化与安全性，提供完整的操作系统底层架构参考，适合研究与教学使用。

适用场景  
- 操作系统内核开发与研究  
- 模块化系统架构设计  
- 安全性与权限管理研究  
- 教学与实验环境搭建  
- 系统级软件开发参考

推荐理由  
作为 Astrid OS 的官方文档与实现，提供全面的系统架构信息，适合开发者深入理解操作系统底层设计与实现机制。
