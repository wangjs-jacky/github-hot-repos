# can1357/oh-my-pi

⌥  AI Coding agent for the terminal — hash-anchored edits, optimized tool harness, LSP, Python, browser, subagents, and more

**项目定位**  
一个面向终端的 AI 编码助手，支持多模型提供商，提供代码编辑、工具调用和智能交互功能。

**核心功能**  
- 基于 AI 的代码生成与编辑  
- 支持多种大语言模型（如 OpenAI、Anthropic）  
- 集成 LSP 和终端 UI 交互  
- 多子代理协同工作  
- 浏览器和 CLI 双模式支持  

**技术特点**  
使用 TypeScript 构建，结合 Rust 工具链，支持多平台运行，具备高效的命令行交互和模块化架构。

**适用场景**  
- 开发者在终端中进行高效编码  
- 需要多模型支持的 AI 编程环境  
- 需要轻量级、快速响应的代码辅助工具  
- 本地与云端混合开发场景  
- 个性化 AI 编程助手定制  

**推荐理由**  
功能全面、支持多模型、界面友好，适合需要强大 AI 编程辅助的开发者提升效率。
