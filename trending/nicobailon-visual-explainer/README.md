# nicobailon/visual-explainer

Agent skill that generates rich HTML pages or slide decks for diagrams, diff reviews, plan audits, data tables, and project recaps

项目定位  
一个生成丰富 HTML 页面或幻灯片的代理技能工具，用于展示图表、代码差异、审计计划、数据表格和项目回顾。

核心功能  
- 生成可视化 HTML 页面  
- 支持幻灯片格式输出  
- 提供图表与数据展示能力  
- 适用于代码差异分析与项目总结  

技术特点  
基于 HTML 技术构建，专注于内容生成与结构化展示，易于集成和扩展，适合开发者快速创建专业级展示页面。

适用场景  
- 代码审查与差异分析  
- 项目汇报与团队沟通  
- 数据可视化展示  
- 技术文档与演示文稿制作  

推荐理由  
简洁高效，适合需要快速生成专业展示内容的开发者和团队，提升沟通与协作效率。
