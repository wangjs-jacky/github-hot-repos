# coleam00/context-engineering-intro

Context engineering is the new vibe coding - it's the way to actually make AI coding assistants work. Claude Code is the best for this so that's what this repo is centered around, but you can apply this strategy with any AI coding assistant!

项目定位  
一个专注于展示如何通过上下文工程提升 AI 编程助手效率的 GitHub 仓库，主要围绕 Claude Code 展开实践。

核心功能  
- 提供 AI 编程助手的上下文工程方法论  
- 展示如何优化提示词以提高代码生成质量  
- 包含实际案例和代码片段演示  
- 探讨不同 AI 工具在上下文工程中的应用  
- 教授如何构建更高效的 AI 编程工作流

技术特点  
使用 Python 实现上下文优化策略，结合实际开发场景，强调如何通过结构化提示提升 AI 代码生成能力，适合初学者和进阶开发者学习。

适用场景  
- 提升 AI 编程助手的代码生成准确性  
- 优化 AI 辅助开发的工作流程  
- 学习上下文工程的基础概念与实践  
- 在实际项目中应用 AI 工具提高效率  
- 为 AI 编程工具选择提供参考

推荐理由  
该项目为 AI 编程提供了实用的上下文工程方法，适合希望提升 AI 工具使用效率的开发者，尤其适合对 Claude Code 或类似工具感兴趣的用户。
