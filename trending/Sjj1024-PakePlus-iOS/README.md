# Sjj1024/PakePlus-iOS

Turn any webpage/HTML/Vue/React and so on into desktop and mobile app under 5M with easy in few minutes. 轻松将任意网站/HTML/Vue/React等项目构建为轻量级(小于5M)多端桌面应用和手机应用仅需几分钟. https://ppofficial.netlify.app

项目定位  
PakePlus-iOS 是一个用于将网页、HTML、Vue、React 等前端项目快速打包为轻量级 iOS 应用的工具，帮助开发者几分钟内构建跨平台应用。

核心功能  
- 支持将网页或前端项目打包为 iOS 应用  
- 生成的 app 文件小于 5MB  
- 提供简单易用的命令行接口  
- 支持多种前端框架如 Vue 和 React  
- 可自定义应用名称、图标和配置

技术特点  
基于 Swift 开发，专注于轻量化和高效打包，结合 Tauri 技术实现跨平台支持，适合快速构建小型 iOS 应用。

适用场景  
- 快速将网站转为移动应用  
- 构建小型工具类 iOS 应用  
- 前端开发者快速测试和发布应用  
- 无需原生开发技能的轻量级项目  
- 需要快速迭代和部署的场景

推荐理由  
PakePlus-iOS 提供了简单高效的打包方式，适合需要快速将网页或前端项目转化为 iOS 应用的开发者，尤其适合轻量级应用和快速原型开发。
