# steipete/agent-rules

Rules and Knowledge to work better with agents such as Claude Code or Cursor

项目定位  
该项目为开发者提供与 AI 代理（如 Claude Code、Cursor）协作的规则和知识，帮助提升使用效率和效果。

核心功能  
- 提供与 AI 代理交互的最佳实践  
- 包含常见任务的指令模板  
- 优化提示词设计以提高响应质量  
- 指导如何有效利用代理进行代码生成与调试  
- 分享实际使用中的经验与技巧

技术特点  
项目以 Shell 脚本为主，专注于提供实用规则和指南，无复杂依赖，易于理解和应用，适合快速上手。

适用场景  
- 开发者在使用 AI 代理编写代码时参考  
- 团队统一 AI 工具使用规范  
- 提升 AI 代理在日常开发中的效率  
- 教育场景中教学 AI 工具使用方法  
- 优化 AI 代理在特定任务中的表现

推荐理由  
该项目为 AI 代理用户提供实用指导，帮助提升工作效率，尤其适合希望充分发挥 Claude Code、Cursor 等工具潜力的开发者。
