# CyberTimon/RapidRAW

A beautiful, non-destructive, and GPU-accelerated RAW image editor built with performance in mind.

项目定位  
RapidRAW 是一款基于 GPU 加速的非破坏性 RAW 图像编辑工具，专注于高性能图像处理与色彩校正。

核心功能  
- 非破坏性编辑  
- GPU 加速处理  
- 色彩校正与调色  
- 图层与蒙版支持  
- 支持多种 RAW 格式

技术特点  
使用 TypeScript 构建，结合 Rust 和 Tauri 实现高性能与跨平台能力，注重性能优化与用户体验。

适用场景  
- 摄影师后期处理  
- 图像处理专业人员  
- 需要高效 RAW 编辑的用户  
- 开发者研究图像处理技术  
- 个人照片美化与调整

推荐理由  
RapidRAW 凭借其高性能、非破坏性编辑和直观界面，适合需要快速处理 RAW 图像的用户，是图像处理领域的优秀选择。
