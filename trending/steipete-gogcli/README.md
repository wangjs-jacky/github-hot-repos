# steipete/gogcli

Google Suite CLI: Gmail, GCal, GDrive, GContacts.

项目定位  
gogcli 是一个用于管理 Google 服务的命令行工具，支持 Gmail、GCal、GDrive 和 GContacts，方便用户通过终端高效操作 Google 套件。

核心功能  
- 通过命令行发送和接收电子邮件  
- 管理日历事件和日程安排  
- 管理 Google Drive 文件和文件夹  
- 同步和管理联系人信息  

技术特点  
使用 Go 语言开发，具备良好的性能和跨平台支持，提供简洁的 CLI 接口，易于集成到自动化工作流中。

适用场景  
- 开发者在终端中快速处理邮件和日历  
- 自动化备份和同步 Google 数据  
- 企业用户集中管理 Google 服务  
- 需要无 GUI 环境下操作 Google 套件的场景  

推荐理由  
gogcli 提供了一种高效、便捷的方式管理 Google 服务，适合需要在命令行环境中操作 Gmail、日历、驱动器和联系人的用户。
