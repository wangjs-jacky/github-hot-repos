# helloianneo/ian-xiaohei-illustrations

中文小黑怪诞正文配图生成 Skill | 16:9 白底手绘 | 少量红橙蓝批注 | Codex Skill

项目定位  
一个基于 AI 的中文小黑怪插画生成工具，提供 16:9 白底手绘风格的配图，适用于文本内容的视觉化表达。

核心功能  
- 根据文本生成符合风格的插画  
- 支持 16:9 白底手绘画面  
- 提供红橙蓝三色批注样式  
- 适配中文语境的插画生成  
- 基于 Codex Skill 的智能生成能力

技术特点  
采用 AI 图像生成技术，结合特定风格与颜色标注，实现高效、高质量的插画创作，适合快速生成符合特定需求的视觉内容。

适用场景  
- 中文内容的图文搭配  
- 教育或教学材料的视觉辅助  
- 社交媒体内容配图  
- 个人创作灵感参考  
- 快速生成插画用于展示或演示

推荐理由  
适合需要快速生成中文风格插画的用户，尤其在教育、内容创作等领域有实用价值，具备良好的视觉风格和生成效率。
