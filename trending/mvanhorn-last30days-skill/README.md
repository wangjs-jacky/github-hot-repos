# mvanhorn/last30days-skill

AI agent skill that researches any topic across Reddit, X, YouTube, HN, Polymarket, and the web - then synthesizes a grounded summary

1. **项目定位**  
该项目是一个 AI 代理技能，用于跨平台研究任意主题，并生成基于事实的总结，解决用户快速获取多源信息的问题。

2. **核心功能**  
- 跨 Reddit、X、YouTube、HN、Polymarket 和网页搜索信息  
- 自动收集和整理数据  
- 生成结构化、可理解的摘要  
- 支持多种数据源的整合与分析  

3. **技术特点**  
使用 Python 构建，结合 API 调用和网络爬虫技术，实现多平台数据抓取与自然语言处理，具备高效的信息整合能力。

4. **适用场景**  
- 快速了解热点话题  
- 市场趋势分析  
- 学术研究辅助  
- 新闻事件背景调查  

5. **推荐理由**  
该工具能高效整合多源信息并生成清晰总结，适合需要快速掌握复杂主题的用户，具有实用性强和扩展性好的特点。
