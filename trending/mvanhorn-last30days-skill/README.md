# mvanhorn/last30days-skill

AI agent skill that researches any topic across Reddit, X, YouTube, HN, Polymarket, and the web - then synthesizes a grounded summary

1. **项目定位**  
该项目是一个AI代理技能，用于跨平台研究任意主题并生成基于事实的总结，解决信息碎片化和多源数据整合的问题。

2. **核心功能**  
- 跨平台搜索（Reddit、X、YouTube、HN、Polymarket、网页）  
- 自动摘要生成  
- 信息验证与整合  
- 支持用户自定义查询  

3. **技术特点**  
使用Python构建，结合自然语言处理和网络爬虫技术，实现多源信息采集与智能合成，具备良好的扩展性和实用性。

4. **适用场景**  
- 研究热点话题  
- 汇总社交媒体观点  
- 快速获取行业动态  
- 支持决策分析  
- 学术或市场调研  

5. **推荐理由**  
功能强大且实用，适合需要快速获取多平台信息并生成结构化摘要的用户，是信息整合领域的优秀工具。
