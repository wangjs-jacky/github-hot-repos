# unicity-astrid/handbook

How to work on Astrid: the polyrepo, the kernel-is-dumb law, the RFC trigger, contribution tiers, and the release process.

项目定位  
一个为 Astrid 项目提供开发指南和流程规范的文档仓库，帮助开发者理解其独特的工程实践和协作方式。

核心功能  
- 解释 polyrepo 架构与工作流程  
- 描述“kernel-is-dumb”原则  
- 指导 RFC 提交与评审机制  
- 明确贡献等级与权限划分  
- 说明发布流程与版本管理

技术特点  
该项目以文档形式呈现复杂工程理念，强调结构化和标准化的开发流程，适合团队协作与知识沉淀，具备良好的可读性和实用性。

适用场景  
- 新成员快速上手 Astrid 项目  
- 了解开源项目的协作机制  
- 学习如何制定和维护技术规范  
- 参考高效的代码管理策略  
- 研究大型项目的组织架构

推荐理由  
该手册为开发者提供了清晰的指导框架，有助于提升协作效率和代码质量，是理解和参与 Astrid 项目的重要资源。
