# vicinaehq/vicinae

A focused launcher for your desktop — native, fast, extensible

项目定位  
Vicinae 是一个专注于桌面的启动器，旨在为用户提供快速、高效、可扩展的本地应用启动体验。

核心功能  
- 快速启动应用程序和文件  
- 支持自定义插件扩展功能  
- 与系统深度集成，提升操作效率  
- 提供简洁直观的用户界面  

技术特点  
使用 C++ 开发，支持 C++23 标准，结合 Qt 框架实现跨平台功能，注重性能和稳定性。

适用场景  
- 高效办公环境中的快速应用启动  
- 开发者需要自定义工具链的场景  
- Linux 用户寻找轻量级启动器  
- PowerUser 追求高度定制化的工具  

推荐理由  
Vicinae 以高性能和可扩展性著称，适合追求效率和定制化的用户，是桌面启动器领域的优秀选择。
