# tonyantony300/dashbeam

Send files and folders anywhere in the world without storing in cloud - any size, any format, no accounts, no restrictions.

项目定位  
Dashbeam 是一个去中心化的文件传输工具，允许用户在不依赖云存储的情况下，安全、快速地传输任意大小和格式的文件。

核心功能  
- 点对点文件传输，无需云端存储  
- 支持大文件和任意格式传输  
- 采用加密技术保障隐私  
- 支持断点续传  
- 跨平台兼容性好  

技术特点  
使用 TypeScript 开发，结合 QUIC 协议、Hole-Punching 技术实现高效传输，支持加密与隐私保护，具备跨平台和终端兼容能力。

适用场景  
- 个人用户快速传输大文件  
- 企业内部安全数据交换  
- 需要隐私保护的文件共享  
- 网络环境受限时的文件传输  
- 无需注册或账户的即时传输需求

推荐理由  
Dashbeam 提供了高效、安全且无限制的文件传输体验，适合注重隐私和性能的用户，尤其适用于需要跨平台和无云存储的场景。
