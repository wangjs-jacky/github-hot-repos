# kyutai-labs/pocket-tts

A TTS that fits in your CPU (and pocket)

项目定位  
一个轻量级的文本转语音工具，可在 CPU 上运行，适合资源受限环境。

核心功能  
- 高质量文本转语音  
- 低内存占用  
- 支持多种语言  
- 快速推理速度  
- 简单易用的 API 接口

技术特点  
基于 Python 实现，优化了模型结构以适应 CPU 运行，兼顾性能与效率，适合嵌入式或本地部署。

适用场景  
- 移动设备上的语音助手  
- 嵌入式系统中的语音反馈  
- 本地化语音合成应用  
- 资源有限的服务器环境  
- 个人开发与测试项目

推荐理由  
轻量高效，无需 GPU 即可实现高质量语音合成，适合对计算资源敏感的场景，易于集成与使用。
