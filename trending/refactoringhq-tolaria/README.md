# refactoringhq/tolaria

Desktop app to manage markdown knowledge bases

**项目定位**  
Tolaria 是一款用于管理 Markdown 知识库的桌面应用程序，帮助用户高效组织和检索知识内容。

**核心功能**  
- 支持 Markdown 格式的笔记管理  
- 提供强大的搜索和标签功能  
- 本地存储与同步支持  
- 可自定义工作流和插件扩展  

**技术特点**  
使用 TypeScript 开发，基于 Electron 构建跨平台桌面应用，注重性能优化和用户体验，具备良好的可扩展性。

**适用场景**  
- 个人知识管理  
- 团队文档协作  
- 学术研究资料整理  
- 技术文档维护  
- 项目笔记系统搭建  

**推荐理由**  
Tolaria 为 Markdown 用户提供了功能丰富、界面友好的知识管理工具，适合需要结构化整理信息的开发者和研究人员。
