# astrid-runtime/sdk-js

JavaScript and TypeScript SDK for building Astrid capsules.

项目定位  
Astrid 运行时的 JavaScript 和 TypeScript SDK，用于构建 Astrid 小程序（capsules）。

核心功能  
- 提供构建和运行 Astrid 小程序的 API  
- 支持 TypeScript 语言开发  
- 实现与 Astrid 运行时的交互  
- 包含模块化架构设计  
- 提供错误处理和日志记录功能

技术特点  
使用 TypeScript 编写，具备良好的类型安全性和可维护性。模块化设计便于扩展和集成，适合构建轻量级的小程序应用。

适用场景  
- 开发基于 Astrid 平台的应用程序  
- 构建微服务或模块化组件  
- 快速原型开发和测试  
- 需要与 Astrid 运行时深度集成的项目  
- 前端或后端开发者需要统一开发体验

推荐理由  
为 Astrid 生态提供强大的开发工具支持，适合希望快速构建高效、可维护小程序的开发者。类型安全和模块化设计提升了开发效率和代码质量。
