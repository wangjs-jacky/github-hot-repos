# StarTrail-org/LEANN

[MLsys2026]: RAG on Everything with LEANN. Enjoy 97% storage savings while running a fast, accurate, and 100% private RAG application on your personal device.

项目定位  
LEANN 是一个基于 RAG（检索增强生成）技术的本地化 AI 应用框架，旨在通过向量搜索实现高效、私密且节省存储的问答系统。

核心功能  
- 本地化部署，无需依赖云端服务  
- 利用向量数据库提升检索效率  
- 支持多种 LLM 模型和框架集成  
- 提供隐私保护机制  
- 实现高达 97% 的存储空间节省

技术特点  
采用向量搜索与 RAG 技术结合，支持本地运行，利用 Faiss 等工具优化检索性能，兼顾速度与准确性，适合资源受限环境。

适用场景  
- 个人设备上的私密 AI 问答系统  
- 需要离线运行的 AI 应用  
- 对数据隐私要求高的企业场景  
- 低带宽或无网络环境下的信息检索  
- 需要高效存储管理的 AI 项目

推荐理由  
LEANN 提供了一个高效、私密且节省资源的 RAG 解决方案，适合希望在本地运行 AI 应用并保护数据隐私的开发者和用户。
