# ruvnet/RuView

π RuView: WiFi DensePose turns commodity WiFi signals into real-time human pose estimation, vital sign monitoring, and presence detection — all without a single pixel of video.

项目定位  
RuView 是一个利用 Wi-Fi 信号实现人体姿态估计、生命体征监测和存在检测的开源项目，无需视频即可完成对人体状态的实时分析。

核心功能  
- 实时人体姿态估计（DensePose）  
- 生命体征监测（如呼吸、心跳）  
- 非接触式存在检测  
- 使用普通 Wi-Fi 信号进行数据采集  

技术特点  
项目采用 Rust 编写，注重性能与安全性，结合机器学习与信号处理技术，实现从 Wi-Fi 数据中提取人体信息的创新方案。

适用场景  
- 智能家居中的无感监控  
- 医疗健康监测  
- 安防领域的非接触式检测  
- 研究机构的人体行为分析  

推荐理由  
RuView 在隐私保护和应用场景上具有显著优势，通过 Wi-Fi 实现人体感知，技术新颖且实用性强，适合对无视频监控有需求的开发者和研究者。
