# ruvnet/RuView

π RuView: WiFi DensePose turns commodity WiFi signals into real-time human pose estimation, vital sign monitoring, and presence detection — all without a single pixel of video.

项目定位  
RuView 是一个利用 WiFi 信号实现实时人体姿态估计、生命体征监测和存在检测的开源项目，无需视频即可完成人体感知。

核心功能  
- 实时人体姿态估计  
- 生命体征监测（如呼吸、心跳）  
- 非接触式存在检测  
- 基于 WiFi 信号的高精度传感  
- 支持多种设备和环境适应

技术特点  
使用 Rust 编写，结合机器学习与无线信号处理技术，实现高效、低延迟的人体感知，具备良好的跨平台兼容性与稳定性。

适用场景  
- 智能家居中的无感监控  
- 医疗健康监测  
- 安防与隐私保护场景  
- 无人值守环境的人体状态分析  
- 研究与教育领域的无线感知技术探索

推荐理由  
RuView 利用 WiFi 信号实现人体感知，突破传统视频监控限制，兼具隐私保护与高精度，适合对非接触式传感有需求的开发者与研究者。
