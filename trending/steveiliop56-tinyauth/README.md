# steveiliop56/tinyauth

The tiniest authentication and authorization server you have ever seen.

**项目定位**  
tinyauth 是一个极简的认证与授权服务器，旨在为开发者提供轻量级、易部署的认证解决方案。

**核心功能**  
- 支持 TOTP 和 2FA 认证  
- 提供 OpenID Connect (OIDC) 兼容接口  
- 可作为中间件集成到 Caddy 或 Nginx  
- 支持自托管部署  
- 简化用户登录和权限管理流程  

**技术特点**  
使用 Go 语言开发，具备高性能和低资源消耗特性，支持多种认证协议，易于集成和扩展，适合快速部署。

**适用场景**  
- 自托管身份验证服务  
- 需要轻量级认证系统的微服务架构  
- 开发者需要快速实现 SSO 功能  
- 企业内部系统安全增强  
- 构建小型 API 网关的认证层  

**推荐理由**  
tinyauth 以极小的体积和强大的功能，提供了高效的认证与授权方案，适合对性能和部署灵活性有要求的项目。
