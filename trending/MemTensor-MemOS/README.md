# MemTensor/MemOS

AI memory OS for LLM and Agent systems(moltbot,clawdbot,openclaw), enabling persistent Skill memory for cross-task skill reuse and evolution.

**项目定位**  
MemOS 是一个为大语言模型和智能体系统设计的 AI 内存操作系统，支持跨任务技能复用与演化，实现持久化技能记忆。

**核心功能**  
- 持久化技能记忆管理  
- 跨任务技能复用与演化  
- 记忆检索与调度机制  
- 支持 MoltBot、ClawDBot 等智能体系统  
- 基于 RAG 的记忆增强生成

**技术特点**  
基于 Python 构建，聚焦于智能体系统的长期记忆管理与高效检索，结合 RAG 技术提升模型推理能力，适用于多智能体协作场景。

**适用场景**  
- 多智能体协作系统开发  
- 需要长期记忆的 AI 应用  
- 个性化技能持续学习与优化  
- RAG（检索增强生成）系统构建  
- 智能客服与自动化流程优化

**推荐理由**  
MemOS 提供了面向 LLM 和智能体的先进内存管理方案，适合需要持续学习与跨任务复用技能的 AI 系统，具备良好的扩展性和实用性。
