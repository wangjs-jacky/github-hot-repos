# tonyantony300/alt-sendme

Send files and folders anywhere in the world without storing in cloud - any size, any format, no accounts, no restrictions.

1. **项目定位**  
该项目是一个去中心化的文件传输工具，允许用户在全球范围内发送文件和文件夹，无需依赖云存储，保障隐私和无限制传输。

2. **核心功能**  
- 支持任意大小和格式的文件传输  
- 无需账户，无需云存储  
- 基于 P2P 技术实现直接传输  
- 支持断点续传  
- 使用加密技术保障数据安全  

3. **技术特点**  
项目采用 TypeScript 开发，结合 P2P、Hole-Punching 和 QUIC 协议，实现高效、安全的文件传输，同时利用 Blake3 算法提升性能与安全性。

4. **适用场景**  
- 个人大文件快速传输  
- 企业敏感数据安全共享  
- 避免云服务限制的文件分享  
- 跨网络环境的稳定传输  
- 需要隐私保护的文件交换  

5. **推荐理由**  
该工具提供无限制、高隐私的文件传输方案，适合对数据安全和自由度有较高要求的用户，技术架构先进且开源可信赖。
