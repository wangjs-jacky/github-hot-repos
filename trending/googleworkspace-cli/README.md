# googleworkspace/cli

Google Workspace CLI — one command-line tool for Drive, Gmail, Calendar, Sheets, Docs, Chat, Admin, and more. Dynamically built from Google Discovery Service. Includes AI agent skills.

**项目定位**  
Google Workspace CLI 是一个集成了多个 Google 服务的命令行工具，旨在简化对 Google Drive、Gmail、Calendar 等服务的操作与自动化。

**核心功能**  
- 提供统一的命令行接口访问 Google Workspace 服务  
- 动态从 Google Discovery API 构建命令  
- 支持 AI 代理技能扩展  
- 支持 OAuth2 认证与权限管理  
- 支持自动化脚本与任务编排  

**技术特点**  
使用 Rust 编写，具备高性能和安全性，通过动态解析 Google API 描述文件实现灵活的功能扩展，支持多服务集成与 AI 技能扩展。

**适用场景**  
- 开发者快速构建 Google 服务自动化脚本  
- IT 管理员管理 Google Workspace 账户与资源  
- 数据分析师自动化处理 Google Sheets 和 Drive 数据  
- AI 工程师扩展 AI 代理功能  
- 自动化办公流程与任务调度  

**推荐理由**  
功能全面、性能优异，适合需要高效操作 Google Workspace 服务的开发者和管理员，同时支持 AI 扩展，具有良好的可扩展性和易用性。
