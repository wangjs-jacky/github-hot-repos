# francescopace/espectre

🛜 ESPectre 👻  - Motion detection system based on Wi-Fi spectre analysis (CSI), with Home Assistant integration.

项目定位  
ESPectre 是一个基于 Wi-Fi CSI 分析的运动检测系统，支持与 Home Assistant 集成，用于无接触式环境监测。

核心功能  
- 通过 Wi-Fi 信号变化检测人体运动  
- 支持 ESP32 设备进行本地数据采集  
- 提供 Home Assistant 插件实现智能联动  
- 实时分析 Wi-Fi 能量分布图（CSI）  
- 可配置灵敏度和检测区域

技术特点  
使用 Python 开发，结合 Wi-Fi CSI 数据处理技术，实现低功耗、高精度的非接触式运动检测，具备良好的可扩展性和集成能力。

适用场景  
- 智能家居中的无感安防  
- 独居老人或儿童的活动监测  
- 办公室或仓库的人员存在检测  
- 无需摄像头的隐私保护监控  
- DIY 爱好者的无线传感实验

推荐理由  
ESPectre 利用 Wi-Fi 信号实现高效运动检测，适合对隐私敏感或无法安装摄像头的场景，且与 Home Assistant 兼容性好，是智能家居领域的创新方案。
