# op7418/CodePilot

A multi-model AI agent desktop client — connect any AI provider, extend with MCP & skills, control from your phone. Built with Electron + Next.js.

项目定位  
CodePilot 是一个基于 Electron 和 Next.js 构建的多模型 AI 代理桌面客户端，允许用户连接不同 AI 提供商并扩展功能，提供跨平台的 AI 交互体验。

核心功能  
- 支持连接多个 AI 提供商（如 Anthropic、Claude 等）  
- 提供可扩展的 MCP 和技能模块  
- 支持通过手机控制 AI 代理  
- 基于 Web 技术构建的图形化界面  

技术特点  
使用 TypeScript 开发，结合 Electron 实现桌面应用，Next.js 提升前端性能，整体架构现代化且易于扩展。

适用场景  
- 开发者需要集成多种 AI 模型进行测试和使用  
- 用户希望在桌面端统一管理多个 AI 工具  
- 需要通过移动设备远程控制 AI 应用的场景  
- 需要高度定制化 AI 代理功能的用户  

推荐理由  
CodePilot 提供了灵活的 AI 代理解决方案，支持多模型接入与扩展，适合开发者和高级用户进行 AI 工具整合与个性化配置。
