# Kodezi/Chronos

Kodezi Chronos is a debugging-first language model that achieves state-of-the-art results on SWE-bench Lite (80.33%) and 67% real-world fix accuracy, over six times better than GPT-4. Built with Adaptive Graph-Guided Retrieval and Persistent Debug Memory. Model available Q1 2026 via Kodezi OS.

**项目定位**  
Kodezi/Chronos 是一个以调试为核心的语言模型，专注于代码修复和错误检测，旨在提升软件工程中的代码质量与调试效率。

**核心功能**  
- 高精度代码错误修复  
- 支持 SWE-bench Lite 和真实世界代码修复  
- 提供持续调试记忆功能  
- 基于自适应图引导检索技术  
- 用于开发人员的代码分析与理解  

**技术特点**  
Chronos 结合了自适应图引导检索和持久化调试记忆，实现了高效的代码修复能力，显著优于现有模型如 GPT-4。

**适用场景**  
- 软件开发过程中的代码调试  
- 自动化代码修复工具集成  
- 代码质量评估与测试  
- 开发者辅助工具  
- 研究人工智能在代码修复中的应用  

**推荐理由**  
Chronos 在代码修复任务中表现卓越，适合需要高效调试和代码修复能力的开发者和研究者，是未来代码辅助工具的重要方向。
