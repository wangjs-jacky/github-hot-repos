# freestylefly/awesome-gpt-image-2

Prompt as Code | GPT-Image2 工业级提示词引擎与模板库，370+ 个案例逆向工程，20+ 套工业级模板，并提炼出Skills，持续更新中

项目定位  
一个专注于 GPT-Image2 的提示词引擎与模板库，提供可复用的提示工程方案，提升 AI 图像生成效率。

核心功能  
- 370+ 逆向工程的图像提示案例  
- 20+ 工业级提示模板  
- 提炼出可复用的 Skills 模块  
- 支持提示作为代码进行管理与扩展  

技术特点  
基于 JavaScript 构建，强调提示词的结构化与模块化，便于集成到自动化工作流中，适合开发者快速调用与扩展。

适用场景  
- AI 图像生成任务自动化  
- 企业级提示词管理  
- 个性化图像生成需求定制  
- 提示工程教学与研究  
- 提升 AI 工具使用效率  

推荐理由  
提供丰富的提示模板和技能模块，帮助用户高效构建 AI 图像生成流程，适合开发者和 AI 爱好者提升工作效率。
