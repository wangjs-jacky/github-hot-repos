# Wei-Shaw/claude-relay-service

CRS-自建Claude Code镜像，一站式开源中转服务，让 Claude、OpenAI、Gemini、Droid 订阅统一接入，支持拼车共享，更高效分摊成本，原生工具无缝使用。

项目定位  
一个自建 Claude Code 镜像的中转服务，支持多平台 AI 接入与成本分摊，提供统一接口和共享使用方案。

核心功能  
- 支持 Claude、OpenAI、Gemini、Droid 等多种 AI 服务接入  
- 提供 API 中转，实现订阅共享与成本分摊  
- 支持原生工具无缝集成与使用  
- 提供一站式开源中转服务  

技术特点  
基于 JavaScript 构建，具备良好的跨平台兼容性，采用代理机制实现多 AI 平台统一接入，便于扩展和维护。

适用场景  
- 个人开发者需要多 AI 服务支持  
- 团队共享 AI 订阅以降低使用成本  
- 需要统一管理多个 AI API 的场景  
- 开发测试中需要灵活切换 AI 模型  

推荐理由  
提供多 AI 服务统一接入方案，降低使用门槛和成本，适合需要灵活调用多种 AI 模型的开发者和团队。
