# alam00000/bentopdf

A Privacy First PDF Toolkit

项目定位  
bentopdf 是一个以隐私为先的 PDF 工具包，提供多种 PDF 相关功能，旨在让用户在本地或自托管环境中安全处理 PDF 文件。

核心功能  
- PDF 转换（如 JPG 到 PDF）  
- PDF 编辑与注释  
- PDF 生成与查看  
- OCR 支持  
- 自托管部署  

技术特点  
使用 JavaScript 和 TypeScript 构建，支持 Docker 部署，注重隐私保护，适合需要本地化处理的用户。

适用场景  
- 企业内部 PDF 处理  
- 个人隐私敏感文件处理  
- 自托管 PDF 工具服务  
- 开发者集成 PDF 功能  
- 需要 OCR 或编辑功能的 PDF 场景  

推荐理由  
bentopdf 提供全面的 PDF 工具集，强调隐私和自托管，适合对数据安全有高要求的用户和开发者。
