# datalab-to/chandra

OCR model that handles complex tables, forms, handwriting with full layout.

项目定位  
Chandra 是一个用于复杂表格、表单和手写内容的 OCR 模型，专注于保留原始布局的文本识别。

核心功能  
- 准确识别复杂表格结构  
- 支持手写文字识别  
- 保留文档原始布局信息  
- 处理多语言和不同格式的表单  

技术特点  
基于 Python 构建，结合深度学习与布局分析技术，实现高精度的 OCR 处理，特别优化了对复杂排版的适应能力。

适用场景  
- 表格数据提取与转换  
- 手写表单数字化  
- 电子文档自动化处理  
- 学术论文或报告的 OCR 转换  

推荐理由  
Chandra 在处理复杂布局和手写内容方面表现出色，适合需要高精度 OCR 的企业级应用和研究场景。
