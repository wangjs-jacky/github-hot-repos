# trycua/cua

Open-source infrastructure for Computer-Use Agents. Sandboxes, SDKs, and benchmarks to train and evaluate AI agents that can control full desktops (macOS, Linux, Windows).

项目定位  
Cua 是一个用于训练和评估能够控制完整桌面的 AI 代理的开源基础设施，支持 macOS、Linux 和 Windows 平台。

核心功能  
- 提供沙盒环境用于 AI 代理测试  
- 包含 SDK 和基准测试工具  
- 支持跨平台桌面自动化  
- 集成容器化与虚拟化技术  
- 支持多种编程语言和框架

技术特点  
该项目采用 Python 开发，结合容器化、虚拟化和跨平台技术，为 AI 代理提供稳定、可扩展的运行环境，适合研究和开发桌面级 AI 自动化应用。

适用场景  
- AI 代理训练与评估  
- 桌面自动化研究  
- 跨平台软件测试  
- 开发者工具链构建  
- 参与 Hacktoberfest 等开源活动

推荐理由  
Cua 提供了全面的桌面 AI 代理开发工具和环境，适合研究人员和开发者快速构建和测试跨平台的自动化应用，具有良好的社区支持和活跃的开发节奏。
