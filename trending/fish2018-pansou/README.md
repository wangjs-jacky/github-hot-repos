# fish2018/pansou

PanSou是一款高性能的网盘资源搜索API服务，支持TG频道和插件搜索。系统设计以性能和可扩展性为核心，支持多频道多插件并发搜索、结果智能排序和网盘类型分类。docker集成前后端，一键启动，开箱即用。 https://so.252035.xyz/

项目定位  
PanSou 是一款高性能的网盘资源搜索 API 服务，支持 Telegram 频道和插件搜索，旨在提供快速、智能的网盘资源检索解决方案。

核心功能  
- 支持多频道、多插件并发搜索  
- 智能排序与网盘类型分类  
- Docker 集成，一键启动  
- 提供 RESTful API 接口  
- 支持 TG 频道和插件资源搜索  

技术特点  
采用 Go 语言开发，注重性能与可扩展性，支持高并发搜索，具备良好的系统架构设计，便于部署和维护。

适用场景  
- 个人用户快速查找网盘资源  
- 开发者集成到自有应用中  
- 网盘资源聚合平台建设  
- Telegram 频道内容检索  
- 多源资源统一搜索需求  

推荐理由  
高性能、易部署、支持多源搜索，适合需要快速构建网盘资源检索系统的开发者和用户。
