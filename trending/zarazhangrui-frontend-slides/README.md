# zarazhangrui/frontend-slides

Create beautiful slides on the web using Claude's frontend skills

**项目定位**  
一个基于前端技术创建精美网页幻灯片的工具，帮助开发者快速生成美观的展示内容。

**核心功能**  
- 提供简洁的 Markdown 语法创建幻灯片  
- 支持自定义主题和样式  
- 实时预览与编辑功能  
- 轻量级且易于集成到网页中  

**技术特点**  
使用 Shell 脚本实现自动化构建流程，结合前端技术打造高效、易用的幻灯片生成方案。

**适用场景**  
- 技术分享与演示  
- 产品介绍与展示  
- 在线课程与教学材料  
- 个人作品集展示  

**推荐理由**  
适合需要快速制作专业级幻灯片的开发者，功能简洁、易于上手，能有效提升展示效率。
