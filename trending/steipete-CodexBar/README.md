# steipete/CodexBar

Show usage stats for OpenAI Codex and Claude Code, without having to login.

项目定位  
CodexBar 是一个用于展示 OpenAI Codex 和 Claude Code 使用统计数据的工具，无需登录即可查看使用情况。

核心功能  
- 显示 OpenAI Codex 的使用统计信息  
- 提供 Claude Code 的使用数据  
- 无需用户登录即可访问  
- 实时更新使用数据  
- 支持 Swift 开发语言

技术特点  
该项目使用 Swift 编写，专注于提供无登录的 API 数据访问，具备简洁高效的架构，适合开发者快速集成和使用。

适用场景  
- 开发者查看 AI 模型使用情况  
- 企业监控 Codex 和 Claude 的使用量  
- 教育机构评估 AI 工具使用频率  
- 个人用户了解代码生成效率  
- 研究人员分析 AI 工具的使用趋势

推荐理由  
CodexBar 提供了便捷的 AI 使用数据查看方式，无需登录，适合开发者和研究人员快速获取 Codex 和 Claude 的使用统计信息。
