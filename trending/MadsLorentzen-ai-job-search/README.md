# MadsLorentzen/ai-job-search

AI-powered job application framework built on Claude Code. Fork it, fill in your profile, and let Claude evaluate jobs, tailor CVs, write cover letters, and prepare you for interviews.

项目定位  
一个基于 Claude Code 的 AI 驱动的求职框架，帮助用户优化简历、撰写求职信并准备面试。

核心功能  
- 根据用户资料自动优化简历  
- 使用 AI 撰写定制化求职信  
- 提供面试准备建议  
- 评估适合的职位机会  
- 支持个性化配置和更新

技术特点  
使用 TypeScript 构建，集成 Claude AI 技术，提供高效的代码结构和可扩展性，专注于提升求职效率。

适用场景  
- 求职者快速优化简历和求职信  
- 企业招聘人员筛选候选人  
- 个人职业发展与技能提升  
- 自动化求职流程管理  
- 个性化求职策略制定

推荐理由  
结合 AI 技术与实用工具，简化求职流程，提高成功率，适合希望高效求职的用户。
