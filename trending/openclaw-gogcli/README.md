# openclaw/gogcli

Google Suite CLI: Gmail, GCal, GDrive, GContacts.

**项目定位**  
一个用于管理 Google 服务的命令行工具，支持 Gmail、GCal、GDrive 和 GContacts，简化用户对 Google 套件的日常操作。

**核心功能**  
- 发送和接收 Gmail 邮件  
- 管理 Google 日历事件  
- 管理 Google Drive 文件  
- 操作 Google 联系人  

**技术特点**  
使用 Go 语言开发，提供高效的 CLI 工具，集成 Google API，支持 OAuth 认证，具备良好的可扩展性。

**适用场景**  
- 自动化邮件处理任务  
- 通过命令行管理日历  
- 快速访问和操作云存储文件  
- 管理联系人信息  

**推荐理由**  
适合开发者和高级用户，提供便捷的 Google 服务 CLI 管理方式，提升工作效率，代码结构清晰，易于维护。
