# justlovemaki/AIClient2API

Simulates Gemini CLI, Antigravity, Codex, Grok, and Kiro client requests, compatible with the OpenAI API. It supports thousands of Gemini model requests per day and offers free use of the built-in Claude model in Kiro. Easily connect to any client via the API, making AI development more efficient!

项目定位  
AIClient2API 是一个模拟多种 AI 客户端请求的工具，兼容 OpenAI API，旨在简化 AI 开发流程，提升效率。

核心功能  
- 模拟 Gemini、Antigravity、Codex、Grok、Kiro 等客户端请求  
- 支持每日数千次 Gemini 模型请求  
- 提供免费内置 Claude 模型使用  
- 通过 API 轻松连接任意客户端  

技术特点  
基于 JavaScript 实现，兼容 OpenAI API 协议，支持多模型模拟和高并发请求，适合快速集成到现有系统中。

适用场景  
- 快速测试不同 AI 客户端行为  
- 开发过程中模拟真实 API 请求  
- 降低对特定 AI 服务的依赖  
- 用于教学或演示目的  

推荐理由  
提供多模型支持与高并发能力，简化 AI 开发流程，适合需要灵活调用多种 AI 服务的开发者使用。
