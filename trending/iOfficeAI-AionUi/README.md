# iOfficeAI/AionUi

Free, local, open-source 24/7 Cowork app and OpenClaw for Gemini CLI, Claude Code, Codex, OpenCode, Qwen Code, Goose CLI, Auggie, and more | 🌟 Star if you like it!

项目定位  
iOfficeAI/AionUi 是一个本地运行的开源协作工具和 AI 代理平台，支持多种大模型 CLI 工具，旨在提升开发效率与协作体验。

核心功能  
- 支持 Gemini CLI、Claude Code、Codex 等多种 AI 模型  
- 提供本地化的 Web UI 界面进行交互  
- 集成 OpenClaw 用于多模型管理和调用  
- 支持代码生成与智能问答  
- 实现 24/7 协作办公功能  

技术特点  
基于 TypeScript 开发，具备良好的跨平台兼容性；采用模块化设计，便于扩展与集成多种 AI 模型，同时注重本地化部署与隐私保护。

适用场景  
- 开发者日常代码生成与调试  
- 团队协作与远程办公  
- 多 AI 模型统一管理与调用  
- 企业级私有化 AI 服务部署  
- 个人开发者快速搭建 AI 工具链  

推荐理由  
开源且支持多种主流 AI 模型，提供本地化部署方案，适合需要高效协作与灵活 AI 工具的开发者与团队使用。
