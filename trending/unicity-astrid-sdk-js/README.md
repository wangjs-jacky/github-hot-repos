# unicity-astrid/sdk-js

JavaScript / TypeScript SDK for building Astrid OS capsules. Companion to unicity-astrid/sdk-rust.

项目定位  
JavaScript/TypeScript SDK 用于构建 Astrid OS 的胶囊应用，为开发者提供与 Astrid OS 交互的工具和接口。

核心功能  
- 提供与 Astrid OS 交互的 API 接口  
- 支持开发和部署自定义胶囊应用  
- 提供类型定义和模块化结构  
- 与 Rust 版 SDK 协同工作  
- 简化开发流程和调试体验

技术特点  
基于 TypeScript 构建，具备良好的类型安全性和可维护性，支持模块化开发，与 Rust SDK 配合使用，适合构建高性能的分布式应用。

适用场景  
- 开发基于 Astrid OS 的自定义服务  
- 构建去中心化应用（DApp）  
- 集成到现有 JavaScript/TypeScript 项目中  
- 为跨平台应用提供后端支持  
- 构建微服务架构的组件

推荐理由  
适合希望在 Astrid OS 上构建高效、可扩展应用的开发者，提供清晰的 API 和良好的类型支持，便于快速开发和维护。
