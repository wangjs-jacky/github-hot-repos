# Narcooo/inkos

Autonomous novel writing AI Agent — agents write, audit, and revise novels with human review gates

项目定位  
Narcooo/inkos 是一个用于自动生成小说的 AI 代理工具，旨在通过自动化写作、审核和修订流程，提升小说创作效率并确保质量。

核心功能  
- 自动化小说生成  
- 写作内容审核与修订  
- 人类审核机制介入  
- 支持中文小说创作  
- 命令行界面（CLI）操作

技术特点  
基于 TypeScript 构建，结合 AI 技术实现小说生成与编辑流程自动化，具备良好的可扩展性和用户交互体验。

适用场景  
- 作家辅助创作  
- 自动化内容生产  
- 中文小说生成  
- AI 与人类协作写作  
- 创意写作工具开发

推荐理由  
Narcooo/inkos 提供了一种创新的小说创作方式，结合 AI 与人工审核，适合需要高效生成高质量内容的创作者。
