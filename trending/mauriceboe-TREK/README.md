# mauriceboe/TREK

A self-hosted travel/trip planner with real-time collaboration, interactive maps, PWA support, SSO, budgets, packing lists, and more.

项目定位  
TREK 是一个自托管的旅行规划工具，支持实时协作、互动地图和多种旅行管理功能，帮助用户高效规划行程。

核心功能  
- 实时协作编辑旅行计划  
- 交互式地图与路线规划  
- 预算跟踪与打包清单管理  
- SSO（单点登录）支持  
- PWA（渐进式网页应用）支持

技术特点  
使用 TypeScript 构建，具备良好的类型安全和可维护性，支持自托管部署，结合现代 Web 技术实现高性能和跨平台体验。

适用场景  
- 个人旅行计划制定  
- 团队或家庭旅行协作  
- 自托管企业级旅行管理系统  
- 需要离线访问的旅行应用  
- 需要集成 SSO 的组织使用

推荐理由  
TREK 提供全面的旅行规划功能，支持实时协作和自托管，适合需要定制化和数据隐私的用户，是开源社区中优秀的旅行管理工具。
