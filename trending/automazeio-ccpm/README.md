# automazeio/ccpm

Project management skill system for Agents that uses GitHub Issues and Git worktrees for parallel agent execution.

项目定位  
一个基于 GitHub Issues 和 Git worktrees 的 AI 代理项目管理工具，用于高效执行和管理多个并行任务。

核心功能  
- 利用 GitHub Issues 进行任务管理和分配  
- 通过 Git worktrees 实现多代理并行执行  
- 自动化任务流程和状态跟踪  
- 支持 AI 编码与协作开发  
- 提供项目进度可视化和监控

技术特点  
使用 Shell 脚本实现轻量级自动化，结合 GitHub API 和 Git 工具链，构建高效的 AI 代理工作流系统。

适用场景  
- AI 代理开发与测试环境搭建  
- 多任务并行处理与管理  
- 自动化代码生成与部署流程  
- 团队协作与项目进度跟踪  
- 开发者工具链集成与优化

推荐理由  
适合需要管理多个 AI 代理任务的开发者，提供简单易用的 GitHub 集成方案，提升开发效率和任务协调能力。
