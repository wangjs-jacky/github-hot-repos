# Imbad0202/academic-research-skills

Academic Research Skills for Claude Code: research → write → review → revise → finalize

项目定位  
一个面向学术研究的工具包，帮助用户通过 Claude Code 实现从研究到最终定稿的全流程自动化。

核心功能  
- 提供学术研究流程的自动化支持  
- 支持文献综述与写作生成  
- 包含同行评审与修改建议功能  
- 集成提示工程优化研究流程  
- 适用于 AI 辅助的学术写作场景

技术特点  
基于 Python 开发，结合 AI 技术实现学术流程自动化，专注于提升研究效率和写作质量。

适用场景  
- 学术论文撰写与修改  
- 文献综述自动化生成  
- AI 辅助的科研工作流构建  
- 研究人员的提示工程实践  
- 学术写作与同行评审支持

推荐理由  
为研究人员提供高效、结构化的学术写作与研究工具，尤其适合结合 AI 工具如 Claude Code 进行创新性研究。
