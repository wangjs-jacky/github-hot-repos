# toon-format/toon

🎒 Token-Oriented Object Notation (TOON) – Compact, human-readable, schema-aware JSON for LLM prompts. Spec, benchmarks, TypeScript SDK.

项目定位  
TOON 是一种面向大语言模型（LLM）的紧凑、可读性强且具有模式感知能力的 JSON 格式，旨在优化 LLM 提示的结构化表示。

核心功能  
- 提供紧凑的 JSON 表达方式  
- 支持 schema-aware 的数据序列化  
- 优化 Token 化效率  
- 适用于 LLM 输入输出格式标准化  
- 提供 TypeScript SDK 和基准测试工具

技术特点  
基于 TypeScript 实现，强调高效序列化与反序列化，支持 Schema 验证，提升 LLM 交互效率和可维护性。

适用场景  
- LLM 提示工程优化  
- 数据格式标准化处理  
- 模型训练数据预处理  
- 多模型间的数据交换  
- 构建高效的提示模板系统

推荐理由  
TOON 为 LLM 相关开发提供了高效、可读性强且结构清晰的数据格式，适合需要优化提示和数据处理流程的开发者和研究者。
