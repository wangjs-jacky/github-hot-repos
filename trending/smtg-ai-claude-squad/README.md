# smtg-ai/claude-squad

Manage multiple AI terminal agents like Claude Code, Codex, OpenCode, and Amp.

项目定位  
一个用于管理多个 AI 终端代理的工具，支持 Claude Code、Codex、OpenCode 等 AI 编程助手，提升开发效率。

核心功能  
- 支持多 AI 代理统一管理  
- 提供命令行界面操作  
- 自动切换不同 AI 模型  
- 集成代码生成与解释功能  
- 支持自定义配置和扩展

技术特点  
使用 Go 语言开发，注重性能与稳定性，提供轻量级 CLI 工具，便于集成到开发流程中。

适用场景  
- 开发者快速调用 AI 编程助手  
- 多模型协作开发环境  
- 代码生成与调试辅助  
- 教学与研究中的 AI 工具演示  
- 提升个人或团队编码效率

推荐理由  
适合需要高效利用多种 AI 编程工具的开发者，简化多模型操作流程，提升整体开发体验。
