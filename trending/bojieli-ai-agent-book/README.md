# bojieli/ai-agent-book

《深入理解 AI Agent：设计原理与工程实践》（李博杰 著）开源主仓库：全书正文、编译版 PDF 与按章配套代码

项目定位  
一本关于 AI Agent 的开源书籍，涵盖设计原理与工程实践，提供正文、PDF 与配套代码。

核心功能  
- 提供 AI Agent 相关的完整书籍内容  
- 包含各章节配套的 Python 代码  
- 支持多模态与强化学习等高级主题  
- 提供编译版 PDF 与源码同步更新  

技术特点  
以 Python 为主，融合 agent、RAG、多智能体等前沿技术，适合深入学习 AI Agent 实现与应用。

适用场景  
- 学习 AI Agent 设计与开发  
- 研究 LLM 与多智能体系统  
- 实践 RAG 与上下文工程  
- 教学与科研参考  

推荐理由  
适合对 AI Agent 感兴趣的开发者与研究者，结合理论与实践，内容全面且持续更新。
