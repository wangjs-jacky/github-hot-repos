# permissionlesstech/bitchat-android

bluetooth mesh chat, IRC vibes

项目定位  
一个基于蓝牙 Mesh 的聊天应用，提供类似 IRC 的实时通信体验，适用于点对点或小型网络中的消息传递。

核心功能  
- 蓝牙 Mesh 网络通信  
- 实时消息发送与接收  
- 简洁的聊天界面设计  
- 支持多设备连接与消息同步  

技术特点  
使用 Kotlin 开发，专注于蓝牙 Mesh 协议实现，具备良好的跨设备兼容性和低延迟通信能力。

适用场景  
- 小型局域网内的即时通讯  
- 无互联网环境下的设备间通信  
- 需要低功耗、高可靠性的物联网场景  
- 个人或小团队的快速原型开发  

推荐理由  
适合对蓝牙通信和实时聊天感兴趣的技术爱好者，提供了一个轻量级且易于扩展的 Bluetooth Mesh 项目参考。
