# kepano/defuddle

Get the main content of any page as Markdown.

项目定位  
Defuddle 是一个命令行工具，用于将网页内容转换为 Markdown 格式，方便用户提取和整理网页中的主要信息。

核心功能  
- 将任意网页内容转换为 Markdown 格式  
- 提取网页中的主要文本内容  
- 支持通过命令行调用  
- 可自定义解析规则  
- 适用于快速获取网页结构化数据

技术特点  
使用 HTML 解析技术，结合 Markdown 转换逻辑，提供轻量级的 CLI 工具，适合需要处理网页内容的开发者或内容创作者。

适用场景  
- 快速提取网页正文内容  
- 生成可编辑的 Markdown 文档  
- 内容整理与归档  
- 自动化网页数据采集  
- 用于知识管理或笔记工具集成

推荐理由  
Defuddle 简洁高效，适合需要将网页内容快速转换为 Markdown 的用户，尤其适合开发者和内容创作者提升工作效率。
