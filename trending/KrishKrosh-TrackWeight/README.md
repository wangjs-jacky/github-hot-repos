# KrishKrosh/TrackWeight

Use your Mac trackpad as a weighing scale

项目定位  
TrackWeight 是一款利用 Mac 轨迹板模拟称重设备的工具，通过触控板压力感知实现简易称重功能。

核心功能  
- 通过 Mac 轨迹板检测压力变化  
- 实现物体重量的估算与显示  
- 支持用户自定义校准设置  
- 提供直观的图形化界面  
- 可记录和导出称重数据

技术特点  
使用 Swift 开发，结合 macOS 的触控板 API 实现压力感应，具备良好的跨版本兼容性和简洁的用户交互设计。

适用场景  
- 家庭日常物品称重  
- 小型实验测量  
- 创意项目中的压力感应应用  
- 无传统秤时的临时替代方案  
- 科学教育演示

推荐理由  
TrackWeight 创新性地利用现有硬件实现称重功能，适合需要便捷、低成本解决方案的用户，具有实用性和趣味性。
