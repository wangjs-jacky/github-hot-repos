# getagentseal/codeburn

See where your AI coding tokens go. Interactive TUI dashboard for Claude Code, Codex, and Cursor cost observability.

1. **项目定位**  
该项目是一个交互式终端用户界面（TUI）工具，用于跟踪和可视化 AI 编码工具（如 Claude Code、Codex 和 Cursor）的 token 使用情况，帮助开发者更好地了解 AI 编码成本。

2. **核心功能**  
- 实时监控 AI 编码工具的 token 使用情况  
- 支持多种 AI 编码平台的成本追踪  
- 提供交互式终端界面，方便查看详细数据  
- 记录并展示每次请求的 token 消耗情况  

3. **技术特点**  
项目使用 TypeScript 开发，具备良好的类型安全性和可维护性。基于终端 UI 构建，适合开发者在命令行环境中高效使用，同时支持多平台运行。

4. **适用场景**  
- 开发者需要优化 AI 编码工具的使用成本  
- 团队希望统一管理多个 AI 编码平台的 token 消耗  
- 用于教学或演示 AI 编码工具的 token 使用情况  
- 个人开发者监控自己的 AI 编码支出  

5. **推荐理由**  
该工具为 AI 编码用户提供直观的成本分析方式，有助于提升开发效率并控制开支，尤其适合频繁使用 AI 编码工具的开发者群体。
