# PicoTrex/Awesome-Nano-Banana-images

A curated collection of fun and creative examples generated with Nano Banana & Nano Banana Pro🍌, Gemini-2.5-flash-image based model. We also release Nano-consistent-150K openly to support the community's development of image generation and unified models(click to website to see our blog)

项目定位  
一个由 Nano Banana 和 Gemini-2.5-flash-image 模型生成的创意图像集合，旨在为图像生成和统一模型开发提供支持。

核心功能  
- 提供丰富的图像示例  
- 支持图像生成技术研究  
- 包含开源数据集 Nano-consistent-150K  
- 促进社区模型开发  
- 鼓励创意和趣味性内容生成

技术特点  
基于 Gemini-2.5-flash-image 模型生成图像，结合 Nano Banana 系列技术，具备高效且高质量的图像生成能力，适用于多种图像任务。

适用场景  
- 图像生成模型训练  
- 创意设计参考  
- AI 技术研究与实验  
- 社区开发者资源  
- 趣味图像内容创作

推荐理由  
该项目提供了高质量的图像数据集和创意示例，适合研究人员、开发者及创作者使用，有助于推动图像生成技术的发展。
