# manaflow-ai/cmux

Ghostty-based macOS terminal with vertical tabs and notifications for AI coding agents

项目定位  
一个基于 Ghostty 的 macOS 终端应用，提供垂直标签和 AI 编程助手通知功能，提升开发者工作效率。

核心功能  
- 支持垂直标签页管理多个终端会话  
- 集成 AI 编程助手（如 Claude、Codex、Gemini）的通知系统  
- 提供更直观的终端界面和交互体验  
- 支持 tmux 风格的多窗口操作  

技术特点  
使用 Swift 开发，结合 macOS 系统特性优化性能与用户体验，支持多种 AI 编程工具集成，具备良好的扩展性。

适用场景  
- 开发者日常代码编写与调试  
- 多任务并行处理时的终端管理  
- 需要实时 AI 代码建议的编程环境  
- 对终端界面有较高要求的用户  

推荐理由  
为 macOS 用户提供一个高效、美观且功能强大的终端替代方案，尤其适合需要与 AI 编程工具协作的开发者。
