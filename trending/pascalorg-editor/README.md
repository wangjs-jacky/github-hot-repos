# pascalorg/editor

Create and share 3D architectural projects.

项目定位  
Pascal.org/editor 是一个用于创建和分享 3D 建筑项目的协作平台，旨在简化建筑设计师的数字化工作流程。

核心功能  
- 支持多人实时协作编辑 3D 建筑模型  
- 提供直观的界面进行建筑结构设计  
- 集成版本控制与项目共享功能  
- 支持多种建筑格式导入导出  
- 实时预览与渲染 3D 模型  

技术特点  
基于 TypeScript 构建，采用现代前端技术栈，支持跨平台使用，具备高性能的 3D 渲染能力和稳定的协作机制。

适用场景  
- 建筑设计团队协作  
- 教育机构教学演示  
- 客户与设计师远程沟通  
- 建筑项目展示与分享  
- 虚拟现实建筑体验开发  

推荐理由  
Pascal.org/editor 提供了强大的 3D 建筑设计与协作功能，适合建筑师、设计师及教育工作者高效创作与分享建筑项目。
