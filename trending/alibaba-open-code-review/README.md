# alibaba/open-code-review

Open-source & free — Battle-tested at Alibaba's scale. Hybrid architecture code review tool: deterministic pipelines + LLM Agent, precise line-level comments, built-in fine-tuned ruleset (NPE, thread-safety, XSS, SQL injection), OpenAI & Anthropic compatible.

项目定位  
一款开源免费的代码审查工具，专为大规模代码管理设计，结合确定性流水线与 LLM 代理，提供精准的代码审查服务。

核心功能  
- 支持线级别代码评论  
- 内置细调规则集（NPE、线程安全、XSS、SQL 注入等）  
- 兼容 OpenAI 和 Anthropic 的 LLM 服务  
- 混合架构：确定性流水线 + LLM 代理  

技术特点  
采用 Go 语言开发，具备高性能和可扩展性，结合传统规则引擎与 AI 代理，实现高效、精准的代码审查。

适用场景  
- 大型企业代码质量管控  
- 开源项目贡献者审查代码  
- 需要自动化代码审核的开发团队  
- 提升代码安全性与规范性  

推荐理由  
结合 AI 与传统规则引擎，提供高精度、自动化的代码审查，适合需要高质量代码管理的团队和开发者。
