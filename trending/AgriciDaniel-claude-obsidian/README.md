# AgriciDaniel/claude-obsidian

Self-organizing AI second brain for Obsidian + Claude Code. Drop any source and Claude reads, links, and files it into one connected knowledge graph of plain Markdown you own. AI note-taking, personal knowledge management (PKM), and an open-source Notion alternative. Based on Karpathy's LLM Wiki pattern.

**项目定位**  
一个基于 Obsidian 和 Claude 的 AI 第二大脑系统，用于知识管理与智能笔记，帮助用户构建个人知识图谱。

**核心功能**  
- 自动读取并链接外部资料  
- 生成结构化 Markdown 知识图谱  
- 支持 AI 笔记与知识管理  
- 集成 Claude 模型进行智能处理  
- 提供开源 Notion 替代方案  

**技术特点**  
使用 Python 实现 AI 驱动的知识组织，结合 Obsidian 插件架构与 Claude API，遵循 Karpathy 的 LLM Wiki 模式，强调开放性和可扩展性。

**适用场景**  
- 个人知识管理与学习记录  
- 研究人员整理文献与数据  
- 开发者构建代码知识库  
- 团队协作知识共享  
- 替代 Notion 进行轻量级知识管理  

**推荐理由**  
结合 AI 与 Obsidian 强大的笔记功能，提供高效、灵活的个人知识管理系统，适合注重隐私和自主性的用户。
