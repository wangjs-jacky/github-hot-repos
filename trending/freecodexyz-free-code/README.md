# freecodexyz/free-code

The free build of Claude Code. All telemetry removed, security-prompt guardrails stripped, all experimental features enabled.

项目定位  
一个去除了所有监控和安全限制的 Claude Code 开源版本，专注于提供无约束的代码生成体验。

核心功能  
- 提供类似 Claude Code 的代码生成能力  
- 移除所有用户数据追踪和安全限制  
- 启用所有实验性功能  
- 支持多语言代码生成  

技术特点  
基于 TypeScript 构建，注重性能与可扩展性，通过移除安全机制实现更自由的使用体验，适合开发者进行深度定制和研究。

适用场景  
- 开发者研究大模型代码生成能力  
- 需要无限制代码生成的实验环境  
- 企业内部定制化 AI 编程工具开发  
- 教学和开源社区研究  

推荐理由  
适合需要无限制代码生成和深度定制的开发者，去除安全限制后提供了更高的灵活性和研究价值。
