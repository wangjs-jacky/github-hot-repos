# eigent-ai/eigent

Eigent: The Open Source Cowork Desktop to Unlock Your Exceptional Productivity. Local and Free Alternative to Claude Cowork.

**项目定位**  
Eigent 是一个开源的桌面协作工具，旨在替代 Claude Cowork，提供本地化、免费的高效生产力解决方案。

**核心功能**  
- 多代理系统协作  
- 本地运行，保障数据隐私  
- 支持自定义工作流和技能扩展  
- 提供与 Claude Cowork 类似的协作体验  

**技术特点**  
使用 TypeScript 构建，具备良好的可扩展性和跨平台兼容性，专注于构建高效的多代理协作系统。

**适用场景**  
- 个人开发者日常任务管理  
- 团队协作与项目开发  
- 需要本地化部署的敏感数据处理  
- 替代商业协作工具以降低成本  

**推荐理由**  
Eigent 提供了一个免费且开源的替代方案，适合追求隐私和自主控制的用户，同时具备强大的多代理协作能力。
