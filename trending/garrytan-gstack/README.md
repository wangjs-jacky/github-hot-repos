# garrytan/gstack

Use Garry Tan's exact Claude Code setup: 15 opinionated tools that serve as CEO, Designer, Eng Manager, Release Manager, Doc Engineer, and QA

项目定位  
gstack 是一个基于 Garry Tan 的 Claude Code 设计的开发工具集合，提供 15 个经过验证的工具，帮助开发者高效管理产品、设计、工程等多角色任务。

核心功能  
- 提供 CEO、设计师、工程师、发布经理等角色的工具支持  
- 自动化代码生成与文档编写  
- 集成测试与质量保障流程  
- 支持快速迭代与版本发布  
- 优化团队协作与项目管理

技术特点  
使用 TypeScript 构建，强调模块化和可扩展性，结合现代开发实践，提供一致的开发体验和高效的工具链集成。

适用场景  
- 个人开发者快速构建项目  
- 小型团队提升协作效率  
- 产品负责人管理多角色任务  
- 开发者自动化日常流程  
- 跨职能团队统一工作流

推荐理由  
gstack 提供一套经过验证的工具组合，适合希望提升效率、规范流程并减少重复劳动的开发者和团队。
