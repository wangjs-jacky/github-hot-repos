# Anionex/banana-slides

一个基于nano banana pro🍌的原生AI PPT生成应用，迈向真正的＂Vibe PPT＂; 支持上传任意模板图片；上传任意素材&智能解析；一句话/大纲/页面描述自动生成PPT；口头修改指定区域、一键导出可编辑ppt - An AI-native slides generator based on nano banana pro🍌

项目定位  
一个基于 nano banana pro 的 AI 原生 PPT 生成应用，支持模板上传、素材解析与智能生成，实现高效可编辑的幻灯片制作。

核心功能  
- 支持上传任意模板图片  
- 上传素材并智能解析  
- 通过一句话/大纲/页面描述自动生成 PPT  
- 口头修改指定区域  
- 一键导出可编辑 PPT

技术特点  
采用 TypeScript 开发，结合 AI 技术与文档识别能力，实现高效的 PPT 生成与编辑功能，适配 nano banana pro 硬件平台。

适用场景  
- 快速制作会议或汇报 PPT  
- 教学材料与课程展示  
- 商业提案与产品介绍  
- 个人创意内容制作  
- 团队协作与内容共享

推荐理由  
该项目结合 AI 与硬件优势，提供高效、智能的 PPT 制作体验，适合需要快速生成高质量演示文稿的用户。
