# asgeirtj/system_prompts_leaks

Collection of extracted System Prompts from popular chatbots like ChatGPT, Claude & Gemini

项目定位  
该项目收集了 ChatGPT、Claude 和 Gemini 等热门聊天机器人的系统提示，用于研究和分析 AI 模型的行为逻辑与安全风险。

核心功能  
- 收集多个主流 AI 模型的系统提示  
- 提供提示内容以供研究和测试  
- 便于理解模型的输入输出规则和限制  

技术特点  
项目以 HTML 形式呈现，结构清晰，便于浏览和分享。专注于提示工程与安全研究，适合对 AI 模型行为感兴趣的开发者和研究人员。

适用场景  
- AI 安全与漏洞研究  
- 提示工程学习与实验  
- 比较不同 AI 模型的系统行为  
- 开发者进行模型行为测试  
- 教育与学术研究用途  

推荐理由  
该仓库为研究 AI 模型行为提供了宝贵的资源，尤其适合关注提示注入、安全性和模型逻辑的研究者，具有较高的实用价值和参考意义。
