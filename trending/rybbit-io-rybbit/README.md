# rybbit-io/rybbit

🐸 Rybbit - open-source and privacy-friendly alternative to Google Analytics that is 10x more intuitive.

项目定位  
Rybbit 是一个开源且注重隐私的 Google Analytics 替代方案，提供更直观的网站分析功能。

核心功能  
- 网站访问数据分析  
- 用户行为追踪  
- 实时数据可视化  
- 无第三方 cookies 跟踪  
- 自托管部署支持  

技术特点  
使用 TypeScript 开发，注重隐私保护，采用轻量级架构，支持自托管和 API 集成，代码结构清晰易维护。

适用场景  
- 需要隐私保护的网站  
- 希望替代 Google Analytics 的开发者  
- 需要实时数据分析的团队  
- 对开源工具有偏好的用户  
- 个人博客或小型企业网站  

推荐理由  
Rybbit 提供直观、隐私友好的分析功能，适合追求数据自主控制和透明性的用户，是 Google Analytics 的优质替代方案。
