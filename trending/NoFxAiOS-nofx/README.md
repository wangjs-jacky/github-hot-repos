# NoFxAiOS/nofx

Your personal AI trading assistant. Any market. Any model. Pay with USDC, not API keys.

项目定位  
NoFxAiOS/nofx 是一个个人 AI 交易助手，支持多种市场和模型，通过 USDC 支付而非 API 密钥，降低交易门槛。

核心功能  
- 支持多交易所接入（如 Binance、Bybit、OKX 等）  
- 提供 AI 驱动的交易策略执行  
- 使用 USDC 作为支付方式，无需 API 密钥  
- 支持加密货币、外汇、股票等多种资产交易  
- 支持套利和做市等高级交易策略  

技术特点  
项目使用 Go 语言开发，具备高性能和可扩展性，支持多链和多交易所集成，采用轻量级架构以提升运行效率。

适用场景  
- 个人投资者进行自动化交易  
- 机构用于策略测试与部署  
- 套利交易者利用跨市场机会  
- 无需 API 密钥的交易需求  
- 多种金融资产的统一管理  

推荐理由  
NoFxAiOS/nofx 提供了一个灵活、安全且低成本的 AI 交易解决方案，适合希望利用 AI 进行多样化投资的用户。
