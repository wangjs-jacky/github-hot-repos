# afkarxyz/SpotiFLAC

Get Spotify tracks in true FLAC from Tidal, Qobuz & Amazon Music — no account required.

**项目定位**  
SpotiFLAC 是一个无需账户即可从 Tidal、Qobuz 和 Amazon Music 获取 Spotify 高保真 FLAC 音乐的工具，解决用户获取无损音乐格式的需求。

**核心功能**  
- 从多个音乐平台下载 Spotify 歌曲的 FLAC 格式  
- 无需账户即可使用  
- 支持高解析度音频  
- 自动识别并转换音质  
- 提供简洁的命令行界面  

**技术特点**  
使用 TypeScript 开发，结合多平台 API 调用与音频处理技术，实现无账户的高保真音乐下载，具备良好的跨平台兼容性。

**适用场景**  
- 需要无损音乐的音频爱好者  
- 不愿注册多个音乐平台的用户  
- 希望在本地存储高质量音乐文件  
- 音乐制作与后期处理需求者  
- 需要批量下载音乐的用户  

**推荐理由**  
SpotiFLAC 提供便捷的无损音乐下载方式，支持多平台且无需账户，适合追求高品质音频体验的用户，是 Spotify 用户的理想补充工具。
