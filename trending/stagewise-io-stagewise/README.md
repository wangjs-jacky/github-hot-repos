# stagewise-io/stagewise

The coding agent built for the web. stagewise is a purpose-built browser for developers with a coding agent built right in.

项目定位  
stagewise 是一个专为开发者设计的浏览器，内置编码代理，旨在提升网页开发效率和代码编写体验。

核心功能  
- 内置编码代理，支持实时代码生成与编辑  
- 与主流 IDE（如 VS Code）集成  
- 支持浏览器内开发环境，无需切换工具  
- 提供开发友好型界面和功能扩展  
- 集成多种开发工具和插件生态

技术特点  
基于 TypeScript 构建，结合 Chromium 浏览器引擎，具备良好的性能和可扩展性，支持现代 Web 开发需求。

适用场景  
- 前端开发与调试  
- 快速原型开发  
- 需要 IDE 和浏览器无缝协作的开发流程  
- 代码生成与辅助编程  
- 多工具协同工作的开发环境

推荐理由  
stagewise 通过整合浏览器与编码代理，提供高效、一体化的开发体验，适合注重效率的开发者群体。
