# kepano/obsidian-skills

Agent skills for Obsidian. Teach your agent to use Markdown, Bases, JSON Canvas, and use the CLI.

项目定位  
一个为 Obsidian 设计的代理技能集，帮助用户训练 AI 代理使用 Markdown、数据库、JSON 画布和命令行工具。

核心功能  
- 提供 Markdown 编辑与格式化技能  
- 支持 Obsidian 数据库操作  
- 实现 JSON 画布交互能力  
- 集成命令行接口（CLI）功能  
- 优化 AI 代理的自动化任务处理

技术特点  
基于 Obsidian 生态构建，结合多种数据格式和 CLI 工具，提升 AI 代理的实用性和灵活性。

适用场景  
- 自动化笔记整理与管理  
- 构建知识库与数据库系统  
- 开发 AI 驱动的文档流程  
- 提升 Obsidian 的智能交互能力  
- 扩展 Obsidian 的命令行功能

推荐理由  
适合希望提升 Obsidian 智能化水平的开发者和高级用户，提供丰富的技能模块以增强 AI 代理的功能。
