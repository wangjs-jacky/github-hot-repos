# op7418/guizang-ppt-skill

A Claude Code Skill that turns prompts into horizontal-swipe magazine-style HTML decks — 10 layouts, 5 curated themes, WebGL hero backgrounds, single-file output.

# GitHub 项目分析报告

1. **项目定位**  
该项目是一个用于将文本提示转换为横向滑动杂志风格 HTML 演示文稿的 Claude 代码技能，提供多种布局和主题，便于快速创建视觉吸引人的幻灯片。

2. **核心功能**  
- 将文本提示转换为 HTML 幻灯片  
- 提供 10 种布局和 5 种主题  
- 支持 WebGL 背景效果  
- 单文件输出，便于分享和部署  

3. **技术特点**  
使用 HTML 实现高效、轻量级的幻灯片生成，结合 WebGL 增强视觉效果，适合快速构建美观的展示内容，且无需复杂依赖。

4. **适用场景**  
- 快速制作演示文稿  
- 在线展示作品集或项目  
- 创作数字杂志或电子书  
- 用于教学或产品展示  
- 个人或团队创意表达  

5. **推荐理由**  
该工具简化了幻灯片制作流程，提供丰富的样式选择和高质量视觉效果，适合需要快速生成专业级演示内容的用户。
