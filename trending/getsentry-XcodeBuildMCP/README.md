# getsentry/XcodeBuildMCP

A Model Context Protocol (MCP) server and CLI that provides tools for agent use when working on iOS and macOS projects.

项目定位  
一个为 iOS 和 macOS 开发提供模型上下文协议（MCP）支持的服务器和命令行工具，帮助开发者在 Xcode 构建过程中更好地集成 AI 模型。

核心功能  
- 提供 MCP 服务器用于处理模型上下文交互  
- 支持通过 CLI 管理和配置 MCP 服务  
- 与 Xcode 和 xcodebuild 工具集成  
- 为代理工具提供模型上下文支持  
- 实现构建过程中的智能代码分析和建议

技术特点  
使用 TypeScript 编写，具备良好的可扩展性和跨平台能力，专注于提升开发效率和模型集成体验。

适用场景  
- 在 Xcode 中进行智能代码补全和分析  
- 自动化构建流程中集成 AI 模型支持  
- 开发者在 macOS 上进行高效代码管理  
- 需要模型上下文支持的 CI/CD 流程  
- 企业级 iOS/macOS 项目开发环境优化

推荐理由  
适合需要在 iOS/macOS 开发中集成 AI 模型上下文的开发者，提升构建效率和代码智能性，是 Xcode 生态中值得尝试的工具。
