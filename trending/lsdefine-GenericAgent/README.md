# lsdefine/GenericAgent

Self-evolving agent: grows skill tree from 3.3K-line seed, achieving full system control with 6x less token consumption

**项目定位**  
一个自我进化的AI代理，通过3.3K行种子代码构建技能树，实现系统控制，降低6倍的token消耗。

**核心功能**  
- 自我演化技能树，逐步提升能力  
- 实现桌面和浏览器自动化操作  
- 通过低消耗方式完成复杂任务  
- 支持多种大模型（如Claude、Gemini）集成  
- 基于记忆系统优化长期任务执行  

**技术特点**  
采用轻量级设计，结合记忆系统与技能树结构，以较少的token实现高效任务处理和系统控制，适合资源受限环境。

**适用场景**  
- 自动化日常办公任务  
- 桌面和浏览器操作自动化  
- 低资源环境下的AI代理应用  
- 需要长期任务执行的系统控制  
- AI代理技能扩展与优化实验  

**推荐理由**  
项目创新性强，通过技能树和记忆系统实现高效代理，适合对自动化和AI进化的研究者及开发者参考。
