# eze-is/web-access

给 Claude Code 装上完整联网能力的 skill：三层通道调度 + 浏览器 CDP + 并行分治

项目定位  
为 Claude Code 提供完整的联网能力，通过三层通道调度、浏览器 CDP 和并行分治实现高效网络访问。

核心功能  
- 实现与 Claude Code 的深度集成，增强其联网能力  
- 采用三层通道调度优化网络请求流程  
- 支持浏览器的 Chrome DevTools Protocol (CDP) 接入  
- 利用并行分治提升任务执行效率  

技术特点  
使用 JavaScript 构建，结合 CDP 实现浏览器控制，通过分治策略提升并发性能，具备良好的扩展性与灵活性。

适用场景  
- 需要增强 AI 模型联网能力的开发场景  
- 需要高效处理多任务网络请求的应用  
- 需要与浏览器深度交互的自动化工具开发  
- 开发需要并行处理能力的网络服务  

推荐理由  
该项目为 Claude Code 增强了强大的联网和浏览器交互能力，适合需要高效网络处理和并行任务管理的开发者使用。
