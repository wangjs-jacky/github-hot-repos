# floci-io/floci

Light, fluffy, and always free - The AWS Local Emulator alternative

**项目定位**  
Floci 是一个轻量级的 AWS 本地模拟器，旨在替代 LocalStack，提供更简单、快速的本地开发和测试环境。

**核心功能**  
- 模拟 AWS 服务如 S3、SQS、EC2 等  
- 支持 Docker 部署和本地运行  
- 提供与 AWS 兼容的 API 接口  
- 轻量级且易于集成到 CI/CD 流程  

**技术特点**  
基于 Java 实现，采用轻量级架构，支持快速启动和低资源消耗，兼容 Docker 和 TestContainers，适合 DevOps 场景。

**适用场景**  
- 本地开发时模拟 AWS 服务  
- 自动化测试环境搭建  
- CI/CD 流程中减少对外部服务依赖  
- 快速验证 AWS 功能而无需真实资源  

**推荐理由**  
Floci 提供了一个简单、高效、低成本的 AWS 本地模拟方案，特别适合开发者和 DevOps 团队在无网络或成本敏感环境下进行测试和开发。
