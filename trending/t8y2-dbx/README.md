# t8y2/dbx

15MB, lightweight, cross-platform database client. Supports MySQL, PostgreSQL, SQLite, Redis, MongoDB, DuckDB, ClickHouse, SQL Server and more.15MB，轻量级跨平台数据库客户端、数据库管理工具。支持 MySQL、PostgreSQL、SQLite、Redis、MongoDB、DuckDB、ClickHouse、SQL Server 等。

**项目定位**  
一个轻量级、跨平台的数据库客户端工具，支持多种数据库类型，简化数据库管理和查询操作。

**核心功能**  
- 支持多种数据库：MySQL、PostgreSQL、SQLite、Redis、MongoDB 等  
- 提供图形化界面（GUI）进行数据库连接与管理  
- 支持 SQL 查询与结果展示  
- 轻量级设计，易于安装和使用  
- 跨平台兼容性好  

**技术特点**  
使用 Rust 语言开发，确保高性能与安全性，结合 Tauri 和 Vue 构建跨平台桌面应用，具备良好的可维护性和扩展性。

**适用场景**  
- 开发人员快速访问和测试数据库  
- 数据库管理员进行日常维护  
- 数据分析师执行复杂查询  
- 多数据库环境下的统一管理  
- 需要轻量级工具的开发或教学场景  

**推荐理由**  
功能全面、轻量高效，支持多种数据库类型，适合开发者和数据库管理者提升工作效率，是跨平台数据库管理的优秀选择。
