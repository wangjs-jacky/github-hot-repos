# earendil-works/pi

AI agent toolkit: coding agent CLI, unified LLM API, TUI & web UI libraries, Slack bot, vLLM pods

项目定位  
earendil-works/pi 是一个 AI 代理工具包，提供多种接口和界面，用于构建和管理 AI 代理应用，简化了 AI 工具的集成与使用。

核心功能  
- 编程代理 CLI 工具  
- 统一 LLM API 接口  
- TUI 和 Web UI 库  
- Slack 机器人支持  
- vLLM 容器化部署支持  

技术特点  
基于 TypeScript 构建，具备良好的类型安全性和可扩展性，支持多平台交互，适合快速开发 AI 相关应用。

适用场景  
- 开发 AI 编程代理应用  
- 部署 AI 服务到 Web 或终端界面  
- 集成 AI 到 Slack 等协作工具  
- 使用 vLLM 进行本地大模型推理  
- 构建自定义 AI 工具链  

推荐理由  
项目功能全面，适配多种 AI 应用需求，适合开发者快速搭建 AI 代理系统，提升开发效率和用户体验。
