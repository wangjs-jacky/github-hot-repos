# coreyhaines31/marketingskills

Marketing skills for Claude Code and AI agents. CRO, copywriting, SEO, analytics, and growth engineering.

项目定位  
该项目为 Claude Code 和 AI 代理提供营销技能，涵盖 CRO、文案撰写、SEO、分析和增长工程，帮助优化 AI 相关产品的市场表现。

核心功能  
- 提供 CRO（转化率优化）策略  
- 包含 SEO 最佳实践  
- 教授高效文案撰写技巧  
- 分析工具和方法论  
- 增长工程策略指导

技术特点  
项目以 JavaScript 为主，聚焦于 AI 营销实践，结合数据分析与内容优化，适合开发者和营销人员协作提升 AI 产品市场竞争力。

适用场景  
- 优化 AI 产品页面转化率  
- 提升 AI 相关内容的搜索引擎排名  
- 支持 AI 代理的营销策略制定  
- 用于 AI 项目团队的内部培训  
- 帮助创业者构建营销知识体系

推荐理由  
该仓库为 AI 相关营销提供了系统性知识，适合希望提升 AI 产品市场表现的开发者和营销人员，内容实用且结构清晰。
