# permissionlesstech/bitchat

bluetooth mesh chat, IRC vibes

项目定位  
bitchat 是一个基于蓝牙 mesh 的聊天应用，提供类似 IRC 的实时通信体验，适用于设备间点对点或小范围群组聊天。

核心功能  
- 蓝牙 mesh 网络通信  
- 实时消息传递与聊天界面  
- 支持多设备连接与互动  
- 类 IRC 的命令行式操作体验  
- 简洁的用户界面与快速响应

技术特点  
使用 Swift 开发，专注于蓝牙低功耗技术实现设备间的可靠通信，具备良好的跨设备兼容性与实时性。

适用场景  
- 朋友间私密聊天  
- 小型团队内部沟通  
- 无网络环境下的设备交互  
- 电子游戏中的实时对战聊天  
- IoT 设备间的数据交换

推荐理由  
bitchat 提供了一种轻量级、低延迟的蓝牙聊天方案，适合需要离线或局域网内即时通讯的场景，具有独特的 IRC 风格交互体验。
