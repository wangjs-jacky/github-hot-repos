# ZhuLinsen/daily_stock_analysis

LLM驱动的 A/H/美股智能分析器：多数据源行情 + 实时新闻 + LLM决策仪表盘 + 多渠道推送，零成本定时运行，纯白嫖. LLM-powered stock analysis system for A/H/US markets.

项目定位  
一个基于大语言模型的股票分析系统，支持 A/H/美股市场，提供多数据源行情、实时新闻和智能决策功能。

核心功能  
- 多数据源行情获取  
- 实时新闻整合与分析  
- LLM 驱动的智能决策仪表盘  
- 多渠道消息推送  
- 定时自动化运行

技术特点  
采用 Python 编写，结合 LLM 和 RAG 技术，实现智能分析与决策，具备良好的扩展性和实时性。

适用场景  
- 个人投资者进行股市分析  
- 量化交易策略研究  
- 实时市场动态监控  
- 自动化投资决策辅助  
- 金融领域 AI 应用探索

推荐理由  
功能全面且开源免费，适合投资者和开发者使用，利用 LLM 提升股票分析效率，具有较高的实用价值和学习意义。
