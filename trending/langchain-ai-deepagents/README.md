# langchain-ai/deepagents

Agent harness built with LangChain and LangGraph. Equipped with a planning tool, a filesystem backend, and the ability to spawn subagents - well-equipped to handle complex agentic tasks.

项目定位  
一个基于 LangChain 和 LangGraph 构建的智能代理框架，用于处理复杂的代理任务，提供规划工具和子代理支持。

核心功能  
- 支持代理任务的规划与执行  
- 提供文件系统后端存储数据  
- 允许创建和管理子代理  
- 集成 LangChain 的强大链式处理能力  
- 适用于多步骤、复杂逻辑的自动化任务

技术特点  
采用 Python 编写，结合 LangChain 和 LangGraph 实现高效代理架构，具备良好的扩展性和模块化设计。

适用场景  
- 自动化流程管理  
- 复杂数据分析任务  
- 智能客服系统构建  
- 自主决策系统开发  
- 多代理协作应用

推荐理由  
适合需要构建复杂代理系统的开发者，结合了 LangChain 和 LangGraph 的优势，功能全面且易于扩展。
