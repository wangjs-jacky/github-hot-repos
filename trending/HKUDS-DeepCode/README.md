# HKUDS/DeepCode

"DeepCode: Open Agentic Coding (Paper2Code & Text2Web & Text2Backend)"

项目定位  
DeepCode 是一个开源的智能编程项目，旨在通过 AI 实现自动化代码生成与开发流程优化，解决传统编码效率低、重复性高的问题。

核心功能  
- 支持从文本生成代码（Text2Code）  
- 提供代码到网页的转换能力（Code2Web）  
- 支持代码到后端服务的构建（Code2Backend）  
- 采用代理机制实现自动化开发流程  

技术特点  
基于 Python 开发，结合大语言模型和代理系统，实现高效的代码生成与自动化开发，具备较强的扩展性和灵活性。

适用场景  
- 快速原型开发  
- 低代码/无代码平台构建  
- 自动化测试与部署  
- AI 辅助编程工具集成  

推荐理由  
DeepCode 为开发者提供了一种全新的智能编程方式，提升开发效率，适合对 AI 编程感兴趣的开发者和研究者。
