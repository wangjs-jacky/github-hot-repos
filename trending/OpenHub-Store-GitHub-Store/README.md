# OpenHub-Store/GitHub-Store

A free, open-source app store for GitHub releases — browse, discover, and install apps with one click. Powered by Kotlin and Compose Multiplatform for Android & Desktop (Linux, MacOS, Windows).

**项目定位**  
一个开源的 GitHub 应用商店，允许用户一键浏览、发现和安装 GitHub 上的发布应用，专注于 Android 和桌面平台。

**核心功能**  
- 浏览 GitHub 项目发布的应用程序  
- 一键安装支持多平台的应用  
- 提供应用分类与搜索功能  
- 支持 Kotlin Multiplatform 技术栈  

**技术特点**  
使用 Kotlin 和 Compose Multiplatform 开发，实现跨平台应用开发，支持 Android 及 Linux、MacOS、Windows 桌面系统，提升开发效率与用户体验。

**适用场景**  
- 开发者快速查找和安装 GitHub 上的应用  
- 用户便捷获取多平台应用  
- 轻量级工具替代传统应用商店  
- 适合需要跨平台支持的开发者项目  

**推荐理由**  
开源且功能实用，提供一站式 GitHub 应用管理体验，技术先进，适合开发者和普通用户使用。
