# sickn33/agentic-awesome-skills

Installable GitHub library of 1,935+ agentic skills for Claude Code, Cursor, Codex CLI, Autohand Code, Gemini CLI, Antigravity, and more. Includes specialized plugins, installer CLI, bundles, workflows, and official/community skill collections.

项目定位  
一个包含 1,935+ 可安装 agentic 技能的库，支持多种 AI 编码工具，提升开发效率和自动化能力。

核心功能  
- 提供多种 AI 工具（如 Claude Code、Cursor、Gemini）的技能集合  
- 支持插件、工作流、技能包等多样化组件  
- 集成 CLI 安装器，方便用户快速部署  
- 包含官方与社区维护的技能资源  
- 支持自定义技能开发与扩展  

技术特点  
基于 Python 构建，结构清晰，模块化设计便于维护和扩展，适配多个 AI 编码平台，具备良好的兼容性与可移植性。

适用场景  
- 开发者快速构建 AI 辅助编程环境  
- 自动化代码生成与调试流程  
- 多 AI 工具协同开发与集成  
- 学习与实验 agentic 技能架构  
- 企业级 AI 工具技能标准化管理  

推荐理由  
丰富的技能资源和对主流 AI 工具的全面支持，极大简化了开发者的工作流程，适合希望提升编码效率和自动化能力的用户。
