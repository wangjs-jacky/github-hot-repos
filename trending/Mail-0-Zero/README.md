# Mail-0/Zero

Experience email the way you want with Mail0 – the first open source email app that puts your privacy and safety first. Join the discord: https://mail0.link/discord

项目定位  
Mail-0/Zero 是一款以隐私和安全为核心理念的开源电子邮件应用，旨在为用户提供更私密、更安全的邮件体验。

核心功能  
- 端到端加密通信  
- 无广告、无跟踪的邮件服务  
- 支持自托管与本地部署  
- 集成 AI 工具提升邮件管理效率  
- 强化用户数据保护机制  

技术特点  
使用 TypeScript 构建，注重代码可维护性与安全性，采用现代化前端框架，支持跨平台使用，并强调隐私保护技术的深度集成。

适用场景  
- 个人用户关注隐私的邮件管理  
- 企业对数据安全有高要求的场景  
- 开发者研究隐私保护技术  
- 对传统邮件服务不满的用户群体  
- 希望自托管邮件系统的组织  

推荐理由  
Mail-0/Zero 以隐私和安全为核心，结合现代技术提供高效且私密的邮件解决方案，适合重视数据保护的用户和开发者。
