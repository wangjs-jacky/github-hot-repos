# rcourtman/Pulse

Real-time monitoring for Proxmox, Docker, and Kubernetes with AI-powered insights, smart alerts, and a beautiful unified dashboard

项目定位  
Pulse 是一个实时监控工具，用于 Proxmox、Docker 和 Kubernetes 环境，提供 AI 分析和智能告警。

核心功能  
- 实时监控 Proxmox、Docker 和 Kubernetes  
- AI 驱动的性能分析与预测  
- 智能告警系统  
- 统一的可视化仪表盘  
- 支持 Webhooks 通知

技术特点  
使用 Go 编写后端，结合 React 和 TypeScript 构建前端，支持自托管部署，具备良好的扩展性和可维护性。

适用场景  
- 家庭实验室环境监控  
- 企业级容器化应用监控  
- Proxmox 虚拟化平台管理  
- 自托管基础设施监控  
- 需要 AI 分析的运维场景

推荐理由  
Pulse 提供了全面的监控与智能分析功能，适合需要实时数据和 AI 支持的自托管环境，界面友好且功能强大。
