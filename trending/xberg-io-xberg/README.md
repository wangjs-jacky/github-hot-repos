# xberg-io/xberg

A polyglot document intelligence framework with a Rust core. Extract text, metadata, images, and structured information from PDFs, Office documents, images, and 97+ formats. Available for Rust, Python, Ruby, Java, Go, PHP, Elixir, C#, R, C, TypeScript (Node/Bun/Wasm/Deno)- or use via CLI, REST API, or MCP server.

项目定位  
xberg 是一个基于 Rust 的多语言文档智能框架，用于从多种格式中提取文本、元数据、图像和结构化信息。

核心功能  
- 支持 PDF、Office 文档、图像等 97+ 格式  
- 提取文本、元数据、图像和表格信息  
- 提供多种语言绑定（Rust、Python、Java 等）  
- 支持 CLI、REST API 和 MCP 服务器  

技术特点  
以 Rust 为核心实现高性能解析，支持多语言绑定与 WebAssembly，具备强大的跨平台兼容性与扩展性。

适用场景  
- 文档内容分析与处理  
- 企业级数据提取与 RAG 应用  
- 多语言开发环境集成  
- 需要高性能文档解析的系统  

推荐理由  
功能全面且性能优越，适合需要高效处理多种文档格式的开发者和企业，支持多语言和多种部署方式。
