# ayangweb/BongoCat

🐱 跨平台互动桌宠 BongoCat，为桌面增添乐趣！

项目定位  
BongoCat 是一款跨平台互动桌宠应用，为用户提供趣味性的桌面宠物体验，增添工作和生活中的乐趣。

核心功能  
- 实时互动的猫咪动画  
- 简单易用的桌面宠物控制  
- 支持多平台运行（Windows/macOS/Linux）  
- 轻量级应用，不占用过多系统资源  
- 可自定义宠物行为和外观

技术特点  
使用 TypeScript 开发，结合 Tauri 框架实现跨平台桌面应用，兼顾性能与开发效率，支持 Rust 后端增强稳定性。

适用场景  
- 办公室或学习时的娱乐陪伴  
- 提升桌面个性化体验  
- 作为开源项目学习跨平台开发  
- 适合喜欢可爱宠物的用户群体  
- 用于测试 Tauri 框架的应用案例

推荐理由  
BongoCat 是一款有趣且实用的桌面应用，兼具美观与功能性，适合追求个性化桌面体验的用户，同时是学习现代跨平台开发的优秀案例。
