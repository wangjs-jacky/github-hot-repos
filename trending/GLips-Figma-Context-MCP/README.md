# GLips/Figma-Context-MCP

MCP server to provide Figma layout information to AI coding agents like Cursor

项目定位  
MCP 服务器用于将 Figma 的布局信息提供给 AI 编码工具，如 Cursor，提升开发效率。

核心功能  
- 提取 Figma 设计文件中的布局数据  
- 将设计信息转换为 AI 可理解的格式  
- 支持与 AI 编码代理无缝集成  
- 实现实时设计与代码同步  
- 提供结构化数据接口

技术特点  
使用 TypeScript 构建，具备良好的类型安全和可维护性，专注于高效的数据转换与 API 交互，适配现代开发工具链。

适用场景  
- AI 编码工具获取设计信息  
- 快速生成符合设计规范的代码  
- 设计与开发团队协作流程优化  
- 自动化 UI 生成与测试  
- 提升前端开发效率

推荐理由  
为 AI 编码工具提供精准的 Figma 数据支持，简化设计到代码的转换流程，适合注重效率的开发者和设计团队。
