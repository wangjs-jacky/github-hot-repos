# TecharoHQ/anubis

Weighs the soul of incoming HTTP requests to stop AI crawlers

项目定位  
Anubis 是一个用于检测和阻止 AI 爬虫的 HTTP 请求分析工具，旨在保护 Web 服务免受自动化恶意流量的攻击。

核心功能  
- 分析 HTTP 请求特征以识别 AI 爬虫  
- 提供实时请求权重评估  
- 支持自定义规则和策略  
- 集成常见安全防护机制  
- 轻量级且易于部署

技术特点  
Anubis 使用 Go 语言开发，具备高性能和低延迟特性，专注于 HTTP 请求的实时分析与防御，适合高并发场景，代码结构清晰，便于维护和扩展。

适用场景  
- 保护网站免受 AI 爬虫攻击  
- 降低服务器负载和资源消耗  
- 提升 API 安全性  
- 用于安全研究和渗透测试  
- 在高流量 Web 应用中作为防护层

推荐理由  
Anubis 是一款高效、轻量且专注于 AI 爬虫防御的工具，适合需要提升 Web 安全性的开发者和运维人员，尤其在面对自动化攻击时表现突出。
