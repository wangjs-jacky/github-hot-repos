# Piebald-AI/claude-code-system-prompts

All parts of Claude Code's system prompt, 18 builtin tool descriptions, sub agent prompts (Plan/Explore/Task), utility prompts (CLAUDE.md, compact,  statusline, magic docs, WebFetch, Bash cmd, security review, agent creation).  Updated for each Claude Code version.

项目定位  
该项目为 Claude Code 提供系统提示和工具描述，帮助开发者理解和使用 Claude Code 的内置功能与子代理机制。

核心功能  
- 提供 Claude Code 的系统提示和工具描述  
- 包含子代理提示（Plan/Explore/Task）  
- 包含多种实用提示（CLAUDE.md、compact、statusline 等）  
- 随 Claude Code 版本更新维护  

技术特点  
项目以 JavaScript 为主，聚焦于提示工程，提供结构化的系统提示模板，便于集成和扩展，适用于 AI 编程辅助工具的开发与优化。

适用场景  
- 开发者调试和优化 Claude Code 工具链  
- 研究 AI 编程助手的提示设计  
- 构建基于 Claude Code 的自动化任务流程  
- 学习系统提示在 AI 代理中的应用  

推荐理由  
适合对 Claude Code 或 AI 编程助手感兴趣的开发者，提供全面的提示资源和结构化设计，有助于提升 AI 工具的使用效率和定制能力。
