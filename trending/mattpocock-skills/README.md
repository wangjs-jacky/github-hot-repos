# mattpocock/skills

My personal directory of skills, straight from my .claude directory.

项目定位  
一个个人技能目录，用于整理和展示用户在 .claude 目录中的技能信息。

核心功能  
- 存储和管理个人技能列表  
- 提供技能分类和标签系统  
- 支持快速查找和更新技能数据  
- 与 .claude 目录直接同步  

技术特点  
使用 Shell 脚本实现轻量级技能管理，结构简单，易于维护和扩展，适合个人技术栈记录。

适用场景  
- 个人技能整理与展示  
- 技术博客或简历的技能部分生成  
- 开发者自我评估和学习计划制定  
- 团队成员技能汇总与匹配  

推荐理由  
适合开发者用于管理和展示个人技能，结构清晰、易于定制，是个人技术成长的有效工具。
