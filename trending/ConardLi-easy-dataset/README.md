# ConardLi/easy-dataset

A powerful tool for creating datasets for LLM fine-tuning 、RAG and Eval

项目定位  
一个用于创建大语言模型微调、RAG 和评估数据集的工具，简化数据处理流程。

核心功能  
- 数据清洗与格式转换  
- 支持多种数据源导入  
- 自动化数据增强与标注  
- 提供评估指标与结果分析  
- 适配 LLM 微调和 RAG 需求

技术特点  
基于 JavaScript 构建，提供灵活的数据处理能力，支持模块化扩展，适合快速集成到现有工作流中。

适用场景  
- 大语言模型微调数据准备  
- RAG 系统的数据构建  
- 模型评估数据生成  
- NLP 项目数据预处理  
- 快速搭建实验数据集

推荐理由  
功能全面且易于使用，适用于需要高效构建高质量数据集的 LLM 和 RAG 项目，提升开发效率。
