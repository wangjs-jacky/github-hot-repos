# getarcaneapp/arcane

Modern Docker Management, Designed for Everyone

项目定位  
Arcane 是一个现代的 Docker 管理工具，旨在为用户提供简单易用的容器管理体验，适合各类用户进行本地或自托管的容器化部署。

核心功能  
- 提供图形化界面管理 Docker 容器  
- 支持 Docker Compose 配置管理  
- 实时监控容器状态和日志  
- 简化容器部署与配置流程  
- 支持多平台使用

技术特点  
该项目采用 Go 语言开发，结合 SvelteKit 和 TypeScript 构建前端，具备良好的性能和跨平台兼容性，专注于提供高效的容器管理解决方案。

适用场景  
- 个人开发者本地开发环境管理  
- 小型团队的容器化部署  
- 自托管服务器上的容器管理  
- 需要图形界面操作 Docker 的用户  
- 快速搭建和维护 Docker 应用

推荐理由  
Arcane 以简洁直观的界面和强大的功能，降低了 Docker 使用门槛，适合希望高效管理容器的用户，尤其适合非技术背景人员快速上手。
