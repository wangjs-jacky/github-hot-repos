# 666OS/ClashMac

Native Proxy Experience Built for macOS

项目定位  
ClashMac 是为 macOS 设计的原生代理工具，旨在提供高效、便捷的网络代理体验。

核心功能  
- 提供 Clash 和 Mihomo 的配置管理  
- 支持多种代理协议和规则集  
- 简洁直观的图形界面操作  
- 实时监控网络状态与连接情况  
- 自动更新配置文件与节点信息

技术特点  
基于 SwiftUI 构建，适配 macOS 系统特性，结合 Clash 和 Mihomo 技术实现高性能代理功能，具备良好的用户交互体验。

适用场景  
- 国内外网络访问需求  
- 企业或个人日常办公网络优化  
- 需要多节点切换的用户  
- 对代理性能有较高要求的开发者  
- 使用 macOS 系统的网络爱好者

推荐理由  
ClashMac 为 macOS 用户提供了稳定、高效的代理解决方案，界面友好且功能全面，适合需要灵活网络配置的用户使用。
