# Nagi-ovo/voyager

Voyager — an all-in-one enhancement suite for AI Studio, Gemini, Claude & ChatGPT: timelines, folders, prompts, usage tracking, chat export, plugins, and more. / 面向 AI Studio、Gemini、Claude 与 ChatGPT 的全能增强套件。

项目定位  
Voyager 是一个面向 AI Studio、Gemini、Claude 与 ChatGPT 的浏览器扩展工具，提供聊天管理、提示词优化、使用追踪等功能，提升用户在多个 AI 平台上的使用效率。

核心功能  
- 聊天历史管理与归档  
- 提示词模板与快速调用  
- 使用数据统计与分析  
- 跨平台聊天导出  
- 插件系统扩展功能  

技术特点  
使用 TypeScript 开发，支持多浏览器扩展（Chrome、Edge、Firefox 等），采用现代前端架构，具备良好的可扩展性和跨平台兼容性。

适用场景  
- 多 AI 平台用户日常使用  
- 需要管理大量聊天记录的开发者  
- 希望优化提示词使用的创作者  
- 需要导出或分析聊天数据的用户  
- 希望通过插件增强 AI 工具功能的用户  

推荐理由  
Voyager 提供全面的 AI 工具增强功能，尤其适合需要高效管理多个 AI 平台的用户，功能丰富且易于扩展，是提升 AI 使用体验的实用工具。
