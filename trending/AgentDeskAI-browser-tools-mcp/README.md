# AgentDeskAI/browser-tools-mcp

Monitor browser logs directly from Cursor and other MCP compatible IDEs.

**项目定位**  
该项目是一个浏览器日志监控工具，支持通过 Cursor 和其他 MCP 兼容 IDE 直接查看浏览器日志，提升开发调试效率。

**核心功能**  
- 实时监控浏览器日志  
- 与 Cursor 和 MCP 兼容 IDE 集成  
- 提供简洁的日志展示界面  
- 支持多浏览器环境  

**技术特点**  
基于 JavaScript 开发，采用轻量级架构，支持与 MCP 协议无缝对接，具备良好的扩展性和兼容性。

**适用场景**  
- 开发者在 Cursor 中调试浏览器应用  
- 需要实时查看浏览器日志的开发环境  
- 使用 MCP 协议进行开发的团队  
- 需要快速定位浏览器错误的场景  

**推荐理由**  
该工具简化了浏览器日志的查看流程，特别适合使用 MCP 兼容 IDE 的开发者，显著提升调试效率和开发体验。
