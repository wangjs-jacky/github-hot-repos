# sirmalloc/ccstatusline

🚀 Beautiful highly customizable statusline for Claude Code CLI with powerline support, themes, and more.

项目定位  
一个为 Claude Code CLI 设计的美观且高度可定制的状态栏，支持 Powerline 和多种主题。

核心功能  
- 支持 Powerline 风格的美化状态栏  
- 提供多种预设主题和自定义配置选项  
- 实时显示代码编辑器状态和 Git 信息  
- 轻量级且易于集成到 CLI 环境  

技术特点  
使用 TypeScript 开发，具备良好的可扩展性和类型安全，支持丰富的插件系统和主题配置，适合开发者深度定制。

适用场景  
- 在 Claude Code CLI 中提升开发体验  
- 为终端用户提供美观的状态显示  
- 需要高度定制化终端界面的开发者  
- 使用 Powerline 风格的用户  

推荐理由  
功能强大、易于配置，适合希望在 CLI 中获得美观且实用状态栏的开发者，尤其适合使用 Claude Code 的用户。
