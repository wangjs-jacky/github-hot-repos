# justlovemaki/AIClient-2-API

Simulates Gemini CLI, Antigravity, Qwen Code, and Kiro client requests, compatible with the OpenAI API. It supports thousands of Gemini model requests per day and offers free use of the built-in Claude model in Kiro. Easily connect to any client via the API, making AI development more efficient!

1. **项目定位**  
该项目是一个模拟多个 AI 客户端请求的 API 工具，兼容 OpenAI API，旨在简化 AI 开发流程，提升开发效率。

2. **核心功能**  
- 模拟 Gemini CLI、Antigravity、Qwen Code 和 Kiro 客户端请求  
- 支持高并发 Gemini 模型请求  
- 提供免费内置 Claude 模型使用  
- 通过统一 API 连接多种 AI 客户端  

3. **技术特点**  
基于 JavaScript 实现，具备良好的兼容性和扩展性，支持高并发处理，易于集成到现有开发流程中。

4. **适用场景**  
- 快速测试不同 AI 模型的调用效果  
- 开发过程中需要多模型支持的项目  
- 需要高效管理多个 AI 客户端的团队  
- 降低 AI 调用门槛，提升开发效率  

5. **推荐理由**  
提供多种 AI 客户端的统一接入方式，支持高并发和免费模型使用，是 AI 开发者提升效率的理想工具。
