# cjpais/Handy

A free, open source, and extensible speech-to-text application that works completely offline.

项目定位  
Handy 是一款完全离线运行的开源语音转文字应用，专注于无障碍和跨平台使用，为用户提供高效、私密的语音输入解决方案。

核心功能  
- 语音转文字功能  
- 支持多种语言识别  
- 离线运行，无需网络连接  
- 可扩展性强，支持自定义插件  
- 跨平台兼容（Windows, macOS, Linux）

技术特点  
该项目使用 Rust 编写，结合 Tauri 框架实现跨平台桌面应用，注重性能与安全性，同时保持轻量级和易于扩展。

适用场景  
- 无障碍辅助工具  
- 会议记录与速记  
- 无网络环境下的语音输入  
- 个人隐私敏感场景  
- 开发者构建自定义语音应用

推荐理由  
Handy 提供了强大且私密的语音转文字功能，适合需要离线操作和高可定制性的用户，是无障碍技术和跨平台开发的优秀实践。
