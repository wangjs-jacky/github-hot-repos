# pshenok/server-survival

Tower defense game that teaches cloud architecture. Build infrastructure, survive traffic, learn scaling.

项目定位  
一个基于 JavaScript 的塔防游戏，通过游戏形式教授云架构知识，帮助玩家学习基础设施构建与扩展。

核心功能  
- 玩家需构建和管理云基础设施以抵御流量攻击  
- 通过升级和扩展系统来应对不断增长的负载  
- 教学性质的关卡设计，融入 DevOps 和云服务概念  
- 使用 Three.js 实现 3D 可视化效果  

技术特点  
使用 JavaScript 构建，结合 Three.js 实现 3D 渲染，专注于教育与互动体验，适合初学者理解云架构和 DevOps 流程。

适用场景  
- 云技术初学者的学习工具  
- 开发者培训和教学辅助  
- 游戏化学习平台的案例参考  
- 提升对 AWS 和云服务的认知  

推荐理由  
项目将复杂的技术概念转化为趣味游戏，适合教育和自学，兼具实用性和娱乐性，是学习云架构的创新方式。
