# AsyncFuncAI/deepwiki-open

Open Source DeepWiki: AI-Powered Wiki Generator for GitHub/Gitlab/Bitbucket Repositories. Join the discord: https://discord.gg/gMwThUMeme

项目定位  
DeepWiki 是一个基于 AI 的开源维基生成器，用于为 GitHub、GitLab 和 Bitbucket 仓库自动生成知识库内容，提升代码可维护性和团队协作效率。

核心功能  
- 自动从代码仓库提取信息生成维基页面  
- 支持多种 AI 模型（如 Gemini、OpenAI、Ollama）进行内容生成  
- 提供可自托管的解决方案  
- 集成 Discord 社区支持  
- 支持多平台代码仓库接入

技术特点  
使用 Python 实现，结合多种 AI 接口进行内容生成，具备良好的扩展性和自托管能力，适合开发者快速部署和定制。

适用场景  
- 开发团队内部知识管理  
- 个人项目文档自动化生成  
- 企业级代码仓库知识库建设  
- 教育机构代码教学资源整理  
- 开源项目社区文档维护

推荐理由  
DeepWiki 通过 AI 技术简化了代码仓库的文档生成流程，提供灵活的自托管方案，适合需要高效管理代码知识的开发者和团队。
