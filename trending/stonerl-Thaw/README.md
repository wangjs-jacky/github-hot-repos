# stonerl/Thaw

Menu bar manager for macOS 26

项目定位  
Thaw 是一款为 macOS 设计的菜单栏管理工具，帮助用户更高效地管理和组织系统中的状态栏应用。

核心功能  
- 管理和隐藏 macOS 菜单栏中的应用程序  
- 支持自定义布局和排序  
- 提供快捷方式访问常用功能  
- 优化系统资源使用，提升性能  

技术特点  
使用 Swift 开发，结合 SwiftUI 实现现代化界面，注重性能优化和用户体验，适配最新 macOS 版本。

适用场景  
- 需要精简菜单栏的用户  
- 频繁切换应用的开发者或设计师  
- 希望提高桌面整洁度的办公用户  
- 对系统效率有较高要求的用户  

推荐理由  
Thaw 提供简洁高效的菜单栏管理方案，适合希望提升 macOS 使用体验的用户，尤其适合开发者和创意工作者。
