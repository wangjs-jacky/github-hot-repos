# liyupi/ai-guide

程序员鱼皮的 AI 资源大全 + Vibe Coding 零基础教程，分享 OpenClaw 保姆级教程、大模型玩法（DeepSeek / GPT / Gemini / Claude）、最新 AI 资讯、Prompt 提示词大全、AI 知识百科（Agent Skills / RAG / MCP / A2A）、AI 编程教程、AI 工具用法（Cursor / Claude Code / TRAE / Lovable / Copilot）、AI 开发框架教程（Spring AI / LangChain）、AI 产品变现指南，帮你快速掌握 AI 技术，走在时代前沿。本项目为开源文档，已升级为鱼皮 AI 导航网站

项目定位  
一个面向程序员的 AI 资源大全与教程集合，帮助开发者快速掌握 AI 技术和工具。

核心功能  
- 提供大模型使用教程（如 GPT、Claude、DeepSeek）  
- 包含 AI 编程、工具用法及开发框架教学  
- 整理 AI 知识百科与 Prompt 提示词  
- 零基础 AI 教程与产品变现指南  

技术特点  
基于 JavaScript 构建，结合 VuePress 搭建文档站点，内容结构清晰，便于维护和扩展。

适用场景  
- 初学者学习 AI 基础知识  
- 开发者寻找 AI 工具和框架教程  
- 企业或个人探索 AI 产品变现路径  
- 寻找最新 AI 资讯与技术动态  

推荐理由  
涵盖全面的 AI 学习资源与实践指南，适合不同层次开发者快速入门并提升 AI 技术能力。
