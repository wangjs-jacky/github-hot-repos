# Graphify-Labs/graphify

AI coding assistant skill (Claude Code, Codex, OpenCode, Cursor, Gemini CLI, and more). Turn any folder of code, SQL schemas, R scripts, shell scripts, docs, papers, images, or videos into a queryable knowledge graph. App code + database schema + infrastructure in one graph.

**项目定位**  
Graphify 是一个 AI 编程助手技能工具，旨在将各种代码和文档转化为可查询的知识图谱，提升开发效率与知识管理能力。

**核心功能**  
- 支持多种 AI 模型（如 Claude Code、Codex、Gemini）  
- 将代码、SQL、脚本、文档等转换为知识图谱  
- 提供结构化数据查询与分析能力  
- 集成数据库架构与基础设施信息  

**技术特点**  
基于 Python 实现，结合知识图谱构建与自然语言处理技术，支持多源数据整合与智能查询，具备良好的扩展性与灵活性。

**适用场景**  
- 企业级代码知识管理  
- 研究机构的论文与数据整合  
- 开发团队的文档与代码分析  
- AI 辅助编程与智能搜索  

**推荐理由**  
Graphify 提供了强大的知识图谱构建能力，适合需要高效管理复杂代码与文档的开发者与研究人员，尤其适用于 AI 辅助开发场景。
