# nanocoai/nanoclaw

A lightweight alternative to OpenClaw that runs in containers for security. Connects to WhatsApp, Telegram, Slack, Discord, Gmail and other messaging apps,, has memory, scheduled jobs, and runs directly on Anthropic's Agents SDK

**项目定位**  
nanoclaw 是一个轻量级的 AI 助手框架，作为 OpenClaw 的替代方案，支持在容器中运行以提高安全性，连接多种消息平台并具备记忆和任务调度功能。

**核心功能**  
- 支持连接 WhatsApp、Telegram、Slack 等多个消息平台  
- 提供记忆功能，保持对话上下文  
- 支持定时任务和自动化流程  
- 直接集成 Anthropic 的 Claude Agent SDK  

**技术特点**  
基于 TypeScript 构建，采用容器化部署方式提升安全性和可移植性，兼容多种 AI 模型和消息服务，具备良好的扩展性与灵活性。

**适用场景**  
- 企业自动化客服系统  
- 个人智能助手开发  
- 多平台消息整合与管理  
- AI 助手的快速原型开发  
- 安全敏感环境下的 AI 部署  

**推荐理由**  
nanoclaw 以其轻量、安全和多平台支持，成为构建 AI 助手的理想选择，尤其适合需要高可控性和跨平台集成的开发者和团队。
