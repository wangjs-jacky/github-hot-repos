# Hmbown/CodeWhale

Coding agent for DeepSeek models that runs in your terminal

项目定位  
一个基于 Rust 的命令行工具，用于与 DeepSeek 模型交互，提供本地化的 AI 编程辅助。

核心功能  
- 在终端中运行 DeepSeek 模型进行代码生成  
- 支持自然语言与代码的双向转换  
- 提供简洁的命令行接口  
- 实现低延迟的本地推理能力  
- 通过 TUI（文本用户界面）提升交互体验

技术特点  
使用 Rust 构建，注重性能和安全性，结合 CLI 和 TUI 技术，实现高效的本地 AI 编程助手。

适用场景  
- 开发者在终端中快速生成代码  
- 无需依赖云端服务的离线开发环境  
- 需要高效响应的编程辅助场景  
- 喜欢命令行工具的开发者群体  
- 对隐私和数据安全有要求的用户

推荐理由  
轻量高效，适合需要本地化 AI 辅助的开发者，结合 Rust 的性能优势和直观的终端交互，提升编码效率。
