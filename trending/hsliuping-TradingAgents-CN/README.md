# hsliuping/TradingAgents-CN

基于多智能体LLM的中文金融交易框架 - TradingAgents中文增强版

项目定位  
基于多智能体LLM的中文金融交易框架，用于开发和测试自动化交易策略。

核心功能  
- 支持多智能体协同交易  
- 集成中文自然语言处理能力  
- 提供历史数据回测功能  
- 可扩展的交易策略模块  
- 实时市场数据接入支持

技术特点  
采用Python实现，结合多智能体系统与大语言模型，专注于中文金融场景，具备良好的可扩展性和易用性。

适用场景  
- 量化交易策略研究  
- 自动化交易系统开发  
- 金融AI实验平台  
- 中文市场数据分析  
- 交易算法优化测试

推荐理由  
适合对金融AI和自动化交易感兴趣的开发者，提供完整的中文支持和灵活的框架结构，便于快速构建和验证交易策略。
