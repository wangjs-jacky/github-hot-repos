# hsliuping/TradingAgents-CN

基于多智能体LLM的中文金融交易框架 - TradingAgents中文增强版

1. **项目定位**  
基于多智能体LLM的中文金融交易框架，用于实现自动化、智能化的金融交易决策。

2. **核心功能**  
- 支持多智能体协同交易  
- 集成中文自然语言处理能力  
- 提供历史数据回测功能  
- 实现策略自动化执行  
- 支持多种交易市场接入

3. **技术特点**  
采用多智能体强化学习与大语言模型结合，针对中文金融场景优化，具备良好的可扩展性和交互性。

4. **适用场景**  
- 量化交易策略开发  
- 金融AI研究与实验  
- 个人投资者辅助决策  
- 机构交易系统原型设计  
- 金融数据分析与可视化

5. **推荐理由**  
适合对金融AI和自动化交易感兴趣的开发者，提供完整的中文支持和丰富的功能模块，是进行相关研究和实践的优质资源。
