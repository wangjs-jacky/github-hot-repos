# NoeFabris/opencode-antigravity-auth

Enable Opencode to authenticate against Antigravity (Google's IDE) via OAuth so you can use Antigravity rate limits and access models like gemini-3-pro and claude-opus-4-5-thinking with your Google credentials.

项目定位  
该项目允许 Opencode 通过 OAuth 认证接入 Antigravity（Google 的 IDE），使开发者能使用 Google 账户访问高级模型和享受速率限制。

核心功能  
- 实现 Opencode 与 Antigravity 的 OAuth 认证集成  
- 支持访问 Gemini-3-Pro 和 Claude-Opus-4-5 等高级模型  
- 利用 Google 身份验证提升安全性  
- 提供更灵活的 API 调用权限管理  
- 优化开发者使用 Google 生态工具的体验

技术特点  
项目使用 TypeScript 编写，具备良好的类型安全性和可维护性，采用现代 OAuth 流程实现与 Google 的集成，代码结构清晰，易于扩展和部署。

适用场景  
- 开发者希望在 Opencode 中使用 Google 身份登录  
- 需要调用 Gemini 或 Claude 模型的 AI 应用开发  
- 企业需要统一身份认证以管理 API 使用权限  
- 个人开发者希望简化认证流程  
- 构建基于 Google 生态的 AI 工具链

推荐理由  
该项目为 Opencode 用户提供了便捷的 Google 身份认证方式，便于访问高级 AI 模型，提升开发效率，适合需要集成 Google 服务的开发者使用。
