# zilliztech/claude-context

Code search MCP for Claude Code. Make entire codebase the context for any coding agent.

**项目定位**  
该项目是一个为 Claude Code 设计的代码搜索 MCP，旨在将整个代码库作为任何编码代理的上下文，提升代码生成和理解能力。

**核心功能**  
- 提供高效的代码搜索功能  
- 整合整个代码库作为上下文  
- 支持多种 AI 编码工具和平台  
- 利用语义搜索和向量数据库优化检索  
- 与 VSCode 等开发工具集成  

**技术特点**  
使用 TypeScript 构建，结合语义搜索、向量数据库和 Merkle 树等技术，实现高效、精准的代码检索与上下文管理。

**适用场景**  
- AI 编码助手开发  
- 代码生成与补全  
- 大规模代码库的智能搜索  
- 开发者工具集成  
- 代码分析与重构  

**推荐理由**  
适合需要高效代码检索和 AI 编码支持的开发者，提供强大的上下文管理能力，提升开发效率和代码质量。
