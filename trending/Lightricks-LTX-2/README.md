# Lightricks/LTX-2

Official Python inference and LoRA trainer package for the LTX-2 audio–video generative model.

项目定位  
LTX-2 是一个用于音频视频生成的 Python 推理与 LoRA 训练包，专注于提供高效的生成式 AI 工具。

核心功能  
- 提供 LTX-2 模型的推理接口  
- 支持 LoRA 微调训练  
- 处理音频和视频数据生成任务  
- 提供模型权重与配置管理  
- 优化训练与推理性能  

技术特点  
基于 Python 构建，结合深度学习框架，支持高效模型训练与部署，具备良好的扩展性和灵活性。

适用场景  
- 音频视频内容生成  
- 个性化媒体创作  
- AI 艺术创作工具开发  
- 研究与实验性项目  
- 多模态应用开发  

推荐理由  
LTX-2 提供完整的生成式 AI 工具链，适合需要音频视频生成能力的开发者和研究者，具有良好的社区支持和实用价值。
