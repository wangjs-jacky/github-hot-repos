# zhaoxuya520/reverse-skill

Reverse Engineering / Authorized Penetration Testing / Security Research Skill Router Pack AI-powered routing + On-demand toolchain bootstrapping + Self-evolving knowledge base  Supports Claude Code, Kiro, Cursor, Cline, and other AI coding clients 逆向/渗透/安全技能路由包 - AI 自动路由 + 按需自举工具链 + 自动进化经验库 | 支持 Claude Code / Kiro / Cursor / Cline 等代码 AI 客户端

1. **项目定位**  
该项目是一个面向逆向工程、渗透测试和安全研究的技能路由工具包，通过 AI 技术实现自动化任务分配与工具链管理。

2. **核心功能**  
- AI 自动路由安全任务到合适的工具或方法  
- 按需启动和配置安全工具链  
- 自动更新和优化知识库以适应新威胁  
- 支持多种 AI 编码客户端如 Claude Code、Cursor 等  
- 提供安全研究人员的技能增强与效率提升

3. **技术特点**  
采用 PowerShell 实现高效脚本化操作，结合 AI 路由机制与自进化知识库，提升安全研究自动化水平。

4. **适用场景**  
- 逆向工程分析  
- 渗透测试环境搭建  
- 安全研究知识管理  
- AI 编程辅助工具集成  
- 动态安全威胁应对

5. **推荐理由**  
该项目为安全研究人员提供了一体化的智能工具链管理方案，显著提升工作效率与任务自动化水平。
