# OpenCoworkAI/open-codesign

Open-source Claude Design alternative. One-click import your Claude Code / Codex API key. Prompt → prototype / slides / PDF. Multi-model (Claude, GPT, Gemini, Kimi, GLM, Ollama). BYOK, local-first, MIT.

项目定位  
OpenCodesign 是一个开源的 Claude Design 替代工具，允许用户一键导入 Claude Code 或 Codex API 密钥，实现从提示到原型、幻灯片和 PDF 的自动化生成。

核心功能  
- 支持多模型（Claude、GPT、Gemini、Kimi 等）  
- 一键导入 API 密钥  
- 从文本提示生成 UI 原型、幻灯片和 PDF  
- 支持 BYOK（Bring Your Own Key）和本地优先架构  

技术特点  
使用 TypeScript 开发，采用 Electron 构建桌面应用，注重本地数据安全和多模型兼容性，符合现代开发趋势。

适用场景  
- 产品设计师快速生成 UI 原型  
- 开发者通过提示生成文档或演示文稿  
- 需要多模型支持的 AI 设计工作流  
- 本地化部署需求的团队  

推荐理由  
OpenCodesign 提供了强大的 AI 设计能力，支持多种模型和本地部署，适合需要高效生成 UI 和文档的开发者与设计师。
