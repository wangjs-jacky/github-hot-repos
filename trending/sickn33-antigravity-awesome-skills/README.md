# sickn33/antigravity-awesome-skills

Installable GitHub library of 1,304+ agentic skills for Claude Code, Cursor, Codex CLI, Gemini CLI, Antigravity, and more. Includes installer CLI, bundles, workflows, and official/community skill collections.

项目定位  
一个面向 AI 编程工具的技能库，提供超过 1,300 个可安装的代理技能，支持多种 AI 编码平台。

核心功能  
- 提供多种 AI 工具的代理技能集合  
- 包含安装 CLI 工具和工作流配置  
- 支持官方与社区贡献的技能包  
- 简化 AI 代理的技能部署与管理  

技术特点  
使用 Python 开发，结构清晰，便于扩展和维护，适配多个 AI 编程平台，具备良好的兼容性与易用性。

适用场景  
- 开发者快速构建 AI 代理应用  
- 提升 AI 编码工具的功能性  
- 集成到自动化开发流程中  
- 学习和测试不同 AI 工具的技能组合  

推荐理由  
丰富的技能资源和多平台支持，适合希望提升 AI 编程效率和功能扩展的开发者，是 AI 代理开发的重要工具库。
