# OthmanAdi/planning-with-files

Claude Code skill implementing Manus-style persistent markdown planning — the workflow pattern behind the $2B acquisition.

项目定位  
该项目是一个基于 Claude 代码技能的 Markdown 持久化规划系统，模仿 Manus 风格的工作流模式，用于高效管理任务和计划。

核心功能  
- 支持使用 Markdown 进行持久化任务规划  
- 实现类似 Manus 的自动化任务管理流程  
- 利用 Claude AI 进行代码生成与规划优化  
- 提供结构化的任务分解与执行路径  
- 支持多步骤任务的自动追踪与更新

技术特点  
项目采用 Python 编写，结合 Claude AI 的代码生成能力，实现智能任务规划与自动化工作流，具备良好的可扩展性和模块化设计。

适用场景  
- 个人任务管理和日程规划  
- 团队协作中的自动化任务分配  
- AI 辅助的代码生成与开发流程  
- 持续集成与交付中的任务追踪  
- 自动化文档与知识管理

推荐理由  
该项目结合了 AI 技术与 Markdown 规划方式，提供高效的自动化任务管理方案，适合需要智能化规划和执行的开发者和团队。
