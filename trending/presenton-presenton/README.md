# presenton/presenton

Open-Source AI Presentation Generator and API (Gamma, Beautiful AI, Decktopus Alternative)

项目定位  
一个开源的 AI 演示文稿生成器和 API，旨在替代 Gamma、Beautiful AI 和 Decktopus，提供自动化 PowerPoint 生成服务。

核心功能  
- 通过自然语言生成演示文稿内容  
- 支持多种设计风格和布局  
- 提供 API 接口供集成使用  
- 自动化幻灯片排版与美化  
- 支持无 PPT 的演示文稿生成

技术特点  
使用 TypeScript 开发，结合 AI 技术实现内容生成与设计优化，具备良好的可扩展性和易用性。

适用场景  
- 快速创建商业汇报 PPT  
- 教育领域课件制作  
- 个人项目展示  
- 团队协作中的演示文稿生成  
- 需要自动化内容生成的业务流程

推荐理由  
开源且功能强大，适合需要快速生成专业演示文稿的用户，尤其适合对 AI 工具感兴趣的开发者和企业用户。
