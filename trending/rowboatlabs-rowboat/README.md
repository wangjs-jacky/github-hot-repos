# rowboatlabs/rowboat

Open-source AI coworker, with memory

项目定位  
Rowboat 是一个开源的 AI 合作伙伴，具备记忆功能，旨在提升开发者在 AI 代理和自动化任务中的效率。

核心功能  
- 支持多 AI 代理协作与任务编排  
- 提供持久化记忆功能以增强 AI 表现  
- 集成多种大模型（如 ChatGPT、Claude）  
- 实现自动化工作流与任务执行  
- 开发者友好的 SDK 与 API

技术特点  
使用 TypeScript 构建，注重类型安全与可扩展性，支持多模型集成与智能任务调度，适合构建复杂的 AI 代理系统。

适用场景  
- 自动化开发与测试流程  
- AI 代理协作与任务管理  
- 多模型协同工作流设计  
- 企业级 AI 工具集成  
- 开发者工具链增强

推荐理由  
Rowboat 为开发者提供了一个强大且灵活的 AI 合作平台，支持多模型与任务编排，适用于提高生产力和自动化复杂流程。
