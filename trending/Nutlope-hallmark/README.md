# Nutlope/hallmark

Anti-AI-slop design skill for Claude Code, Cursor, and Codex.

**项目定位**  
该项目旨在为 Claude Code、Cursor 和 Codex 等 AI 编程工具提供反 AI 拖延设计技巧，提升开发者效率与代码质量。

**核心功能**  
- 提供 AI 编程工具的优化使用方法  
- 包含代码编写与调试的最佳实践  
- 针对不同 AI 工具进行定制化建议  
- 促进高效编程流程设计  
- 减少 AI 生成代码中的冗余和低效部分  

**技术特点**  
项目聚焦于 CSS 技术，结合 AI 编程工具的使用场景，强调实用性和可操作性，适合开发者快速上手并提升编码效率。

**适用场景**  
- 使用 AI 编程工具的开发者  
- 希望提高代码质量与效率的团队  
- 对 AI 辅助开发感兴趣的初学者  
- 需要优化现有开发流程的项目  
- 寻找最佳实践以减少 AI 生成代码问题的用户  

**推荐理由**  
该仓库提供了针对 AI 编程工具的实用技巧，帮助开发者更高效地利用 AI 技术，减少冗余工作，提升整体开发体验。
