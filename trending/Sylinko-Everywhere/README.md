# Sylinko/Everywhere

Context-aware AI assistant for your desktop. Ready to respond intelligently, seamlessly integrating multiple LLMs and MCP tools.

项目定位  
一个面向桌面的上下文感知 AI 助手，集成多种大语言模型和 MCP 工具，提供智能化、无缝的交互体验。

核心功能  
- 支持多 LLM（如 OpenAI、Claude、Gemini）和 MCP 工具  
- 实现上下文感知的智能响应  
- 提供桌面级 UI 自动化能力  
- 融合 RAG 技术增强回答准确性  

技术特点  
基于 C# 和 Avalonia 构建，具备跨平台 UI 能力，支持多种 AI 模型接入，强调上下文理解和自动化集成。

适用场景  
- 个人桌面智能助手  
- 多模型协作开发环境  
- 自动化任务处理与脚本执行  
- 需要上下文理解的对话系统  

推荐理由  
集成多种 AI 模型与工具，提供强大的桌面级智能助手功能，适合需要高效、灵活 AI 协作的开发者和用户。
