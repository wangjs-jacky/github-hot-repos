# VoltAgent/awesome-agent-skills

Claude Code Skills and 1000+ agent skills from official dev teams and the community, compatible with Codex, Antigravity, Gemini CLI, Cursor and others.

项目定位  
一个汇集多种 AI 代理技能的开源项目，支持多平台如 Codex、Antigravity、Gemini CLI 等，提供丰富的技能资源。

核心功能  
- 收集了超过 1000 个 AI 代理技能  
- 兼容多个 AI 开发平台和工具  
- 提供官方与社区贡献的技能列表  
- 便于开发者快速集成和使用  

技术特点  
项目基于开源协作模式，整合主流 AI 平台技能，注重兼容性和实用性，适合开发者快速构建智能代理系统。

适用场景  
- AI 代理开发与测试  
- 多平台技能集成  
- 开发者快速获取技能资源  
- 企业智能系统构建  
- 个人 AI 工具扩展  

推荐理由  
项目内容丰富、兼容性强，适合各类开发者快速获取和应用 AI 代理技能，提升开发效率和系统智能化水平。
