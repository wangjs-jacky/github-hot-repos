# ayghri/i-have-adhd

A skill for your coding agent to stop it from burying the answer. ADHD-friendly output.

项目定位  
一个帮助开发者提高效率的工具，专为 ADHD 用户设计，防止代码代理隐藏答案。

核心功能  
- 确保代码代理清晰输出结果  
- 防止信息被过度包装或隐藏  
- 提供简洁直接的编程帮助  
- 优化开发流程中的注意力管理  
- 支持与 Claude 等 AI 工具集成

技术特点  
使用 Python 编写，专注于提升开发者体验，特别针对 ADHD 用户优化输出方式，增强可读性和专注力。

适用场景  
- 开发者在使用 AI 编程助手时  
- 需要快速获取明确答案的场景  
- 注意力容易分散的编程任务  
- 提高工作效率和减少认知负担  
- 与 Claude 等 AI 工具配合使用

推荐理由  
适合 ADHD 开发者提升效率，确保 AI 输出清晰易懂，优化编程体验，提高生产力。
