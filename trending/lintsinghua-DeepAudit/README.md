# lintsinghua/DeepAudit

DeepAudit：人人拥有的 AI 黑客战队，让漏洞挖掘触手可及。国内首个开源的代码漏洞挖掘多智能体系统。小白一键部署运行，自主协作审计 + 自动化沙箱 PoC 验证。支持 Ollama 私有部署 ，一键生成报告。支持中转站。​让安全不再昂贵，让审计不再复杂。

项目定位  
DeepAudit 是一个基于 AI 的开源代码漏洞挖掘系统，旨在降低安全审计门槛，提供自动化漏洞检测与验证能力。

核心功能  
- 多智能体协作进行代码审计  
- 自动化沙箱 PoC 验证  
- 支持 Ollama 私有部署  
- 一键生成审计报告  
- 支持中转站与多语言框架集成  

技术特点  
结合 AI 技术与传统 SAST 工具，采用模块化架构，支持多种开发框架与部署方式，提升漏洞检测效率与准确性。

适用场景  
- 开发者日常代码审查  
- 安全团队自动化渗透测试  
- 敏捷开发中的持续安全检测  
- 教育与研究领域的安全分析  
- 企业级代码质量与安全评估  

推荐理由  
DeepAudit 通过 AI 技术简化漏洞挖掘流程，适合各类开发者和安全人员使用，具备高效、易用与可扩展性，是代码安全审计的理想工具。
