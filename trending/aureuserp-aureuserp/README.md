# aureuserp/aureuserp

Free and Open Source ERP platform

aureuserp 是一个免费且开源的 ERP 平台，旨在为企业提供一体化的管理解决方案，涵盖财务、人力资源、库存等核心业务模块。

- 企业资源计划（ERP）系统  
- 财务与会计管理  
- 人力资源与招聘管理  
- 库存与采购管理  
- 销售与发票管理  

基于 PHP 开发，采用 FilamentPHP 框架提升开发效率，支持 Laravel 框架扩展，具备良好的可维护性和可扩展性。

- 中小型企业管理  
- 多部门协同办公  
- 财务与库存一体化管理  
- 电商与销售流程管理  
- 人力资源与招聘流程优化  

功能全面且开源免费，适合预算有限但需要专业 ERP 功能的企业，社区活跃，持续更新，易于定制和扩展。
