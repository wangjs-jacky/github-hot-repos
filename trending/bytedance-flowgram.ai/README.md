# bytedance/flowgram.ai

FlowGram is an extensible workflow development framework with built-in canvas, form, variable, and materials that helps developers build AI workflow platforms faster and simpler.

项目定位  
FlowGram 是一个用于快速构建 AI 工作流平台的可扩展框架，提供可视化界面和模块化组件，简化工作流开发流程。

核心功能  
- 可视化流程图编辑器  
- 内置表单与变量管理  
- 支持 AI 模块集成  
- 图形化数据流展示  
- 无代码/低代码开发支持  

技术特点  
基于 TypeScript 构建，采用 React 实现节点化 UI，具备良好的可扩展性和类型安全，适合构建复杂的工作流系统。

适用场景  
- AI 应用开发平台搭建  
- 自动化流程设计  
- 数据处理与分析流程可视化  
- 企业级工作流管理系统  
- 无代码工具开发  

推荐理由  
FlowGram 提供强大的可视化和模块化能力，帮助开发者高效构建 AI 工作流应用，适合需要快速开发和灵活扩展的项目。
