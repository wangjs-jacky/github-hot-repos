# teng-lin/notebooklm-py

Unofficial Python API and agentic skill for Google NotebookLM. Full programmatic access to NotebookLM's features—including capabilities the web UI doesn't expose—via Python, CLI, and AI agents like Claude Code, Codex, and OpenClaw.

项目定位  
该项目为 Google NotebookLM 提供了一个非官方的 Python API 和智能技能，允许开发者通过编程方式访问 NotebookLM 的功能，包括网页界面未提供的高级能力。

核心功能  
- 提供 Python API 实现对 NotebookLM 的全面控制  
- 支持通过 CLI 和 AI 代理（如 Claude、Codex）进行交互  
- 开发者可扩展和定制 NotebookLM 的功能  
- 可用于构建自动化工作流和智能应用  
- 暴露了网页 UI 未提供的底层功能

技术特点  
项目基于 Python 实现，提供灵活的 API 和 CLI 工具，支持与多种 AI 代理集成，具备良好的扩展性和可定制性，适合开发者深度使用 NotebookLM 功能。

适用场景  
- 自动化笔记整理与生成  
- 构建 AI 驱动的智能助手  
- 扩展 NotebookLM 的功能至自定义应用  
- 用于研究和实验 NotebookLM 的底层能力  
- 与 AI 代理结合实现复杂任务处理

推荐理由  
该仓库为开发者提供了对 NotebookLM 的全面程序化访问，适合希望深入利用其功能或构建智能应用的用户，尤其适合对 AI 代理和自动化流程感兴趣的开发者。
