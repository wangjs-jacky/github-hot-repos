# headroomlabs-ai/headroom

Compress tool outputs, logs, files, and RAG chunks before they reach the LLM. 60-95% fewer tokens, same answers. Library, proxy, MCP server.

项目定位  
Headroom 是一个用于压缩 LLM 输入内容的工具，旨在减少 token 使用量，同时保持输出质量。  

核心功能  
- 压缩工具输出、日志、文件和 RAG 段落  
- 提供库、代理和 MCP 服务器支持  
- 支持多种 LLM 平台如 OpenAI、Anthropic 等  
- 优化上下文窗口，提升效率  
- 自动处理文本，保持关键信息  

技术特点  
使用 Python 开发，结合上下文工程和 token 优化技术，提供高效、灵活的文本压缩方案，兼容主流 AI 框架和 API。  

适用场景  
- 大模型推理时的上下文优化  
- RAG 系统中减少冗余信息  
- 降低 API 调用成本  
- 提升代理服务性能  
- 日志和输出管理  

推荐理由  
Headroom 有效减少 token 使用，提升 LLM 效率，适用于需要优化上下文和成本的 AI 应用场景，是提升模型性能的重要工具。
