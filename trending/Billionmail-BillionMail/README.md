# Billionmail/BillionMail

BillionMail gives you open-source MailServer, NewsLetter,  Email Marketing — fully self-hosted, dev-friendly, and free from monthly fees. Join the discord: https://discord.gg/asfXzBUhZr

项目定位  
BillionMail 是一个开源的自托管邮件服务器、新闻通讯和电子邮件营销平台，旨在提供免月费、开发友好的邮件解决方案。

核心功能  
- 自托管邮件服务器（支持 SMTP、POP3、IMAP）  
- 电子邮件营销与新闻通讯功能  
- 集成反垃圾邮件工具（如 Rspamd）  
- 支持多域名管理与用户管理  

技术特点  
使用 Go 语言开发，具备高性能和可扩展性，集成主流邮件服务组件，适合快速部署和定制化开发。

适用场景  
- 企业自建邮件系统  
- 个人或小型团队的邮件营销  
- 需要避免云服务费用的场景  
- 开发者进行邮件相关功能测试  

推荐理由  
BillionMail 提供完整邮件服务功能，开源且无月费，适合需要自托管邮件系统的开发者和企业，具备良好的扩展性和社区支持。
