# Lake1059/FFmpegFreeUI

3FUI 是 ffmpeg 在 Windows 上的轻度专业交互外壳，收录大量参数，界面美观，交互友好。此项目面向国内使用环境，让普通人也能够轻松压制视频和转换格式。

项目定位  
FFmpegFreeUI 是一个为 Windows 用户设计的轻量级 ffmpeg 图形界面工具，旨在简化视频处理和格式转换操作，使非专业用户也能轻松使用 ffmpeg。

核心功能  
- 提供丰富的 ffmpeg 参数选项  
- 支持视频转码、格式转换、剪辑等操作  
- 界面简洁美观，交互友好  
- 针对国内用户优化使用体验  

技术特点  
采用 Visual Basic .NET 开发，结合 ffmpeg 强大的命令行功能，实现图形化操作，兼顾性能与易用性。

适用场景  
- 视频格式转换需求频繁的用户  
- 需要批量处理视频文件的场景  
- 想避免直接使用复杂命令行的初学者  
- 国内用户需要本地化支持的视频处理工具  

推荐理由  
FFmpegFreeUI 为 ffmpeg 提供了直观的图形界面，降低了使用门槛，适合需要高效处理视频的用户，尤其适合国内环境下的视频压制需求。
