# router-for-me/CLIProxyAPI

Wrap Gemini CLI, Antigravity, ChatGPT Codex, Claude Code, Qwen Code, iFlow as an OpenAI/Gemini/Claude/Codex compatible API service, allowing you to enjoy the free Gemini 2.5 Pro, GPT 5, Claude, Qwen model through API

项目定位  
CLIProxyAPI 是一个将多个 AI 编程模型封装为 OpenAI/Gemini/Claude 等兼容 API 的服务，让用户可通过统一接口调用多种免费模型。

核心功能  
- 支持 Gemini、ChatGPT Codex、Claude Code 等模型的 API 调用  
- 提供与 OpenAI 兼容的接口，方便集成  
- 通过 CLI 工具快速部署和使用  
- 支持多模型切换和配置  
- 实现模型资源的统一管理和调用

技术特点  
项目采用 Go 语言开发，具备高性能和跨平台特性，支持多种 AI 模型的封装与代理，提供简洁易用的 API 接口，便于开发者快速集成和使用。

适用场景  
- 开发者需要调用多种 AI 模型进行代码生成  
- 企业希望统一管理不同模型的 API 接口  
- 快速测试和对比不同 AI 模型的性能  
- 在无 API 访问权限时通过代理方式使用模型  
- 构建自定义 AI 服务或工具链

推荐理由  
CLIProxyAPI 为开发者提供了便捷的多模型接入方式，简化了 AI 工具的使用流程，适合需要灵活调用多种 AI 模型的场景，具有良好的扩展性和实用性。
