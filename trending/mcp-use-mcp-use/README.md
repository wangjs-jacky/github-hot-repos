# mcp-use/mcp-use

The fullstack MCP framework to develop MCP Apps for ChatGPT / Claude & MCP Servers for AI Agents.

项目定位  
MCP-USE 是一个全栈框架，用于开发适用于 ChatGPT、Claude 及 MCP 服务器的 AI 代理应用，旨在简化 AI 应用的构建与部署流程。

核心功能  
- 提供 MCP 应用开发工具链  
- 支持与 ChatGPT 和 Claude 等大模型集成  
- 实现 MCP 客户端与服务器通信  
- 包含 MCP 检查器和 UI 工具  
- 支持 AI 代理的上下文管理与技能扩展

技术特点  
基于 TypeScript 构建，支持前后端一体化开发，具备模块化架构和丰富的 MCP 协议支持，便于构建高效、可扩展的 AI 应用系统。

适用场景  
- 开发 AI 代理应用  
- 部署 MCP 服务端与客户端  
- 构建 AI 聊天机器人  
- 实现多模型交互与上下文管理  
- 构建 AI 技能库与工具集

推荐理由  
MCP-USE 提供了完整的 AI 应用开发框架，适合希望快速构建和部署 AI 代理系统的开发者，尤其适合需要与多种大模型集成的场景。
