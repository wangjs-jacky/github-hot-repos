# anthropics/knowledge-work-plugins

Open source repository of plugins primarily intended for knowledge workers to use in Claude Cowork

项目定位  
一个为知识工作者设计的开源插件库，用于增强 Claude Cowork 的功能和效率。

核心功能  
- 提供多种实用工具插件，提升工作效率  
- 支持与外部服务集成，扩展功能边界  
- 简化复杂任务流程，自动化重复操作  
- 优化信息处理与知识管理体验  

技术特点  
基于 Python 开发，结构清晰、模块化设计，易于扩展和维护，适合知识工作者快速集成使用。

适用场景  
- 文档整理与内容生成  
- 数据查询与分析辅助  
- 日常任务自动化  
- 多平台信息整合  

推荐理由  
为知识工作者提供高效、灵活的插件支持，提升协作与生产力，适合需要自动化和扩展功能的用户。
