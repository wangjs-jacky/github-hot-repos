# santinic/audiblez

Generate audiobooks from e-books

**项目定位**  
audiblez 是一个将电子书转换为有声书的工具，解决用户无法直接听电子书的问题。

**核心功能**  
- 支持多种电子书格式（如 EPUB）  
- 使用 TTS 技术将文本转为语音  
- 自动处理书籍结构与章节划分  
- 生成可播放的音频文件  

**技术特点**  
基于 Python 开发，结合 TTS 和 epub 解析库，实现高效、自动化的有声书生成流程。

**适用场景**  
- 需要将电子书转为音频的用户  
- 喜欢在通勤时听书的读者  
- 有特殊需求的残障人士  
- 教育或内容创作者需要批量处理书籍  

**推荐理由**  
简单易用，支持多格式，适合希望将电子书转化为音频的用户，提升阅读体验。
