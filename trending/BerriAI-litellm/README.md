# BerriAI/litellm

Python SDK, Proxy Server (AI Gateway) to call 100+ LLM APIs in OpenAI (or native) format, with cost tracking, guardrails, loadbalancing and logging. [Bedrock, Azure, OpenAI, VertexAI, Cohere, Anthropic, Sagemaker, HuggingFace, VLLM, NVIDIA NIM]

项目定位  
一个 Python SDK 和代理服务器，用于以 OpenAI 格式调用 100+ LLM API，支持成本追踪、安全控制和负载均衡。

核心功能  
- 统一接口调用多种大模型 API  
- 实时成本跟踪与监控  
- 防护机制（如内容过滤）  
- 自动负载均衡与故障转移  
- 日志记录与调试支持

技术特点  
采用 Python 构建，支持多云模型接入，具备高性能代理架构，集成成本管理与安全策略，适合企业级应用。

适用场景  
- 多模型服务聚合平台  
- AI 应用开发中的统一接口层  
- 企业级 AI 成本管控  
- 快速部署 AI 服务原型  
- 支持混合云与本地模型部署

推荐理由  
提供统一的 AI 接口解决方案，简化多模型调用流程，提升开发效率与系统稳定性，适合需要灵活集成多种大模型的开发者与企业。
