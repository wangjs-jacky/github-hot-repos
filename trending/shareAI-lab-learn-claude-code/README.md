# shareAI-lab/learn-claude-code

Bash is all you need -  A nano claude code–like 「agent harness」, built from 0 to 1

项目定位  
一个基于 TypeScript 构建的轻量级 AI 代理框架，旨在模拟 Claude Code 的功能，适合教学和实验。

核心功能  
- 提供类似 Claude Code 的代码生成能力  
- 支持与 LLM 对话并执行代码  
- 模拟代理运行环境  
- 简化 AI 代理开发流程  
- 适用于教学和研究用途

技术特点  
使用 TypeScript 构建，具备良好的可扩展性，结合 LLM 实现代码生成与执行，适合快速搭建 AI 代理原型。

适用场景  
- AI 代理开发教学  
- LLM 功能实验  
- 代码生成工具研究  
- 自动化任务模拟  
- 开发者学习 AI 工具

推荐理由  
适合初学者和教育者了解 AI 代理机制，提供简单易用的框架，便于快速上手和实验。
