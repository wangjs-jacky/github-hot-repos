# darrylmorley/whatcable

macOS menu bar app that tells you, in plain English, what each USB-C cable plugged into your Mac can actually do

项目定位  
一个 macOS 菜单栏应用，用于以通俗语言解释 USB-C 线缆的功能和兼容性，帮助用户了解连接设备的实际能力。

核心功能  
- 识别连接的 USB-C 线缆类型  
- 显示线缆支持的协议（如 USB 3.1、Thunderbolt 3 等）  
- 提供线缆的供电能力和数据传输能力信息  
- 实时在菜单栏显示结果，方便快速查看  

技术特点  
使用 Swift 编写，结合 IOKit 和系统底层接口获取硬件信息，界面采用 SwiftUI 构建，轻量且高效，适合 macOS 系统环境。

适用场景  
- 开发者调试硬件连接问题  
- 普通用户确认 USB-C 线缆是否支持特定功能  
- 选购 USB-C 线缆时参考实际性能  
- 在 macOS 上进行硬件兼容性测试  

推荐理由  
实用性强，帮助用户直观理解 USB-C 线缆的实际能力，尤其适合需要精确控制外设连接的 macOS 用户。
