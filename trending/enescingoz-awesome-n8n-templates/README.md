# enescingoz/awesome-n8n-templates

280+ free n8n automation templates — ready-to-use workflows for Gmail, Telegram, Slack, Discord, WhatsApp, Google Drive, Notion, OpenAI, and more. AI agents, RAG   chatbots, email automation, social media, DevOps, and document processing. The largest open-source n8n template collection.

项目定位  
一个包含 280 多个免费 n8n 自动化模板的开源集合，帮助用户快速构建自动化工作流。

核心功能  
- 提供多种平台（Gmail、Slack、Telegram 等）的自动化模板  
- 支持 AI 代理、RAG 聊天机器人、邮件自动化等高级功能  
- 模板可直接导入 n8n 使用，无需从头开发  
- 包含 DevOps、文档处理等实用场景  

技术特点  
基于 n8n 构建，支持低代码/无代码自动化，结构清晰，易于扩展和集成，适合开发者和非技术人员使用。

适用场景  
- 企业自动化流程搭建  
- 个人任务自动化管理  
- AI 聊天机器人开发  
- 社交媒体内容发布  
- 数据同步与文件处理  

推荐理由  
丰富的模板资源，覆盖广泛应用场景，适合希望快速实现自动化的企业和个人，是 n8n 用户不可多得的参考库。
