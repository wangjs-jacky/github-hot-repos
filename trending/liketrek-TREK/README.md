# liketrek/TREK

A self-hosted travel/trip planner with real-time collaboration, interactive maps, PWA support, SSO, budgets, packing lists, and more.

项目定位  
TREK 是一个自托管的旅行规划工具，支持实时协作、互动地图和多种旅行管理功能，帮助用户高效规划行程。

核心功能  
- 实时协作编辑旅行计划  
- 交互式地图与路线规划  
- 预算跟踪与打包清单管理  
- 支持 SSO 和 PWA 应用  
- 多用户共享与同步功能

技术特点  
使用 TypeScript 开发，具备实时通信、PWA 支持和自托管架构，适合构建可扩展的 Web 应用。

适用场景  
- 个人旅行计划制定  
- 团队或家庭行程协作  
- 自托管企业级旅行管理  
- 需要离线访问的旅行应用  
- 需要集成 SSO 的组织使用

推荐理由  
TREK 提供全面的旅行规划功能，支持自托管和实时协作，适合需要灵活控制数据和多人协作的用户。
