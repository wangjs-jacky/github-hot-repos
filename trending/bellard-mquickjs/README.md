# bellard/mquickjs

Public repository of the Micro QuickJS Javascript Engine

项目定位  
Micro QuickJS 是一个轻量级的 JavaScript 引擎，专为嵌入式系统和高性能场景设计，提供快速执行和低内存占用。

核心功能  
- 支持 ECMAScript 2020 标准  
- 提供高效的垃圾回收机制  
- 可嵌入到 C/C++ 应用中  
- 支持异步编程模型  
- 轻量级且可定制  

技术特点  
Micro QuickJS 采用 C 语言实现，注重性能与内存效率，具备高度可移植性，适合资源受限环境。

适用场景  
- 嵌入式设备开发  
- 移动端应用中的脚本执行  
- 需要轻量级 JS 引擎的服务器端应用  
- IoT 设备中的动态逻辑处理  
- 自定义脚本扩展功能  

推荐理由  
Micro QuickJS 以高效、轻量和易集成著称，是需要在资源受限环境中运行 JavaScript 的理想选择。
