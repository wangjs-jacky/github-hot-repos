# tinyauthapp/tinyauth

The tiniest authentication and authorization server you have ever seen.

项目定位  
一个极简的认证与授权服务器，提供轻量级身份验证解决方案，适用于需要快速部署的自托管场景。

核心功能  
- 支持多种认证方式，如 TOTP、2FA、OIDC  
- 提供中间件集成能力，可嵌入现有 Web 服务  
- 简洁配置，易于部署和维护  
- 支持自托管，无需依赖第三方服务  

技术特点  
使用 Go 语言开发，性能高效，代码简洁。支持 Caddy、Nginx 等常见反向代理，具备良好的扩展性和跨平台兼容性。

适用场景  
- 个人或小型团队搭建私有认证服务  
- 需要快速集成认证功能的 Web 应用  
- 自托管环境下的 SSO 和多因素认证需求  
- 对轻量级系统有要求的开发项目  

推荐理由  
项目小巧易用，功能全面，适合需要简单、高效认证方案的开发者和运维人员，尤其适合自托管和私有部署场景。
