# datawhalechina/all-in-rag

🔍大模型应用开发实战一：RAG 技术全栈指南，在线阅读地址：https://datawhalechina.github.io/all-in-rag/

项目定位  
一个专注于 RAG（检索增强生成）技术的实战指南，帮助开发者构建基于大模型的应用。

核心功能  
- 提供 RAG 技术全流程讲解  
- 包含多种 RAG 实现方案  
- 整合常用工具和框架使用示例  
- 涵盖嵌入、索引、查询等关键环节  

技术特点  
采用主流 AI 框架如 LangChain、Llama-Index，结合 Milvus、Neo4j 等技术，全面覆盖 RAG 技术栈。

适用场景  
- 大模型应用开发学习  
- 企业知识库问答系统构建  
- 智能客服系统开发  
- 学术研究与技术验证  

推荐理由  
内容详实、结构清晰，适合初学者和进阶开发者，是掌握 RAG 技术的重要资源。
