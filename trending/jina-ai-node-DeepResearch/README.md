# jina-ai/node-DeepResearch

Keep searching, reading webpages, reasoning until it finds the answer (or exceeding the token budget)

项目定位  
一个基于 TypeScript 的深度搜索工具，通过持续搜索、阅读网页和推理来找到答案，适用于需要自动化信息检索的场景。

核心功能  
- 自动化网页搜索与内容提取  
- 持续推理以寻找答案  
- 限制 Token 使用以控制成本  
- 支持多步骤任务分解  
- 提供可扩展的搜索流程

技术特点  
采用 TypeScript 构建，注重模块化设计和可扩展性，结合 AI 推理与传统搜索技术，实现高效的自动化信息处理。

适用场景  
- 学术研究与文献分析  
- 市场调研与竞争分析  
- 自动化客服与问答系统  
- 数据收集与情报分析  
- 信息验证与事实核查

推荐理由  
适合需要高效、自动化信息处理的开发者和研究人员，功能强大且易于集成，是深度搜索领域的优秀实践。
