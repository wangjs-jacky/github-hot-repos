# EKKOLearnAI/hermes-studio

Web dashboard for Hermes Agent — multi-platform AI chat, session management, scheduled jobs, usage analytics 

项目定位  
Hermes Studio 是一个为 Hermes Agent 提供的 Web 仪表盘，支持多平台 AI 聊天、会话管理、定时任务和使用分析，便于用户管理和监控 AI 代理系统。

核心功能  
- 多平台 AI 聊天界面  
- 会话管理和历史记录  
- 定时任务调度与管理  
- 用户使用数据分析  
- 自托管部署支持

技术特点  
基于 TypeScript 和 Vue3 构建，具备良好的可扩展性和组件化设计，适合构建现代化的 Web UI 系统，支持自托管部署。

适用场景  
- 企业内部 AI 代理系统管理  
- 开发者调试和测试 AI 模型  
- 需要多模型支持的 AI 应用  
- 需要定时任务管理的自动化流程  
- 个人或团队的 AI 项目监控

推荐理由  
Hermes Studio 提供了全面的 AI 代理管理功能，结合自托管特性，适合需要灵活控制和高度定制化的 AI 项目使用。
