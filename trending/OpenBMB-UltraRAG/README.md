# OpenBMB/UltraRAG

A Low-Code MCP Framework for Building Complex and Innovative RAG Pipelines

项目定位  
UltraRAG 是一个低代码的 MCP 框架，用于构建复杂且创新的 RAG（检索增强生成）流程，简化了 RAG 系统的开发与部署。

核心功能  
- 提供模块化组件支持 RAG 流程构建  
- 支持多种 LLM 和嵌入模型集成  
- 提供可视化界面和 API 接口  
- 支持多模态数据处理  
- 降低 RAG 系统开发门槛

技术特点  
基于 Python 开发，融合多种主流 AI 框架和模型，具备良好的扩展性和易用性，适合快速搭建 RAG 应用。

适用场景  
- 智能问答系统开发  
- 文档知识库构建  
- 多模态内容生成  
- 企业级 RAG 应用部署  
- 快速原型验证与测试

推荐理由  
UltraRAG 通过低代码方式简化 RAG 流程开发，适合开发者和研究人员快速构建高效、灵活的 RAG 系统。
