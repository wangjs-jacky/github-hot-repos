# Thysrael/Horizon

📡 Your own AI-powered news radar. Generates daily briefings in English & Chinese. | 用 AI 构建你专属的新闻雷达

**项目定位**  
Horizon 是一个 AI 驱动的新闻聚合工具，为用户提供中英文每日简报，帮助用户高效获取关键信息。

**核心功能**  
- 自动抓取多来源新闻  
- 使用 AI 生成中英文每日简报  
- 支持通过 Webhook 或飞书机器人推送  
- 可自定义新闻主题和过滤规则  
- 提供实时新闻雷达功能  

**技术特点**  
基于 Python 实现，结合 LLM 技术进行内容生成，支持多种推送方式，具备良好的扩展性和灵活性。

**适用场景**  
- 个人每日信息摘要  
- 企业内部新闻监控  
- 跨语言信息获取  
- 自动化新闻分发系统  
- 研究机构情报收集  

**推荐理由**  
功能实用、易于部署，结合 AI 和新闻聚合，适合需要高效获取信息的用户和团队。
