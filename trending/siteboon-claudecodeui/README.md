# siteboon/claudecodeui

Use Claude Code, Cursor CLI or Codex on mobile and web with CloudCLI (aka Claude Code UI). CloudCLI is a free open source webui/GUI that helps you manage your Claude Code session and projects remotely

项目定位  
ClaudeCodeUI 是一个用于在移动和网页端管理 Claude Code 会话和项目的开源 Web UI 工具，帮助用户远程操作 AI 编码助手。

核心功能  
- 提供 Claude Code 的图形化界面  
- 支持通过 CloudCLI 管理代码会话  
- 兼容 Cursor CLI 和 Codex  
- 移动优先设计  
- 本地与云端代码管理

技术特点  
基于 TypeScript 构建，采用 React 框架，注重跨平台兼容性和用户体验，提供轻量级的 Web 界面解决方案。

适用场景  
- 开发者远程编写和调试代码  
- 移动端使用 AI 编码辅助  
- 快速切换多个代码项目  
- 需要图形化界面的 Claude Code 用户  
- 希望简化 AI 编码工具使用的开发者

推荐理由  
ClaudeCodeUI 为用户提供了一个直观、高效的 AI 编码工具界面，特别适合希望在移动端或网页端便捷使用 Claude Code 的开发者。
