# jundot/omlx

LLM inference server with continuous batching & SSD caching for Apple Silicon — managed from the macOS menu bar

1. **项目定位**  
该项目是一个为 Apple Silicon 优化的 LLM 推理服务器，支持持续批处理和 SSD 缓存，并可通过 macOS 菜单栏进行管理，提升本地大模型推理效率。

2. **核心功能**  
- 支持 Apple Silicon 上的 LLM 推理  
- 提供持续批处理以提高吞吐量  
- 集成 SSD 缓存加速模型加载  
- 通过 macOS 菜单栏进行可视化管理  

3. **技术特点**  
基于 Python 构建，利用 Apple 的 MLX 框架优化推理性能，结合 SSD 缓存与连续批处理机制，实现高效、低延迟的本地模型服务。

4. **适用场景**  
- 本地部署大型语言模型  
- macOS 用户需要轻量级推理服务  
- 需要通过菜单栏快速管理模型服务  
- 对推理延迟敏感的应用场景  

5. **推荐理由**  
针对 Apple Silicon 优化，提供高效的本地推理方案，适合希望在 macOS 上便捷运行 LLM 的开发者和研究人员。
