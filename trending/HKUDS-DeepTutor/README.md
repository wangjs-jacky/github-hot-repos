# HKUDS/DeepTutor

"DeepTutor: AI-Powered Personalized Learning Assistant"

**项目定位**  
DeepTutor 是一个基于 AI 的个性化学习助手，旨在通过智能技术提供定制化的教育体验，解决传统教学中个性化不足的问题。

**核心功能**  
- 个性化学习路径推荐  
- 基于知识图谱的互动式学习  
- 多智能体协作教学  
- RAG（检索增强生成）支持知识更新  
- 与大型语言模型集成以提升交互体验  

**技术特点**  
该项目融合了大语言模型、知识图谱和多智能体系统，采用 RAG 技术提升信息准确性，强调 AI 在教育场景中的深度应用与交互性。

**适用场景**  
- 在线教育平台个性化辅导  
- 自主学习者知识补充  
- 教育机构智能助教  
- 研究人员探索 AI 教育应用  
- 企业培训中的智能学习支持  

**推荐理由**  
DeepTutor 结合前沿 AI 技术与教育需求，具备高度可扩展性和实用性，适合关注 AI 教育创新的开发者和研究者。
