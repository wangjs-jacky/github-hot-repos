# OpenPipe/ART

Agent Reinforcement Trainer: train multi-step agents for real-world tasks using GRPO. Give your agents on-the-job training. Reinforcement learning for Qwen3.5, GPT-OSS, Llama, and more!

项目定位  
OpenPipe/ART 是一个用于训练多步骤代理的强化学习框架，支持 Qwen3.5、GPT-OSS 等大模型，旨在提升代理在真实任务中的表现。

核心功能  
- 提供 GRPO（Generalized Reinforcement Learning for Prompt Optimization）训练方法  
- 支持多种大语言模型进行代理训练  
- 实现代理的持续优化和实时反馈学习  
- 提供可扩展的代理架构和训练流程  
- 集成 LoRA 技术以降低训练成本

技术特点  
基于 Python 构建，结合强化学习与大模型技术，支持多模型适配与高效微调，具备良好的扩展性和灵活性。

适用场景  
- 智能客服系统中的代理训练  
- 自动化任务执行与决策系统  
- 企业级 AI 代理开发与部署  
- 大模型的持续优化与迭代  
- 研究机构进行强化学习实验

推荐理由  
适用于希望提升代理在真实场景中表现的开发者和研究者，尤其适合需要多模型支持和高效训练的 RL 应用。
