# ryoppippi/ccusage

A CLI tool for analyzing Claude Code/Codex CLI usage from local JSONL files.

项目定位  
ccusage 是一个命令行工具，用于分析本地 JSONL 文件中的 Claude Code/Codex 使用情况，帮助用户了解模型调用数据。

核心功能  
- 读取并解析 JSONL 格式的使用日志文件  
- 提供详细的使用统计信息  
- 支持按时间、模型等维度筛选数据  
- 输出简洁的文本报告  

技术特点  
基于 TypeScript 开发，具备良好的类型安全和可维护性，适合处理结构化日志数据，易于扩展和集成到现有工作流中。

适用场景  
- 分析 AI 模型调用频率与成本  
- 优化代码生成或查询的使用策略  
- 监控和审计模型使用情况  
- 数据驱动的开发决策支持  

推荐理由  
ccusage 简洁高效，专为分析 Claude Code/Codex 使用数据设计，适合需要深入理解模型使用情况的开发者和团队。
