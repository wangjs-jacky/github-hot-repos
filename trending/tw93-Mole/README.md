# tw93/Mole

🐹 Deep clean and optimize your Mac.

**项目定位**  
Mole 是一个用于深度清理和优化 Mac 的命令行工具，帮助用户清理系统垃圾、优化性能并释放磁盘空间。

**核心功能**  
- 清理临时文件和缓存  
- 删除无用的安装包和残留文件  
- 优化系统配置提升运行效率  
- 支持自动检测和清理不必要文件  
- 提供详细的清理报告

**技术特点**  
使用 Shell 脚本实现高效系统清理，结合多种 macOS 系统特性进行智能分析，具备良好的兼容性和可扩展性。

**适用场景**  
- 用户需要手动清理 Mac 系统垃圾  
- 开发者或高级用户希望优化系统性能  
- 需要释放磁盘空间以提升设备运行速度  
- 用于自动化脚本集成到系统维护流程中  
- 适合对命令行工具熟悉的用户

**推荐理由**  
Mole 是一款轻量且高效的 Mac 清理工具，适合追求系统整洁与性能优化的用户，尤其适合熟悉命令行操作的开发者。
