# RunanywhereAI/runanywhere-sdks

Production ready toolkit to run AI locally

项目定位  
提供一套可在本地运行 AI 的生产级工具包，解决在设备端高效部署和运行 AI 模型的问题。

核心功能  
支持多种 AI 模型的本地推理，跨平台开发支持，优化模型性能，简化 AI 集成流程，提供多语言 SDK。

技术特点  
基于 C++ 开发，注重性能与效率，兼容多种 AI 框架，支持多平台部署，强调隐私与低延迟。

适用场景  
移动应用集成 AI 功能，边缘计算设备部署模型，本地化 AI 服务构建，跨平台应用开发，实时语音与视觉处理。

推荐理由  
高性能、跨平台、易于集成，适合需要在本地运行 AI 的开发者和企业，提升隐私安全与响应速度。
