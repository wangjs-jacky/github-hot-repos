# bytedance/Dolphin

The official repo for “Dolphin: Document Image Parsing via Heterogeneous Anchor Prompting”, ACL, 2025.

项目定位  
Dolphin 是一个用于文档图像解析的开源项目，专注于通过异构锚点提示技术提升布局分析和 OCR 准确性。

核心功能  
- 文档图像布局分析  
- 多格式 PDF 解析与转换  
- 基于 VLM 的 OCR 识别  
- 文本与结构化数据提取  
- 支持复杂排版和表格识别

技术特点  
采用异构锚点提示方法，结合视觉语言模型提升文档解析精度，适用于复杂排版和多语言文档处理。

适用场景  
- 学术论文或报告的数字化处理  
- 企业文档自动化归档  
- PDF 内容提取与再利用  
- 研究人员进行文档分析实验  
- 需要高精度 OCR 和布局理解的场景

推荐理由  
Dolphin 提供先进的文档解析能力，适合需要高精度 OCR 和布局分析的开发者及研究人员，尤其在学术和企业级应用中具有广泛价值。
