# chatfire-AI/huobao-drama

🎬 火宝短剧 - 基于AI的一站式短剧生成平台 《一句话生成完整短剧，从剧本到成片全自动化》  Huobao Drama - An AI-Powered End-to-End Short Drama Generator "One Sentence to Complete Drama: Fully Automated from Script to Final Video"

项目定位  
一个基于AI的短剧生成平台，用户可通过一句话生成完整的短剧内容，涵盖剧本到成片的全流程自动化。

核心功能  
- 一句话输入生成完整短剧  
- 自动化剧本创作与分镜设计  
- AI生成视频内容  
- 支持多角色与场景配置  
- 提供可视化编辑界面

技术特点  
使用 Vue 构建前端框架，结合 AI 技术实现短剧生成流程自动化，注重用户体验与交互设计。

适用场景  
- 短视频创作者快速生成内容  
- 广告公司制作创意短片  
- 教育领域用于剧本教学  
- 个人用户娱乐创作  
- 媒体公司批量生产内容

推荐理由  
提供高效、便捷的短剧生成解决方案，适合需要快速产出内容的创作者和企业，具备较强的创新性和实用性。
