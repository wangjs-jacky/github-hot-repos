# StrayMeteor3337/WechatRealFriends

微信好友关系一键检测，基于微信ipad协议，看看有没有朋友偷偷删掉或者拉黑你

项目定位  
微信好友关系检测工具，帮助用户检查是否被朋友删除或拉黑，基于微信iPad协议实现。

核心功能  
- 检测用户是否被微信好友删除或拉黑  
- 一键分析好友关系状态  
- 提供清晰的检测结果与提示  

技术特点  
使用JavaScript实现，基于微信iPad协议进行通信，具备轻量级和高效性，适合快速部署和使用。

适用场景  
- 用户怀疑被朋友删除或拉黑时自查  
- 微信社交关系维护与管理  
- 研究微信协议及好友关系机制  

推荐理由  
操作简单、功能实用，能快速帮助用户确认微信好友关系状态，适合日常社交需求。
