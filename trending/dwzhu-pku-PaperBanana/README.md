# dwzhu-pku/PaperBanana

PaperBanana: Automating Academic Illustration For AI Scientists

**项目定位**  
PaperBanana 是一个自动化生成学术插图的工具，帮助 AI 科学家高效制作高质量的图表和可视化内容。

**核心功能**  
- 自动生成符合学术规范的图表  
- 支持多种数据格式输入  
- 提供丰富的可视化模板  
- 一键导出为 PDF 或图像格式  
- 集成常见 AI 实验数据处理逻辑  

**技术特点**  
基于 Python 开发，结合 Matplotlib 和 Seaborn 等库实现可视化自动化，具备良好的可扩展性和易用性。

**适用场景**  
- 学术论文写作时快速生成图表  
- 会议或报告中展示实验结果  
- 教学演示中可视化复杂数据  
- 团队协作中统一图表风格  
- 重复性数据可视化任务  

**推荐理由**  
PaperBanana 节省科研人员手动绘图时间，提升论文质量和工作效率，适合 AI 领域研究者使用。
