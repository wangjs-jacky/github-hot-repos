# cobusgreyling/loop-engineering

Practical patterns, starters & CLI tools for loop engineering with AI coding agents. Design systems that prompt and orchestrate agents (inspired by Addy Osmani and Boris Cherny). Includes loop-audit, loop-init, loop-cost.

项目定位  
一个面向 AI 编码代理的工程化工具集，提供模式、启动模板和 CLI 工具，用于构建和管理 AI 驱动的开发流程。

核心功能  
- 提供 AI 代理的提示工程和编排模式  
- 包含 loop-audit、loop-init 和 loop-cost 等 CLI 工具  
- 支持自动化代码生成与审查流程  
- 集成 GitHub Actions 实现持续开发自动化  
- 适用于多种大模型平台如 Anthropic、Claude、Grok 等

技术特点  
基于 JavaScript 构建，融合 AI 代理与 DevOps 自动化，支持多模型集成，强调可扩展性和可维护性，适合构建智能开发流水线。

适用场景  
- 开发 AI 驱动的自动化代码生成系统  
- 构建智能提示工程和代理编排系统  
- 在 CI/CD 流程中集成 AI 代理进行代码审查  
- 优化开发效率并减少重复劳动  
- 为团队提供统一的 AI 编码工具集

推荐理由  
该项目为 AI 编码代理提供了结构化工具和实践指南，适合希望提升开发效率、探索 AI 在软件工程中应用的开发者和团队。
