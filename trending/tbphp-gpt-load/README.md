# tbphp/gpt-load

Multi-channel AI proxy with intelligent key rotation. 智能密钥轮询的多渠道 AI 代理。

项目定位  
一个支持多 AI 服务的代理系统，通过智能密钥轮询提升使用效率和稳定性。

核心功能  
- 支持 OpenAI、Claude、Gemini 等多渠道 AI 服务  
- 自动轮换 API 密钥以避免限流或封禁  
- 提供统一接口访问不同 AI 平台  
- 支持自定义配置和扩展  

技术特点  
基于 Go 语言开发，采用 Gin 框架，具备高性能和良好的可扩展性，适合构建稳定可靠的 AI 代理服务。

适用场景  
- 多 AI 服务集成项目  
- 需要密钥管理的 AI 应用  
- 高并发 AI 请求处理  
- 开发 AI 聊天机器人或工具  

推荐理由  
提供多渠道 AI 代理能力，结合智能密钥管理，有效提升 AI 接口的可用性和稳定性，适合需要多平台接入的开发者。
