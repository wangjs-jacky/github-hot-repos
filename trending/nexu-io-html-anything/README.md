# nexu-io/html-anything

✨ The agentic HTML editor — your local AI agent writes the HTML, you ship it. 🚀 75 Skills × 9 Surfaces (magazine · deck · poster · XHS / tweet · prototype · data report · Hyperframes) 🛡️ Sandboxed preview · 📤 1-click to WeChat / X / Zhihu / HTML / PNG 🔑 Zero API key — Claude Code / Cursor / Codex / Gemini / Copilot / OpenCode / Qwen / Aider.

**项目定位**  
一个基于 AI 的本地 HTML 编辑器，允许用户通过自然语言指令生成 HTML 内容，简化网页开发流程。

**核心功能**  
- 通过 AI 生成 HTML 内容  
- 支持多种输出格式（WeChat、X、Zhihu、HTML、PNG）  
- 提供沙盒预览环境  
- 集成多种 AI 模型（Claude、Qwen 等）  
- 支持 9 种不同内容场景（如杂志、幻灯片、数据报告等）

**技术特点**  
使用 HTML 和 JavaScript 构建，强调本地运行和无 API 密钥依赖，结合 AI 代理技术实现高效内容生成。

**适用场景**  
- 快速生成网页原型  
- 制作社交媒体内容（如微博、小红书）  
- 创建数据可视化报告  
- 无需编程知识的网页编辑  
- 本地化 AI 内容生成

**推荐理由**  
无需 API 密钥，支持多平台输出，适合希望快速生成 HTML 内容的开发者和非技术人员。
