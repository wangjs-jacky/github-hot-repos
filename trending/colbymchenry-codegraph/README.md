# colbymchenry/codegraph

Pre-indexed code knowledge graph for Claude Code, Codex, Cursor, and OpenCode — fewer tokens, fewer tool calls, 100% local

项目定位  
一个本地预索引代码知识图谱，用于提升 AI 编码工具的效率与准确性。

核心功能  
- 提供预索引的代码知识图谱  
- 支持 Claude Code、Codex 等多种 AI 工具  
- 减少 Token 使用和工具调用次数  
- 保证数据完全本地化处理  

技术特点  
使用 TypeScript 构建，专注于本地化代码知识管理，优化 AI 编码工具的交互效率。

适用场景  
- 开发者在本地增强 AI 编码助手  
- 需要隐私保护的代码分析任务  
- 提升 AI 代码生成的准确性和速度  
- 降低云服务依赖和成本  

推荐理由  
适合注重隐私和效率的开发者，提供高效、本地化的代码知识支持，提升 AI 编码工具的使用体验。
