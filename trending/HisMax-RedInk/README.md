# HisMax/RedInk

Red Ink - A one-stop Xiaohongshu image-and-text generator based on the 🍌Nano Banana Pro🍌, "One Sentence, One Image: Generate Xiaohongshu Text and Images."

**项目定位**  
RedInk 是一个基于 Nano Banana Pro 的小红书图文生成工具，通过一句文案生成对应的图文内容，提升内容创作效率。

**核心功能**  
- 一句文案生成小红书风格图文  
- 集成 AI 图像生成能力  
- 支持文本与图像的同步生成  
- 提供简洁的 Web 界面操作  
- 可扩展的插件系统支持

**技术特点**  
使用 Python 开发，结合 Flask 和 Docker 实现快速部署，集成 Gemini 等 AI 模型，适配小红书平台内容格式要求。

**适用场景**  
- 小红书博主快速生成内容  
- 品牌营销自动化图文制作  
- 内容创作者提高生产效率  
- 电商产品图文生成  
- 社交媒体运营辅助工具

**推荐理由**  
RedInk 为小红书内容创作提供一站式解决方案，结合 AI 技术实现高效图文生成，适合需要频繁产出内容的用户和团队。
