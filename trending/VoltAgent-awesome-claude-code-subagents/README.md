# VoltAgent/awesome-claude-code-subagents

A collection of 100+ specialized Claude Code subagents covering a wide range of development use cases

项目定位  
该项目是一个专注于 Claude Code 的子代理集合，提供多种开发场景的自动化解决方案。

核心功能  
- 提供 100+ 个专用子代理  
- 支持代码生成与优化  
- 覆盖多种开发任务和工具集成  
- 适用于不同编程语言和框架  

技术特点  
基于 Shell 脚本实现，便于集成与使用，专注于提升开发效率和自动化能力。

适用场景  
- 代码生成与重构  
- 自动化测试与部署  
- 开发环境配置  
- 多语言支持的开发任务  

推荐理由  
适合希望利用 Claude AI 提升开发效率的开发者，提供丰富的子代理工具，覆盖广泛的技术需求。
