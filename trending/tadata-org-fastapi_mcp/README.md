# tadata-org/fastapi_mcp

Expose your FastAPI endpoints as Model Context Protocol (MCP) tools, with Auth!

项目定位  
将 FastAPI 端点暴露为 Model Context Protocol (MCP) 工具，支持身份验证，便于集成到 AI 模型工作流中。

核心功能  
- 将 FastAPI 接口转换为 MCP 工具  
- 提供基于角色的认证与授权机制  
- 支持与 AI 模型（如 Claude、Cursor）无缝对接  
- 通过 OpenAPI 自动生成 MCP 接口描述  
- 简化模型与后端服务之间的交互流程

技术特点  
基于 Python 和 FastAPI 构建，结合 MCP 协议实现服务接口标准化，支持灵活的身份验证方式，具备良好的可扩展性和易用性。

适用场景  
- AI 模型调用后端服务时的接口封装  
- 需要统一管理多个 API 的场景  
- 需要集成认证机制的 AI 工作流系统  
- 快速搭建 MCP 兼容的服务端接口  
- 开发需要与 LLM 对话的智能应用

推荐理由  
适合希望将 FastAPI 服务与 AI 模型结合的开发者，提供标准化接口和安全认证，提升开发效率和系统兼容性。
