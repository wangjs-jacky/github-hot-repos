# Snapchat/Valdi

Valdi is a cross-platform UI framework that delivers native performance without sacrificing developer velocity.

1. **项目定位**  
Valdi 是一个跨平台 UI 框架，旨在提供原生性能的同时提升开发效率，适用于需要同时支持 Android 和 iOS 的应用开发。

2. **核心功能**  
- 跨平台 UI 开发支持  
- 原生性能优化  
- 与 C++ 高度集成  
- 支持类型安全的开发体验  
- 提供高效的构建和渲染机制  

3. **技术特点**  
Valdi 使用 C++ 实现核心逻辑，结合 TypeScript 提供灵活的 UI 构建方式，确保高性能和良好的开发体验，适合需要兼顾性能与开发速度的项目。

4. **适用场景**  
- 需要同时支持 Android 和 iOS 的应用  
- 对性能要求较高的 UI 场景  
- 希望减少重复代码的跨平台开发  
- 需要快速迭代和高效开发的团队  
- 企业级移动应用开发  

5. **推荐理由**  
Valdi 提供了高性能的跨平台解决方案，结合 C++ 和 TypeScript 的优势，适合希望在保持原生体验的同时提升开发效率的开发者和团队。
