# xbtlin/ai-berkshire

AI 时代的伯克希尔：基于 Claude Code / Codex 的价值投资研究框架。巴菲特·芒格·段永平·李录四大师方法论 + 多Agent并行研究。| AI-era Berkshire: a value investing research framework built for Claude Code / Codex. 4 masters' methodologies + multi-agent adversarial analysis.

项目定位  
一个基于 AI 的价值投资研究框架，融合巴菲特、芒格等大师方法论，利用 Claude Code / Codex 进行多 Agent 并行分析，提升投资决策效率。

核心功能  
- 集成多位投资大师的方法论进行策略构建  
- 利用 AI 模型进行股票基本面分析  
- 支持多 Agent 协同研究与对抗性分析  
- 提供投资组合管理工具  

技术特点  
采用 Python 实现，结合 Anthropic 的 Claude Code / Codex 进行代码生成与分析，具备良好的扩展性和智能化研究能力。

适用场景  
- 个人投资者进行股票基本面分析  
- 金融研究机构优化投资策略  
- AI 爱好者探索智能投顾应用  
- 价值投资学习与实践  

推荐理由  
结合经典投资理论与前沿 AI 技术，为价值投资提供创新工具，适合对量化投资和 AI 应用感兴趣的用户。
