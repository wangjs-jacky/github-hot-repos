# reconurge/flowsint

A modern platform for visual, flexible, and extensible graph-based investigations. For cybersecurity analysts and investigators.

项目定位  
flowsint 是一个面向网络安全分析师和调查人员的可视化、灵活且可扩展的图状调查平台，用于处理和分析网络取证与开放源情报（OSINT）数据。

核心功能  
- 图形化界面支持交互式调查流程设计  
- 支持多种数据源集成与自动化分析  
- 提供可定制的调查工作流模板  
- 实时数据可视化与关联分析  
- 便于团队协作与报告生成

技术特点  
基于 TypeScript 构建，具备良好的可扩展性和跨平台兼容性，采用现代化前端架构，支持模块化开发与插件系统，适合快速迭代和定制化需求。

适用场景  
- 网络安全事件调查  
- 开放源情报（OSINT）收集与分析  
- 威胁情报整合与可视化  
- 跨部门协作调查项目  
- 安全团队内部知识库构建

推荐理由  
flowsint 为网络安全调查提供了直观、高效的图形化工具，结合灵活的工作流设计和强大的数据整合能力，是提升调查效率和深度的理想选择。
