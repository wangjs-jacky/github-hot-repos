# superset-sh/superset

IDE for the AI Agents Era - Run an army of Claude Code, Codex, etc. on your machine

项目定位  
一个面向 AI 代理时代的集成开发环境（IDE），允许用户在本地运行多个 AI 编码代理，如 Claude Code 和 Codex，提升开发效率。

核心功能  
- 支持多 AI 代理并行运行  
- 提供终端和 TUI 界面交互  
- 集成 Git 工作树管理  
- 支持命令行操作与自动化任务  
- 与多种 AI 模型兼容，如 Codex、Claude Code

技术特点  
基于 TypeScript 构建，采用 Electron 实现桌面应用，结合 CLI 和 TUI 提供高效交互体验，支持多代理协同与工作流编排。

适用场景  
- 开发者本地测试 AI 编码代理  
- 自动化代码生成与调试  
- 多代理协作开发项目  
- 本地化 AI 工具集成  
- 提升开发效率与代码质量

推荐理由  
适合希望在本地高效利用 AI 编码代理的开发者，提供强大的多代理管理和交互能力，提升开发体验与生产力。
