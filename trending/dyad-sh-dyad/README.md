# dyad-sh/dyad

Local, open-source AI app builder for power users ✨ v0 / Lovable / Replit / Bolt alternative 🌟 Star if you like it!

项目定位  
一个本地开源的 AI 应用构建工具，面向高级用户，提供替代 Replit、Bolt 等平台的解决方案，支持多种大语言模型。

核心功能  
- 本地运行，无需依赖云端服务  
- 支持多种 AI 模型（如 OpenAI、Anthropic、Qwen 等）  
- 提供可视化界面和代码编辑功能  
- 轻量级，适合快速开发 AI 应用  

技术特点  
基于 TypeScript 构建，采用现代前端框架，支持多模型集成，强调本地化和开源特性，适合开发者自定义扩展。

适用场景  
- 快速构建 AI 应用原型  
- 本地测试和调试 AI 模型  
- 开发者在无网络环境下工作  
- 需要高度定制化的 AI 工具  

推荐理由  
适合希望在本地构建和管理 AI 应用的开发者，支持多种模型，开源且轻量，是 Replit 和 Bolt 的优质替代方案。
