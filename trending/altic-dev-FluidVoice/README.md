# altic-dev/FluidVoice

Fastest and only macOS Dictation app with on-device STT and custom trained AI enhancement model. A local Wispr Flow alternative. ⭐ helps a ton :) Windows & iOS waitlist open. Linux soon.

项目定位  
FluidVoice 是一款基于 macOS 的快速语音输入应用，提供本地化语音转文字功能，并结合自定义训练的 AI 模型提升识别效果，旨在替代 Wispr Flow。

核心功能  
- 本地化语音转文字（STT）  
- 自定义 AI 增强模型  
- 支持 macOS 系统  
- 提供更高效的语音输入体验  
- 未来将支持 Windows、iOS 和 Linux

技术特点  
使用 Swift 开发，采用本地化处理以保障隐私，结合自定义 AI 模型提升识别准确率，符合现代开发趋势，具备良好的性能和扩展性。

适用场景  
- 日常办公中的语音输入  
- 需要隐私保护的语音处理  
- 替代现有语音助手工具  
- 为开发者提供 AI 模型训练参考  
- 多平台用户需求未满足时的临时方案

推荐理由  
FluidVoice 提供高效、本地化的语音输入体验，结合 AI 技术提升准确性，适合注重隐私和效率的用户，是 macOS 用户值得尝试的创新工具。
