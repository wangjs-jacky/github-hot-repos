# TauricResearch/TradingAgents

TradingAgents: Multi-Agents LLM Financial Trading Framework

项目定位  
TradingAgents 是一个基于多智能体 LLM 的金融交易框架，用于构建和测试自动化交易策略，解决传统金融交易中效率低、响应慢的问题。

核心功能  
- 支持多智能体协同交易  
- 集成 LLM 生成交易决策  
- 提供回测与实时模拟功能  
- 支持多种金融数据源接入  
- 可扩展性强，便于自定义策略

技术特点  
使用 Python 实现，结合强化学习与大语言模型，具备高效的数据处理能力和灵活的策略部署方式，适合金融领域智能化交易开发。

适用场景  
- 量化交易策略开发与测试  
- 金融市场模拟与预测  
- 多智能体协作交易实验  
- 机构投资者策略优化  
- 金融科技研究与创新

推荐理由  
TradingAgents 提供了先进的多智能体 LLM 交易框架，适合研究人员与开发者快速构建和验证智能交易系统，具有高可扩展性和实用性。
