# yichuan-w/LEANN

[MLsys2026]: RAG on Everything with LEANN. Enjoy 97% storage savings while running a fast, accurate, and 100% private RAG application on your personal device.

项目定位  
LEANN 是一个基于本地存储的 RAG（检索增强生成）系统，旨在提供高效、私密且节省存储的问答服务，适用于个人设备上的离线应用。

核心功能  
- 利用向量数据库实现高效的文本检索  
- 支持本地运行，保障用户隐私  
- 通过 LEANN 技术实现高达 97% 的存储空间节省  
- 兼容多种 LLM 框架，如 Ollama 和 LangChain  
- 提供快速且准确的生成式问答能力

技术特点  
项目结合了向量搜索与本地存储技术，采用 Python 实现，支持离线运行，具备高性能和高隐私性，适合对数据安全要求高的场景。

适用场景  
- 个人设备上的私有知识库管理  
- 离线环境下的智能问答系统  
- 需要保护数据隐私的企业应用  
- 对存储成本敏感的 RAG 应用  
- 快速部署的本地化 AI 服务

推荐理由  
LEANN 提供了一个高效、私密且节省资源的 RAG 解决方案，适合希望在本地运行 AI 服务并保护数据隐私的开发者和用户。
