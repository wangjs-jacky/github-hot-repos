# ogulcancelik/herdr

agent multiplexer that lives in your terminal.

项目定位  
herdr 是一个运行在终端中的代理中继器，用于管理和协调多个 AI 代理，提升开发效率和工作流管理。

核心功能  
- 管理多个 AI 代理的输入输出  
- 提供终端内统一的交互界面  
- 支持与多种 AI 模型（如 Claude、Codex）集成  
- 实现代理之间的协作与任务分发  

技术特点  
使用 Rust 编写，注重性能与安全性，结合终端 UI 和多代理架构，提供高效、灵活的开发工具体验。

适用场景  
- 开发者在终端中同时调用多个 AI 工具  
- 自动化代码生成与调试流程  
- 多代理协作完成复杂任务  
- 提升命令行环境下的 AI 工具整合能力  

推荐理由  
herdr 通过终端集成 AI 代理，简化开发流程，提升效率，适合需要高效利用 AI 工具的开发者和团队。
