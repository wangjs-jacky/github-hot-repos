# open-multi-agent/open-multi-agent

From a goal to a task DAG, automatically. TypeScript-native multi-agent orchestration with MCP and live tracing. Three runtime dependencies.

项目定位  
一个基于 TypeScript 的多智能体编排框架，通过 MCP 和实时追踪实现从目标到任务 DAG 的自动化生成，支持多种 LLM 和工具调用。

核心功能  
- 自动将目标转化为任务有向无环图（DAG）  
- 支持多智能体协作与调度  
- 提供实时追踪与调试功能  
- 兼容多种大模型（如 OpenAI、Anthropic 等）  
- 结构化输出与工具使用集成

技术特点  
采用 TypeScript 构建，强调类型安全和模块化设计，支持运行时依赖注入，具备良好的扩展性和跨模型兼容性。

适用场景  
- 复杂任务的自动化分解与执行  
- 多智能体协作系统开发  
- LLM 驱动的流程编排与优化  
- 本地与云端混合模型部署  
- 需要结构化输出的 AI 应用场景

推荐理由  
适合需要构建复杂多智能体系统的开发者，提供强大的任务调度、实时追踪和跨模型支持，提升 AI 应用的灵活性与可维护性。
