# EKKOLearnAI/hermes-web-ui

Web dashboard for Hermes Agent — multi-platform AI chat, session management, scheduled jobs, usage analytics 

项目定位  
Hermes Web UI 是一个为 Hermes Agent 提供的多平台 AI 聊天管理界面，支持会话管理、任务调度和使用分析，帮助用户更高效地管理和监控 AI 代理服务。

核心功能  
- AI 聊天界面与交互管理  
- 多平台支持与自托管能力  
- 定时任务与作业调度  
- 使用数据分析与可视化  
- 用户会话历史记录与管理

技术特点  
基于 TypeScript 和 Vue3 构建，具备良好的可维护性和扩展性，支持多模型和多平台部署，适合需要自托管 AI 服务的开发者和团队。

适用场景  
- 企业内部 AI 代理系统管理  
- 开发者自托管 AI 服务  
- 需要任务调度和数据分析的 AI 应用  
- 多模型 AI 代理的统一管理界面  
- 快速构建 AI 聊天与控制面板

推荐理由  
Hermes Web UI 提供了强大的 AI 代理管理功能，结合 Vue3 和 TypeScript 技术栈，适合希望自托管 AI 服务并进行深度定制的开发者，具有良好的可扩展性和用户体验。
