# tiagozip/cap

The privacy-first, self-hosted CAPTCHA for the modern web.

**项目定位**  
Tiagozip/cap 是一个注重隐私的自托管 CAPTCHA 解决方案，旨在防止网络滥用、机器人和垃圾信息。

**核心功能**  
- 提供隐私友好的 CAPTCHA 验证机制  
- 支持自托管，无需依赖第三方服务  
- 采用工作量证明（Proof-of-Work）技术增强安全性  
- 简洁易集成的 JavaScript API  

**技术特点**  
该项目使用 JavaScript 构建，强调隐私和安全，结合工作量证明与哈希现金算法，实现轻量级且高效的反滥用机制。

**适用场景**  
- 网站注册与登录保护  
- 防止表单提交被机器人滥用  
- 自托管网站的安全增强  
- 需要避免第三方跟踪的隐私敏感应用  

**推荐理由**  
Tiagozip/cap 是一个轻量、隐私优先的 CAPTCHA 实现，适合需要自托管和高安全性的现代 Web 应用。
