# sansan0/TrendRadar

⭐AI-driven public opinion & trend monitor with multi-platform aggregation, RSS, and smart alerts.🎯 告别信息过载，你的 AI 舆情监控助手与热点筛选工具！聚合多平台热点 +  RSS 订阅，支持关键词精准筛选。AI 智能筛选新闻 + AI 翻译 +  AI 分析简报直推手机，也支持接入 MCP 架构，赋能 AI 自然语言对话分析、情感洞察与趋势预测等。支持 Docker ，数据本地/云端自持。集成微信/飞书/钉钉/Telegram/邮件/ntfy/bark/slack 等渠道智能推送。

**项目定位**  
TrendRadar 是一个基于 AI 的多平台舆情监控与趋势分析工具，帮助用户高效筛选热点信息，解决信息过载问题。

**核心功能**  
- 多平台热点聚合（包括 RSS 和社交媒体）  
- AI 智能筛选、翻译与分析简报生成  
- 支持多种渠道（微信、钉钉、邮件等）推送  
- 可接入 MCP 架构实现自然语言对话分析  
- 关键词精准过滤与趋势预测  

**技术特点**  
使用 Python 开发，支持 Docker 部署，具备本地或云端数据管理能力，结合 AI 技术实现自动化内容处理与分析，技术架构灵活可扩展。

**适用场景**  
- 企业舆情监控与品牌管理  
- 媒体机构实时热点追踪  
- 个人或团队的信息筛选与学习  
- 跨平台新闻聚合与分析  
- 自动化通知与报告生成  

**推荐理由**  
功能全面且易于部署，结合 AI 与多渠道推送，适合需要高效获取和分析热点信息的用户，尤其适合关注舆情与趋势的开发者和企业。
