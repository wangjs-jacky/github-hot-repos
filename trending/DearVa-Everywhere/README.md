# DearVa/Everywhere

Context-aware AI assistant for your desktop. Ready to respond intelligently, seamlessly integrating multiple LLMs and MCP tools.

项目定位  
一个面向桌面的上下文感知 AI 助手，集成多种大语言模型和工具，提供智能化、无缝的交互体验。

核心功能  
- 支持多 LLM 集成（如 OpenAI、Claude、Gemini 等）  
- 提供基于 RAG 的智能问答能力  
- 实现 UI 自动化与 MCP 工具整合  
- 本地化部署与跨平台支持  

技术特点  
使用 C# 开发，结合 Avalonia 框架实现跨平台 UI，支持多种 AI 模型和工具的灵活接入，具备良好的扩展性与可维护性。

适用场景  
- 个人办公自动化  
- 多模型协作开发环境  
- 智能客服系统构建  
- 企业级 AI 助手部署  

推荐理由  
适合需要集成多种 AI 模型并实现桌面智能化操作的开发者与用户，具备良好的扩展性和用户体验。
