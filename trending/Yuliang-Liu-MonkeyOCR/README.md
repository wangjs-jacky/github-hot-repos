# Yuliang-Liu/MonkeyOCR

A lightweight LMM-based Document Parsing Model

1. **项目定位**  
MonkeyOCR 是一个基于轻量级大模型（LMM）的文档解析模型，旨在高效处理和理解文档内容，适用于需要快速提取信息的场景。

2. **核心功能**  
- 文档内容解析与结构化  
- 支持多种文档格式输入  
- 高效文本提取与分类  
- 轻量级模型设计，适合部署  

3. **技术特点**  
采用轻量级大模型架构，优化推理速度与资源占用，兼顾准确率与效率，适合实际应用环境。

4. **适用场景**  
- 企业文档自动化处理  
- 票据与表单信息提取  
- 学术论文内容分析  
- 客户端文档解析服务  

5. **推荐理由**  
轻量且高效，适合对性能有要求的文档解析任务，尤其适合需要快速部署的生产环境。
