# greensock/gsap-skills

Official AI skills for GSAP. These skills teach AI coding agents how to correctly use GSAP (GreenSock Animation Platform), including best practices, common animation patterns, and plugin usage.

项目定位  
GSAP 技能库，用于指导 AI 编码代理正确使用 GSAP 动画库，涵盖最佳实践和常见动画模式。

核心功能  
- 提供 GSAP 的使用指南和示例  
- 教授常见动画模式和插件用法  
- 优化 AI 编码代理的动画实现能力  
- 包含最佳实践和错误避免方法  

技术特点  
基于 GSAP 的官方技能，专注于提升 AI 对动画库的理解和应用，强调代码质量和效率，适合与 AI 工具结合使用。

适用场景  
- AI 编码代理学习 GSAP 使用  
- 动画开发人员快速上手 GSAP  
- 教育和培训 AI 开发者  
- 提升自动化动画生成的准确性  

推荐理由  
为 AI 和开发者提供清晰的 GSAP 使用路径，帮助提高动画开发效率和质量，是学习和应用 GSAP 的重要资源。
