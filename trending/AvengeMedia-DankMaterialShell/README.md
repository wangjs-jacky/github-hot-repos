# AvengeMedia/DankMaterialShell

Desktop shell for wayland compositors built with Quickshell & GO, optimized for niri, hyprland, sway, MangoWC, labwc, and MiracleWM.

AvengeMedia/DankMaterialShell 是一个为 Wayland 显示服务器构建的桌面外壳，专注于提升用户在 niri、hyprland、sway 等窗口管理器上的使用体验。  

- 提供 Material 风格的 UI 设计  
- 支持多种 Wayland 窗口管理器  
- 基于 QuickShell 和 Go 构建，性能优化  
- 可自定义主题和布局  
- 与主流 Wayland 组合器兼容性好  

采用 QML 实现图形界面，结合 Go 语言实现后端逻辑，具备良好的性能和可扩展性，适合需要高度定制的 Wayland 用户。  

- 为 Wayland 用户提供美观且高效的桌面环境  
- 适用于开发者和高级用户进行个性化配置  
- 适配多种主流 Wayland 窗口管理器  
- 适合追求简洁与功能平衡的桌面体验  

DankMaterialShell 是一个轻量级、高度可定制的 Wayland 桌面外壳，适合希望在不同窗口管理器上获得一致体验的用户，尤其适合技术爱好者和开发者。
