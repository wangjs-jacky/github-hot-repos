# CloakHQ/CloakBrowser

Stealth Chromium that passes every bot detection test. Drop-in Playwright replacement with source-level fingerprint patches. 30/30 tests passed.

**项目定位**  
CloakBrowser 是一个隐身 Chromium 浏览器，旨在通过所有机器人检测测试，作为 Playwright 的替代品，解决网页自动化中的反检测问题。

**核心功能**  
- 通过所有 bot 检测测试（30/30 通过）  
- 提供源码级指纹修改功能  
- 支持与 Playwright 兼容的 API 接口  
- 集成反 CAPTCHA 和 Cloudflare 绕过机制  

**技术特点**  
基于 Python 实现，采用源码级指纹修改技术，确保浏览器指纹与真实用户一致，有效规避反爬虫机制，支持多种自动化工具集成。

**适用场景**  
- 网站数据抓取与网络 scraping  
- 自动化测试与 bot 模拟  
- 反 CAPTCHA 和 Cloudflare 绕过需求  
- 多账号管理与 anti-detect 场景  

**推荐理由**  
CloakBrowser 提供高效、稳定的无头浏览器解决方案，专为绕过现代网站检测机制设计，适合需要高隐蔽性自动化操作的开发者和研究人员。
