# bytedance/deer-flow

An open-source SuperAgent harness that researches, codes, and creates. With the help of sandboxes, memories, tools, skill, subagents and message gateway, it handles different levels of tasks that could take minutes to hours.

**项目定位**  
一个开源的超级代理框架，用于研究、编码和创造，通过沙箱、记忆、工具等组件处理复杂任务。

**核心功能**  
- 支持多级任务处理  
- 集成沙箱环境进行安全执行  
- 提供记忆存储与工具调用能力  
- 支持子代理协作与消息网关  

**技术特点**  
基于 Python 构建，采用模块化设计，支持灵活扩展，结合沙箱和记忆机制提升任务处理能力。

**适用场景**  
- 自动化研究与开发流程  
- 复杂任务的分步执行  
- 多代理协作系统构建  
- 安全环境下的代码执行  

**推荐理由**  
适合需要高效处理复杂任务和多代理协作的开发者，具备良好的扩展性和安全性，是构建智能代理系统的优秀选择。
