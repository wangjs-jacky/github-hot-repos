# czlonkowski/n8n-mcp

A MCP for Claude Desktop / Claude Code / Windsurf / Cursor to build n8n workflows for you 

项目定位  
该项目是一个用于 Claude 系列工具的 MCP（Model Control Protocol）服务器，帮助用户通过 n8n 构建自动化工作流，提升开发效率。

核心功能  
- 与 Claude Desktop、Claude Code 等工具集成  
- 提供 n8n 工作流构建支持  
- 实现模型调用与数据处理的自动化  
- 支持多种 AI 模型的接入与管理  

技术特点  
使用 TypeScript 开发，具备良好的类型安全和可维护性，适配现代前端与后端架构，支持灵活扩展和集成。

适用场景  
- 自动化代码生成与测试  
- 数据处理与分析流程构建  
- AI 模型调用与任务调度  
- 开发者工具链整合  

推荐理由  
项目定位清晰，技术成熟，适合需要将 Claude 系列工具与 n8n 结合使用的开发者，提升工作效率和开发体验。
