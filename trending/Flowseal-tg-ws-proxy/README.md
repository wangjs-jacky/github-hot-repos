# Flowseal/tg-ws-proxy

Local MTProto proxy server for partial bypassing of Telegram loading

项目定位  
一个本地 MTProto 代理服务器，用于部分绕过 Telegram 的加载限制，提升访问速度和稳定性。

核心功能  
- 提供本地代理服务，优化 Telegram 连接  
- 支持多平台部署与使用  
- 简化 Telegram 的网络连接过程  
- 可配置性强，便于定制化设置  

技术特点  
基于 Python 开发，轻量高效，支持异步处理，具备良好的可扩展性和跨平台兼容性。

适用场景  
- 国内用户绕过网络限制访问 Telegram  
- 企业或个人需要稳定 Telegram 连接  
- 需要自建代理服务的开发者  

推荐理由  
适合需要稳定、快速访问 Telegram 的用户，尤其在受限网络环境下表现优异，且易于部署和维护。
