# BlockRunAI/ClawRouter

The agent-native LLM router for OpenClaw. 41+ models, <1ms routing, USDC payments on Base & Solana via x402.

**项目定位**  
ClawRouter 是一个为 OpenClaw 设计的 AI 代理原生 LLM 路由器，旨在高效地分配模型请求并优化成本。

**核心功能**  
- 支持 40+ 种大语言模型  
- <1ms 的快速路由响应  
- 集成 USDC 微支付功能  
- 基于 x402 协议实现智能路由  
- 支持 Base 和 Solana 区块链

**技术特点**  
使用 TypeScript 构建，具备高性能和低延迟特性，结合微支付系统实现成本可控的模型调用路由。

**适用场景**  
- AI 代理系统中的模型调度  
- 多模型成本优化管理  
- 区块链上的微支付应用  
- 实时推理任务的高效处理  
- 开发者构建 AI 服务的基础设施

**推荐理由**  
ClawRouter 提供了高效的模型路由与微支付集成，适合需要多模型管理和成本控制的 AI 应用场景，是 OpenClaw 生态的重要组件。
