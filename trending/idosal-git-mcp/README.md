# idosal/git-mcp

Put an end to code hallucinations! GitMCP is a free, open-source, remote MCP server for any GitHub project

项目定位  
GitMCP 是一个开源的远程 MCP 服务器，旨在解决代码生成中的幻觉问题，支持 GitHub 项目使用 AI 工具进行更可靠的代码协作。

核心功能  
- 提供安全的 AI 代码生成环境  
- 集成多种 LLM（如 Claude、Copilot）  
- 支持 Git 集成与版本控制  
- 防止代码生成过程中的错误或幻觉  
- 允许团队协作和审查 AI 生成的代码  

技术特点  
基于 TypeScript 构建，支持与 GitHub 深度集成，提供可扩展的 AI 代理架构，注重安全性和可靠性，适用于各类开发流程。

适用场景  
- 开发团队使用 AI 辅助编码  
- 需要防止 AI 生成错误代码的项目  
- 需要与 Git 系统无缝集成的开发流程  
- 企业级 AI 代码审查与协作  
- 自动化代码生成与测试流程  

推荐理由  
GitMCP 提供了一个安全、可靠且易于集成的 AI 代码生成平台，特别适合希望减少代码幻觉风险并提升协作效率的开发者和团队。
