# VoltAgent/awesome-openclaw-skills

The awesome collection of OpenClaw skills. 5,400+ skills filtered and categorized from the official OpenClaw Skills Registry.🦞

项目定位  
一个整理和分类 OpenClaw 技能的精选集合，提供 5,400 多个经过筛选的技能资源，便于开发者快速查找和使用。

核心功能  
- 收集并分类超过 5,400 个 OpenClaw 技能  
- 提供清晰的技能标签和搜索功能  
- 持续更新和维护技能库内容  
- 为开发者提供高效技能检索工具  

技术特点  
项目基于开源社区协作模式，采用结构化数据管理方式，确保技能信息的准确性和易用性，适合技能生态系统的扩展与维护。

适用场景  
- 开发者快速查找可用技能  
- 技能创作者发布和管理技能  
- 教育机构用于教学参考  
- 技术爱好者探索技能多样性  

推荐理由  
丰富的技能资源、良好的分类体系和持续更新机制，是 OpenClaw 相关开发者的必备参考库。
