# alirezarezvani/claude-skills

+192 Claude Code skills & agent plugins for Claude Code, Codex, Gemini CLI, Cursor, and 8 more coding agents — engineering, marketing, product, compliance, C-level advisory.

1. **项目定位**  
该项目为 Claude Code 及其他代码生成 AI 提供技能和插件扩展，提升开发效率与功能多样性。

2. **核心功能**  
- 提供多种 AI 编程代理的技能包  
- 支持 Claude Code、Codex、Gemini CLI 等工具  
- 包含工程、营销、合规等领域的专业技能  
- 优化 AI 代码生成与执行流程  

3. **技术特点**  
基于 Python 实现，兼容多个 AI 编程代理，强调模块化与可扩展性，便于开发者快速集成和使用。

4. **适用场景**  
- 开发者提升 AI 编程效率  
- 企业构建定制化 AI 工具链  
- 研究人员测试不同 AI 编程代理性能  
- 教育领域用于 AI 编程教学  

5. **推荐理由**  
提供丰富技能与插件支持，适配多种 AI 编程平台，适合希望扩展 AI 工具功能的开发者和团队。
