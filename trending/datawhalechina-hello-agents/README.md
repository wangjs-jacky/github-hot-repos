# datawhalechina/hello-agents

📚 《从零开始构建智能体》——从零开始的智能体原理与实践教程

项目定位  
《从零开始构建智能体》教程，帮助开发者学习智能体原理与实践，适合初学者入门 AI 智能体开发。

核心功能  
- 提供智能体基础概念讲解  
- 包含 LLM（大语言模型）应用示例  
- 教程涵盖 RAG（检索增强生成）技术  
- 提供实际项目开发步骤指导  
- 支持 Python 代码实现与调试

技术特点  
基于 Python 的智能体开发教程，融合 LLM 和 RAG 技术，注重实践操作，适合初学者系统学习智能体构建方法。

适用场景  
- AI 开发者入门学习  
- 企业进行智能体技术培训  
- 学生进行人工智能课程实践  
- 研究人员探索智能体应用  
- 个人项目开发参考

推荐理由  
内容系统全面，适合零基础学习智能体开发，结合 LLM 和 RAG 技术，提供实用教程和代码示例，助力快速上手。
