# farion1231/cc-switch

A cross-platform desktop All-in-One assistant tool for Claude Code, Codex, OpenCode, openclaw & Gemini CLI.

项目定位  
一个跨平台的桌面全能助手工具，支持 Claude Code、Codex、OpenCode、OpenClaw 和 Gemini CLI，提供统一界面与功能管理。

核心功能  
- 支持多 AI 模型接入与切换  
- 提供代码生成与执行功能  
- 集成桌面应用与命令行工具  
- 管理不同 AI 服务提供商  
- 支持 WSL 和 Tauri 跨平台框架  

技术特点  
使用 Rust 开发，结合 Tauri 构建跨平台桌面应用，支持多种 AI 模型和 CLI 工具，具备良好的性能和可扩展性。

适用场景  
- 开发者快速调用 AI 生成代码  
- 多模型切换与测试环境搭建  
- 在 Windows 和 Linux 系统中使用 CLI 工具  
- 企业级 AI 工具统一管理  
- 个人开发者提升工作效率  

推荐理由  
功能全面、跨平台支持好、基于 Rust 的高性能实现，适合需要整合多个 AI 工具的开发者和团队使用。
