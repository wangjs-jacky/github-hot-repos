# caelestia-dots/shell

‼️ No waybar here ‼️

项目定位  
一个基于 QML 的轻量级桌面环境，专为 Hyprland Wayland 桌面打造，提供简洁美观的界面和高效的操作体验。

核心功能  
- 提供自定义的桌面界面和控件  
- 支持快速启动器和系统托盘  
- 集成 Hyprland 管理器，优化窗口布局  
- 轻量级设计，资源占用低  
- 可定制化程度高，适合高级用户

技术特点  
使用 QML 构建，结合 Qt6 框架，专注于高性能与简洁性，适用于 Wayland 环境下的桌面美化与功能扩展。

适用场景  
- 个人 Linux 桌面美化  
- Hyprland 用户提升操作体验  
- 开发者调试和测试 QML 应用  
- 追求轻量级桌面环境的用户  
- 自定义系统界面的爱好者

推荐理由  
适合希望在 Hyprland 上获得高度定制化、美观且高效的桌面体验的用户，尤其适合对 QML 和 Qt 技术感兴趣的开发者。
