# jarrodwatts/claude-hud

A Claude Code plugin that shows what's happening - context usage, active tools, running agents, and todo progress

项目定位  
Claude Code 插件，用于实时展示代码上下文、工具状态、代理运行情况和待办进度，提升开发效率。

核心功能  
- 显示当前代码上下文信息  
- 实时更新工具使用状态  
- 展示正在运行的代理状态  
- 跟踪待办事项进度  

技术特点  
基于 JavaScript 和 TypeScript 构建，适配 Claude Code 环境，提供轻量级 CLI 工具与状态展示功能。

适用场景  
- 开发者在使用 Claude Code 时需要实时反馈  
- 需要监控代码生成过程中的上下文和工具使用  
- 提升多代理协作和任务管理的可视化体验  

推荐理由  
为 Claude Code 用户提供直观的状态监控与进度反馈，提升代码生成和任务管理的透明度与效率。
