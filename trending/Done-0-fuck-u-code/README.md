# Done-0/fuck-u-code

Legacy-Mess Detector – assess the “legacy-mess level” of your code and output a beautiful report

项目定位  
一个用于检测代码“遗留混乱程度”的工具，帮助开发者评估和改善代码质量。

核心功能  
- 分析代码中的复杂度和冗余问题  
- 识别潜在的维护难点  
- 生成可视化报告  
- 支持多种代码格式分析  

技术特点  
使用 TypeScript 开发，具备良好的类型安全和可维护性，支持灵活的插件系统，适合集成到开发流程中。

适用场景  
- 团队代码审查  
- 项目重构前评估  
- 新成员快速理解代码结构  
- 持续集成流程中自动检测代码质量  

推荐理由  
提供直观的代码质量评估，帮助团队及时发现和解决潜在的维护问题，提升整体代码可读性和可维护性。
