# jnsahaj/tweakcn

A visual no-code theme editor for shadcn/ui components

**项目定位**  
一个用于 shadcn/ui 组件的可视化无代码主题编辑器，帮助开发者快速自定义 UI 主题。

**核心功能**  
- 可视化界面编辑 shadcn/ui 组件主题  
- 实时预览主题变化效果  
- 支持导出和集成到项目中  
- 简化 Tailwind CSS 配置流程  
- 与 shadcn-ui 生态深度集成  

**技术特点**  
使用 TypeScript 构建，基于 React 和 Tailwind CSS，提供直观的 UI 编辑体验，简化主题管理流程。

**适用场景**  
- 快速定制 shadcn/ui 组件样式  
- 无需编码的 UI 主题调整  
- 提升开发效率和一致性  
- 适用于前端开发者和设计团队  
- 适合需要快速迭代 UI 的项目  

**推荐理由**  
为 shadcn/ui 用户提供高效的无代码主题编辑工具，提升 UI 自定义灵活性和开发效率。
