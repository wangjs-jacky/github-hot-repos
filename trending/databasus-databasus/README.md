# databasus/databasus

PostgreSQL backup tool (with MySQL\MariaDB and MongoDB support)

项目定位  
databasus 是一个支持 PostgreSQL、MySQL/MariaDB 和 MongoDB 的数据库备份工具，旨在为 DevOps 提供高效、可靠的数据库备份解决方案。

核心功能  
- 支持多种数据库类型的备份与恢复  
- 提供 Web 界面进行管理操作  
- 支持通过 Docker 部署和运行  
- 可将备份文件存储到 S3 或本地系统  
- 适用于 Kubernetes 环境中的自动化备份流程  

技术特点  
使用 Go 语言开发，具备高性能和跨平台能力，结合 Docker 和 Web 界面提升易用性，适合云原生和自托管环境。

适用场景  
- 企业级数据库的定期备份  
- 开发和测试环境的快速恢复  
- Kubernetes 集群中的数据库管理  
- 自托管数据库系统的备份需求  
- 需要集成 S3 存储的备份方案  

推荐理由  
databasus 功能全面且易于部署，支持多数据库类型和现代运维工具，是 DevOps 工程师进行数据库备份的理想选择。
