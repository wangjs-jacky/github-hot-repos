# Egonex-AI/Understand-Anything

Graphs that teach > graphs that impress. Turn any code into an interactive knowledge graph you can explore, search, and ask questions about. Works with Claude Code, Codex, Cursor, Copilot, Gemini CLI, and more.

**项目定位**  
Egonex-AI/Understand-Anything 是一个将代码转换为可交互知识图谱的工具，帮助开发者更好地理解和探索代码结构与逻辑。

**核心功能**  
- 将任意代码转换为知识图谱  
- 支持搜索和提问功能  
- 与多个 AI 编码工具兼容（如 Claude Code、Copilot）  
- 提供代码结构的可视化分析  

**技术特点**  
使用 TypeScript 构建，专注于代码分析与知识图谱生成，具备良好的扩展性和与主流 AI 工具的兼容性。

**适用场景**  
- 开发者快速理解复杂代码库  
- 教学与代码讲解场景  
- 代码审查与维护  
- AI 辅助编程工具集成  
- 构建代码知识库与记忆系统  

**推荐理由**  
该工具通过知识图谱提升代码理解效率，适合需要深入分析代码结构的开发者和 AI 工具用户，具有很强的实用价值和扩展性。
